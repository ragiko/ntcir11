CPU資源を複数のアプリに分割して割り当てるCPU資源管理システムにおいて、各アプリの動作周期またはCPU制御周期ごとに、前記動作周期またはイベント応答要求時間の短いアプリから順にCPU資源を割り当てることを特徴とする。In the CPU resource management system for dividing and assigning CPU resources to a plurality of applications, the CPU resources are assigned, for each operation period or CPU control period of each application, in order from an application shorter in operation period or event answer request time. - 特許庁
