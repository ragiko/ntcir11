KEYWORD-BASED BREADCRUMBS : A Way to Search Unknown Contents in a Content-Oriented Network
近年,コンテンツ配信や共有をインターネットに代表されるIPネットワークよりも効率的に行うアプローチとして,コンテンツ指向ネットワーク(CON)が注目されている.CONでは,コンテンツ取得要求は当該サーバの位置ではなくコンテンツ名に基づいてルーチングされる.このようなルーチングを実現する方式の1つとしてBreadcrumbs (BC)が挙げられる.この方式では,コンテンツのダウンロード経路上にある各ルータは,BCと呼ばれるコンテンツ転送方向を示す情報を保存することで,取得要求のインネットワーク誘導を行う.取得要求は,基本的にはコンテンツ保持サーバヘルーチングされるが,その経路上で当該コンテンツに対するBCをもつルータに達した場合,BCの示す方向へコンテンツキャッシュを発見するまで誘導される.CONにおいて,ユーザはコンテンツを名前に基づいて検索しなければならないが,膨大な量のコンテンツにそれぞれ一意の名前を付与する必要があり,人間にとって扱いにくい名前が付けられることもある.よって,ユーザが行えるコンテンツ名のみに基づく検索には限界がある.そこで,本稿では,ユーザに対して親和性のあるCONをBCに基づいて実現する方式として,Keywords-based BC (KBC)を提案する.このKBCでは,キーワードという概念をコンテンツとBCに追加し,ユーザは名前が不明なものを含む様々なコンテンツをキーワードを用いて効率的に検索することが可能となる.最後に,計算機シミュレーションを用いた性能評価により提案方式の有効性を示す.
Content-Oriented Network (CON) is a new approach thought to be more efficient than the Internet for content delivery and sharing. For this purpose, the CON works with content names instead of host location as in the current Internet. Breadcrumbs (BC) is one of the representative routing methods to realize CON. In this BC, each router on the download path of a content makes a BC entry which shows the direction of the content transfer so that in-network guiding of request is available. Basically, a request for a content is routed to the corresponding server which contains the content. If the request encounters a BC corresponding to this content in a router on its path, the router will redirect the request to the direction shown by the BC entry and follow the BC trail until finding the desired content or the invalidated one. In CONs, users search desired contents only by names. Hence all content names are unique and the naming policy can make the content names non-human readable. A consequence is that users have difficulties to search contents only by their names. In this paper, we present a user-friendly CON based on BC which is named Keywords-based BC (KBC). In this KBC system, keywords are added to both contents and BCs. Users can search various contents including name-unknown contents efficiently. Finally, we demonstrate the effectiveness of the proposed method by computer simulation.
