
オンデマンド・ネットワーク型
日本語音声教育システム
三輪 譲二
岩手大学工学部情報工学科
webmaster
1. 概要  
インターネットとWWW(World Wide Web)の進歩に伴って,世界各地のどこからでも,
自分の都合の良い時間にいつでも,だれでも,
オンデマンドで世界中の情報をアクセスすることが出来るようになったきた。
また,マルチメディア技術の進歩に伴って,
文書ばかりでなく音声や画像等を容易に扱うことができるようになってきた。
さらに,Java言語を利用することにより,
コンピュータの機種に依存しない
マルチプラットホームとなり,
言語の文字種によらないマルチランゲージに
対応することができるようになってきた。
これらの技術的背景をもとに,
いつでも,どこでも,だれにでも利用できるバーチャル・ランゲージスクール
(仮想語学学校: Vertual Language School)の構築を目指し,
オンデマンド・ネットワーク型日本語音声教育システム(LESSON/J: 
Japanese-Language Education System for Speech on an On-demand Network)を,
Java言語を用いてWWW上に構築した。
2. 背景  
日本語音声教育システム開発の背景を以下に示す。
規制緩和による貿易障壁の除去とともに,
日本語の言語障壁を取り除くために,
日本語教育の効果的なシステムの開発が望まれている。
特に,実際に,会話として日本語を行うためには,文法教育の他に、
音声言語の教育が重要である。
学生ばかりでなく職業を持つ社会人もいつでも利用できるようにするため,
時間に束縛されない仮想学校(バーチャルスクール)が必要である。
また,世界中の人がどこでも利用するため,インターネットとWWWを利用することが必要である。
さらに,だれにでも簡単に使用できるように,ヒューマン・インタフェースに優れたGUI(Graphical User Interface)を利用する必要がある。
従来の音声言語教育システムは、
(1)CD−ROMを用いたスタンドアローンのシステムであり,ネットワークに対応していないこと,
(2)コンピュータのOSに依存しておりマルチプラットホーム(他機種)でないこと,
(3)母語の言語毎に異なったプログラムであり,マルチランゲージ(多言語)に対応していない
こと、等の問題があった。
本システムでは、インターネット,マルチメディア,WWW,
Javaの現存の技術を統合し、
図1に示すように、いつでも、どこでも、だれにでも
使用できるシステムを構築した。
図1: システム構成図
3. 構成・新規性・有用性  
システムの構成,新規性,有用性を以下に示す。
3.1.構成  
日本語音声教育システムは、図2に示すように、
音節,
単語,
基本文型,
子音組,
長母音,
促音,
アクセント
の7種類の日本語音声の聞き取りクイズ
ができるアプレットで構成されている。
なお,アプレットは、拡張性とアクセスの高速化のために、
7種類全部の教材classが、
基本dictation.classをextendsするようになっている。
また,英語,中国語,マレー語,韓国語、日本語,ローマ字のマルチランゲージ(多言語)に
対応している。
さらに、この画面から分かるように、種々の日本語音声学習教材を、
学習者のレベルに応じて選択できるようになっている。
図3は、英語を母語とする学習者のための日本語単語音声学習画面の例であり、
この例では、ヒントとしてボタンをクリックして、蟹(かに)の絵を表示することにより、
「かに」と正しく解答している。
このように、正答の場合には、青の丸が表示され、また、英語版では、Output Boxに、
"You are right"と表示される。
さらに、問題数と正答数が下中のBoxに表示される。
この例では、1問中1問が正解であることを示している。
以下で、ボタンの説明を行う。
Conformボタンは、答えの確定入力の確定を行う。
Replayボタンは、音声が聞き取れなかった場合、再出力をする。
Soundボタンは、音節の音声を出力するトグルスイッチであり、
かな文字の部分をクリックすると対応する音声を出力することができる。
Delボタンは、文字入力を1文字削除する。
Hintボタンは、音声に対応する絵を表示する。
Helpボタンは、使用法等の説明を行う。
このボタンを使用すれば、Javaプログラムの変数を
リセットすることなく、再実行できる。
Nextボタンは、次の音声資料の提示を行う。
3.2.新規性  
このシステムの新規的特色について述べる。
いつでも,どこでも,だれにでも利用できるバーチャル・
ランゲージスクール(仮想語学学校)の
オンデマンド・ネットワーク型日本語音声教育システムである。
実際に使用に耐える機能を有する日本語音声教育システムを構築している。
音声データは、日本語音声の標準化のためNHKアナウンサーに依頼した。
外国人の日本語教育ばかりでなく、幼稚園や小学生の国語教育に役立てるため、
ひらがなの文字も用いている。
日本語学習の利用状況の統計情報の収集が出来、
にがてな音声を重点的に指導できる。(ロギング機能)
日本語音声の出力やヒントの絵等の画像表示を,
Javaのマルチメディア機能により実現している。
Javaの乱数関数により,教材の提示順序を定型化しないようにした。
音声聞き取りの解答を,音節表からマウスで選択しても良いし,ローマ字またはひらがなで日本語を直接キーボードから入力しても良いように,マルチインプット法を用いている。
Windows, MacOS, UNIX等のOSや機種に依存しないマルチプラットホーム対応である。
JavaのGUI(グラフィカル・ユーザ・インタフェ−ス)を用いることにより、
利用者に使いやすいシステムが構築できている。
Javaのユニコードを用いることにより,1個のJava classを用いて,
英語、
中国語、
マレー語、
韓国語、
日本語
のマルチランゲージに対応しており、
これは、Javaのみで可能である。
マルチランゲージに対応していないブラウザのバージョン用に,
ローマ字の表記で対応した。
日本語教育ばかりでなく,ひらがな,カタカナ,
ローマ字等の
国語教育にも対応できるようにアプレットの文字をひらがな表記とした。
学習教材の変更は、WWWサーバのコンテンツの変更のみで行うので,
迅速であり,容易である。
Java技術をもちいることにより、日本語音声教育の仮想学校を構築でき、
マルチプラットホームによるオープン化が、
Java技術の前進や普及に貢献している。
3.3.有効性  
現在アクセス統計等により評価を行っている途中であるが,
ある留学生は以下のように評価している。
自分の能力に応じて個別に学習を進めていけるので良い。
'し'と'ひ'のように,異聴が生じやすい音声があり,
その音声を重点的に聞き取り訓練ができるのでよい。
音声とヒントの絵の対応で単語の習得に役立っている。
基本音節と濁音等を含む音節に分けて有るので,良い。
自分の母国語が使われているので,うれしい。
ゲーム性が有って楽しい。
無料であるのでうれしい。
これより,外国の大学の日本語学科や日本の大学の留学生センター等でも
組織的に使用するに値するシステムとなっていると思われる。
なお,欠点として以下が指摘されている。
アクセスに時間がかかる。
音質が悪い音声資料がある。
4. システムの実行例  
システムの動作例を以下に簡単に説明する。
図4は、外国人のための英語版の日本語音声教育教材選択画面である。
音声学習において、正答の場合は青の丸印、
誤答の場合は赤のバツ印を表示する。
また、対応する音で応答する。
学習者のレベルに応じて対応するため、
基本音節学習面においては、
濁音等の答の選択枝にない音節は表示しないようになっている。また、
音節学習画面(濁音等を含む)
では、全ての音節を表示している。
文章学習においては、入力する文字数が
多くなるので、日本語入力ができるコンピュータでは、
Input Boxに直接ひらがなやローマ字を入力することができる
マルチインプット方式を導入している。
5. まとめ  
いつでも,どこでも,だれにでも利用できるバーチャル・ランゲージスクール
(仮想語学学校: Vertual Language School)として,
Javaを用いることにより,マルチプラットホームやマルチランゲージ等の面から,
オンデマンド・ネットワーク型日本語音声教育システム(LESSON/J)を簡単に構築でき,
Javaの有効性を示した。
今後,VRML(Vertial Reality Modeling Language)による3D技術を
融合することにより,よりゲーム性を高め,視覚化表現を高めることにより,
より使いやすい日本語音声教育システムに改善していかなければならない。
Java OSにより、音声言語教育のスタンドアローン携帯型専用機器の構築も可能である。
また、cgiではなくJavaプログラムを使っているため、
本ページのコンテンツのみをCD-ROMに格納するだけで、
独習用日本語音声学習システムとすることができる。
本システムでは、日本語音声教育システムを構築したが、
学習熱の高い英語や中国語の他に、
アイヌ語等のようにほろびる危機に面している言語等に、
本システムのコンセプトは利用可能と考えられる。
最近、文部省もオンデマンド英語教育システムの構築を目指していることを
平成9年8月27日付けの新聞が報じており、生涯学習でも、
いつでも、どこでも、だれにでもという本システムのコンセプトが必要である。
