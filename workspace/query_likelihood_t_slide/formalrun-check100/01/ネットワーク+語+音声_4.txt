マルチメディアパソコンとインターネットを用いて, いつでも, どこでも, だれでもが日本語音声教育を可能とすることを目指して, オンデマンド・ネットワーク型日本語音声教育システム(LESSON/J:Language Education System for Japanese Speech using On-demand and Network)を, WWW(http://sp.cis.iwate-u.ac.jp/sp/)上に構築した. このシステムでは, (1)オンラインで, 日本語の単音節, 単語, 文章の音声をスピーカから聴くことができ, また, (2)Java言語により, 独習向きに単語音声の聴取結果から書取テストをするディクテーション機能も実現しており, さらに, (3)CD-ROMを配布媒体としてオフラインで利用することもできる特色を持っている.
As progress of the technology of multimedia and Internet, a personal computer is a very useful equipment for spoken language education. We have developed the Language Education System for Japanese Speech using On-demand and Network (LESSON/J) with our WWW server (http://sp.cis.iwate-u.ac.jp/sp/). (1) Students for language education can hear Japanese speech of 50 syllables, 120 words and 30 basic sentences using the on-line system, (2) can learn Japanese words using the dictation system programmed with Java language (3) and can study Japanese language using speech contents in only CD-ROM for off-line mode for every time, everywhere and everybody.
