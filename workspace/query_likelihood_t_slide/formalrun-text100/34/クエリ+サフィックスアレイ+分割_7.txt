Provided is a method to provide a data structure of index information for performing pair character string retrieval at a high speed on a calculator, and to retrieve a pair character string which appears at close positions in a document at a high speed, using the index information. The suffix array bits of reference document data are rearranged, and index information (LSA) which can be localized, namely, can be used as an index for a partial region of the document, is created. Using the index information, halving processes of a region are repeated using the entirety of the document as the initial region, positions of the index information corresponding to a query character string in the reference document data are gradually detailed, distances in the pairs are assessed to narrow candidates, and finally, a position where the pair character string is proximally positioned is determined.
