P6 Web Accessがコンテンツリポジトリを使用するように設定されている場合、検索にアクセスしてドキュメントの全文検索を実行できます。 
プロジェクトワークスペースでは、すべてのプロジェクトドキュメントを検索できます。検索にアクセスするには、プロジェクトワークスペースのプロジェクトドキュメントポートレットにあるをクリックします。
ダッシュボードでは、すべてのプライベートドキュメント、およびアクセス権を持つすべてのプロジェクトドキュメントを検索できます。検索にアクセスするには、ダッシュボードのドキュメントポートレットにあるをクリックします。
ワークグループでは、そのワークグループに属するすべてのドキュメントと、その他すべてのプロジェクトドキュメントを検索できます。検索にアクセスするには、ワークグループのドキュメントポートレットにあるをクリックします。
キーワードと演算子を使用した基本的な検索を実行したり、ドキュメントのより複雑な検索クエリを作成して高度な検索を実行したりすることができます。 
ヒント:コンテンツリポジトリを設定してインストールしているユーザがP6 Web Accessに追加したプロジェクトドキュメントのみ検索できます。
検索を実行すると、検索条件と一致するすべてのドキュメントが表示されます。ドキュメントは検索条件と一致する順序でリストされます。検索条件に最も一致するドキュメントがリストの先頭に表示されます。
プロジェクトドキュメントの基本的な検索を行うには
プロジェクトワークスペースのプロジェクトドキュメントポートレットで、[検索]ドロップダウンリストから[検索]をクリックします。[ドキュメント検索]ダイアログボックスが表示されます。
テキストボックスに検索条件を入力します。最新バージョンのドキュメントのみを検索したり、検索結果で表示するアイテムの最大数を指定することができます。 
[検索のタイトルと内容]は、検索条件に対してドキュメントのタイトルと内容を検索します。 
[現行バージョンのみ検索]は、検索条件に対して現在のバージョンのドキュメントのみ検索します。 
[結果の制限(最大100)]を使用すると、検索中に返すドキュメントの数を制御できます。 
検索条件を選択してから、[検索]をクリックします。

プロジェクトドキュメントの高度な検索を行うには
プロジェクトワークスペースのプロジェクトドキュメントポートレットで、[検索]ドロップダウンリストから[検索]をクリックします。[ドキュメント検索]ダイアログボックスが表示されます。 
[上級]をクリックします。人物または日付で検索できます。
[作成者]:ドキュメントを作成または追加した人の名前を指定してドキュメントを検索します。P6 Web AccessまたはProgress Reporterのユーザのリストから選択するには、をクリックします。検索に含める各ユーザを選択してから、[追加]をクリックします。ユーザの選択が終了したら、[閉じる]をクリックします。
選択したユーザを削除するには、ユーザ名を選択してからをクリックします。複数のユーザを選択するには、Ctrlキーを押しながら各ユーザをクリックします。
[最終修正者]:ドキュメントを最後に更新した人の名前を指定してドキュメントを検索します。P6 Web AccessとProgress Reporterの
ユーザのリストから選択するには、をクリックします。検索に含める各ユーザを選択してから、[追加]をクリックします。ユーザの選択が終了したら、[閉じる]をクリックします。
選択したユーザを削除するには、ユーザ名を選択してからをクリックします。複数のユーザを選択するには、Ctrlキーを押しながら各ユーザをクリックします。
[作成日]:P6 Web Accessにドキュメントが作成または追加された日付を指定してドキュメントを検索します。検索する期間を選択するには、をクリックして開始日と終了日を選択します。
[修正日]:ドキュメントの最終更新日を指定してドキュメントを検索します。検索する期間を選択するには、をクリックして開始日と終了日を選択します。
表示される検索フィールドのリストから検索条件を選択します。検索条件をいくつでも選択して組み合わせて使用することができます。選択した各条件はAND文を構成します。
たとえば、[作成者]フィールドでユーザ、山田太郎を選択し、[作成日]フィールドで開始日に2004年6月3日、終了日に2004年6月5日を指定した場合、山田太郎さんによって追加され、かつ2004年6月3日から2004年6月5日の間に作成されたすべてのドキュメントが検索されます。 
ヒント:[高度検索]のオプションを使用する場合、最新バージョンのドキュメントのみを検索できます。  
検索結果をクリアするには 
プロジェクトワークスペースのプロジェクトドキュメントポートレットで、[検索]ドロップダウンメニューを選択します。
[検索結果のクリア]をクリックします。既存のドキュメントとフォルダがすべてプロジェクトドキュメントポートレットに表示されます。

[最近のプロジェクトドキュメント]リストのフォルダにプロジェクトドキュメントを追加するには 
ドキュメントポートレットに表示される[最近のプロジェクトドキュメント]リストのフォルダで、ドキュメントのタイトルを選択してからをクリックします。このフォルダにはプロジェクトドキュメントを50個まで追加できます。
注意:リストにはプロジェクトドキュメントのみを追加でき、ドキュメントはそれぞれ1回だけ追加できます。
