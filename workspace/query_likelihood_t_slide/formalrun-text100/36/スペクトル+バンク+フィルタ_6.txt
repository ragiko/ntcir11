周波数の利用効率を向上するために、時間・場所的に未利用の周波数を二次利用することで周波数の有効利用を図るダイナミックスペクトラムアクセスが注目されている。ダイナミックスペクトラムアクセスの実現には周波数・チャネルを柔軟に変更できるチャネルアクセス方式が必要になる。このため、OFDMAベースのダイナミックチャネルアクセス方式やフィルタバンクを用いたシングルキャリア変調方式によるダイナミックチャネルアクセス方式が提案されている。しかしOFDMA方式はマルチキャリア伝送方式であるためPAPR(Peak to Average Power Ratio)が大きい、フィルタバンクを用いたシングルキャリア変調方式ではチャネル毎にFFT/IFFTプロセッサが必要になるため、ハードウェア規模が大きいという問題がある。これを解決する方法として筆者らはオーバーラップFFTフィルタバンクを適用したスペクトル分割チャネルアクセス方式を提案している。本報告では、提案方式におけるPAPR特性を評価し、従来方式と比較した。
A dynamic spectrum access to efficiently utilize the unused frequency in a time or space has been drawing a great deal of attentions to improve the frequency of utilization efficiency. To implement dynamic spectrum access in a real system, a flexible dynamic channel access technique, which can change the bandwidth and a channel frequency flexibly, is required. An OFDMA based dynamic channel access scheme and a single carrier modulation based scheme using filter bank are proposed for dynamic channel access. However, the OFDMA based dynamic channel scheme has a drawback of large PAPR(Peak to Average Power Ratio) as it is a multi-carrier modulation technique, and the single carrier modulation based scheme using a filter bank also has a drawback of large amount of signal processing since per channel FFT/IFFT is required. To solve these problems, the authors have been proposing the spectrum splitting channel access scheme using overlap FFT filter-bank. This paper evaluates PAPR performance of the proposed scheme and compares it with the conventional schemes.
