英語のウインドウズを日本語化!! (Windows8 Pro編)
パソコン
やっぱり一番旬のWindowsということで、あまり使っていないのですが、Windows 8 Proを日本語化します。Windows7に慣れているので、ちょっと使いずらいのですが…。今、新品でパソコンを買うと全部基本的にWindows8がインストールされています。これから、買い替える予定の人にはWindows8以外に選択肢がありません。なので、これから解説していこうかなと思います。まず最初に、新品を手に入れ、電源を投入。アカウント設定等を済ませると…。このような画面になると思います。そうしたらすぐにマウスポインターを画面の右端コーナー(上でも下でも構いません。)へ持っていってください。そして、Settingsのアイコンをクリックしてください。そうすると上のような画面になります。そこで、一番右下の、Change PC settingsをクリックしましょう。画面全体が変わりました。左側のメニューの中のGeneralをクリックしましょう。そうすると右側の画面が上の写真のように変わります。その中のちょっと下の方にLanguageという項目があります。そのすぐ下の、Language preferencesという文字の上をクリックしましょう。そうすると、見慣れたWindows7のようなデスクトップ画面に変わって、上の写真のウインドウが広がります。その中の、Add a languageボタンをクリックしましょう。たくさんのいろいろな国の言語が出てきたと思います。それがアルファベット順に並んでいます。その中からJapaneseのJまで移動してください。そして、日本語を選んでAddを押してください。それか、日本語をダブルクリックでも構いません。そうすると、上の写真のような前の画面に戻ります。そして、日本語が増えていると思います。ここで大事なのが一番上に表示されている言語がウインドウズの標準言語になります。なので、このままでは完全な日本語表示にはなりません。日本語を一番上にしましょう。それには、日本語をクリックして選択して(写真のように色が変わります。)、Move upをクリックしてください。そうすると写真のように上に移動します。まだ、このままでは必要なファイルがインストールされていません。日本語関係のファイルをダウンロードしてインストールする必要があります。次に、日本語の右にあるOptionsをクリックしてください。上の写真のようなウインドウが開いたと思います。次にDownload and install language packをクリックしてください。上の写真のようなウインドウが開いてダウンロードが始まります。ダウンロードが完了すると自動的にインストールも始まります。(結構時間がかかります。終わるまで待ちましょう)インストールが終わると上の写真のようになります。あとはCloseを押してインストールの終了です。ここで、再起動の必要はありません。そのまま作業を続けてください。そして、次に左側にあるChange date, time, or number formatsをクリックしましょう。また画面が上の写真のように変わったと思います。その中の、Format:の下のボックスの中の、English(United States)をクリックしてください。そうするとたくさんの言語が出てくると思います。その中からJapaneseを選んでください。それで、時間や日付の表示が日本語になります。だんだん日本で買ったパソコンぽっくなってきましたね。次に、ウインドウの一番上にあるタブの中の、Administrativeをクリックしてください。上の写真のように変わったと思います。もうすぐ完了します。あと一息です。その中の、Welcome screen and new user accountsという上のボックスの中の、Copy settings...というボタンをクリックしてください。上のようなウインドウが開いたと思います。その中の一番下の、Copy your current settings to:の中の、二つの小さな四角のボタンをクリックしてチェックマークを付けてください。そうすると上のボックスの中の、Display language:と、Input language:が、上の写真のように、Japaneseに変わったと思います。そうしたら、一番下のボタンの、OKをクリックしてください。そうすると前の写真の画面に戻ると思います。今度は、Language for non-Unicode programsの中の、Current language for non-Unicode programs:がEnglish(United States)になっているので、これを、Japaneseに変更します。そのために、Change system locale...をクリックしてください。上の写真のウインドウが開いたと思います。その中の、Current system locale:の中のボックスをクリックしてください。たくさんの言語がずらずらと並んでいると思います。その中から、Japanese(Japan)を探してクリックしてください。上の写真のように変更されたと思います。そして、OKをクリックしてください。これで、すべての設定は終わりです。再起動すると日本で買ったものと同じようにすべてが日本語に切り替わったと思います。逆の手順ですべてを英語に戻せば、元の英語に戻ります。上の写真のように完全に日本語に切り替わりましたね。最後に設定をした、Current system locale:を日本語にし忘れたとしても、上の写真のように見た目は日本語になります。しかし、Current system locale:が英語のままだと、日本語のソフトをインストールすると、文字化けしてしまいます。これを忘れて、日本語のソフトをインストールして文字化けしてしまうと、とても大変なことが起こります。あとから、設定を変更して日本語に変えたとしても、文字化けは元に戻らず、アンインストールしてソフトを入れなおしても、完全に元に戻らない場合があります。最後で、気を抜かないで、ちゃんと最後まで、Current system locate:を日本語に変更してから、安心しましょう。(私はこれを忘れて、文字化けが直らずにWindows8を入れなおす羽目になってしまいました…)ここで終わってもいいのですが、最後に、もう一度同じ手順で、言語設定の画面にいってください。右端の下にいって、設定をクリックして、全般をクリックしてください。上の写真のようになったと思います。そして、言語設定を開きましょう。上の写真の中の、English (United States)をクリックして指定して、その上の、削除ボタンをクリックしましょう。これで、完璧に日本で買ったものと同じ状態になります。キーボードが英語仕様になっただけですね。日本のキーボードよりキーの数が少なくて、その分キー一つ一つが大きくなっています。日本のキーボードより打ちやすいと思います。私は英語のキーボードが大好きです。日本のキーボードは小さすぎる!(私の手が大きすぎるのか…)日本では売っていないノートブックを日本に持っていって、友達に自慢するのもありかな?長文を最後まで読んでいただいて、ありがとうございました。皆さんのお役に立てれば、これ幸いであります。間違っているところがあれば、教えてください。すぐに変更します。
