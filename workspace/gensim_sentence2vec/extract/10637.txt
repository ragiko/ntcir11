多くの顧客に送付する定型書簡や宛名ラベルのシートなどの文書のセットを作成する場合は、差し込み印刷を使用します。各レターおよびラベルには同じ種類の情報が印刷されますが、その内容は一意の内容です。たとえば、顧客への定型書簡では、各顧客の名前を挿入して各顧客宛ての定型書簡を作成できます。各定型書簡やラベルの一意の情報は、データ ファイルから挿入されます。 
差し込み印刷のプロセスには、次のような手順が含まれます。
メイン文書を設定します。メイン文書には、差し込み印刷後の各文書に共通のテキストおよびグラフィックが含まれます。たとえば、定型書簡の差出人住所やあいさつ文などです。
文書をデータ ファイルに接続します。データ ファイルは、文書に差し込まれる情報が保存されているファイルです。たとえば、定型書簡の宛先の名前や住所などです。 
宛先または項目のリストを絞り込みます。Microsoft Office Word では、データ ファイルの各項目、つまりレコードについて、メイン文書のコピーが生成されます。データ ファイルが宛先リストである場合、これらの項目は郵便物の受取人になります。データ ファイル内の特定の項目についてのみコピーを生成する場合は、含める項目 (レコード) を選択できます。
文書に差し込み印刷フィールドと呼ばれるプレースホルダを追加します。差し込み印刷を実行すると、差し込み印刷フィールドにデータ ファイルの情報が挿入されます。
差し込み印刷をプレビューして完了します。すべての文書を印刷する前に、メイン文書の各コピーをプレビューできます。
差し込み印刷を実行するには、[差し込み文書] タブのコマンドを使用します。
ヒント   [差し込み印刷] 作業ウィンドウを使用して、画面に表示されるプロセスの手順に従って差し込み印刷を実行することもできます。この作業ウィンドウを使用するには、[差し込み文書] タブの [差し込み印刷の開始] で [差し込み印刷の開始] をクリックし、[差し込み印刷ウィザード] をクリックします。

Word を起動します。 
既定では空の文書が開きます。この文書は開いたままにしておきます。この文書を閉じると、次の手順のコマンドを使用できません。
[差し込み文書] タブの [差し込み印刷の開始] で [差し込み印刷の開始] をクリックします。

定型書簡や電子メール メッセージ   すべてのレターまたはメッセージで基本的な内容は同じですが、それぞれのレターやメッセージには、名前や住所などの各受取人に固有の情報が含まれます。このような種類の文書を作成するには、[レター] または [電子メール メッセージ] をクリックします。

差し込みを再開するには、まず文書を開きます。

Outlook のユーザー プロファイルは、電子メール アカウントとアドレス帳のグループです。通常、必要なプロファイルは 1 つだけですが、コンピュータを他のユーザーと共有している場合や、目的別に複数のアドレス帳を使い分ける場合は、複数のプロファイルを設定できます。差し込み印刷を開始したときに、使用する連絡先フォルダを含まないユーザー プロファイルを使用している場合、そのフォルダを使用できません。
別のユーザー プロファイルに切り替えるには、Outlook を再起動します。プロファイルの選択を求めるメッセージが表示されたら、差し込み印刷で使用する連絡先フォルダが含まれているプロファイルを選択します。
複数のプロファイルが存在しているにもかかわらず、プロファイルの選択を求めるメッセージが表示されない場合は、Outlook の設定を変更する必要があります。そのためには、次の操作を行います。
Outlook を閉じます。

[プロファイルの表示] をクリックします。
Outlook を起動するたびにプロファイルの選択を求めるようにするには、[使用するプロファイルを選択する] をクリックし、[OK] をクリックします。
Outlook のヒントに戻る

[次へ] をクリックします。エクスポートするフィールドの横にあるチェック ボックスをオンにし、[完了] をクリックします。

ページの先頭へ宛先または項目のリストを絞り込む
特定のデータ ファイルに接続したときに、データ ファイルの一部のレコードの情報だけをメイン文書に差し込むことができます。
データ ファイルの宛先リストを絞り込んだり、項目のサブセットを使用するには、次の操作を行います。
[差し込み文書] タブの [差し込み印刷の開始] で [アドレス帳の編集] をクリックします。
[差し込み印刷の宛先] ダイアログ ボックスで、次のいずれかの操作を行います。
個々のレコードを選択する   この方法は、リストの項目が少ない場合に使用します。使用する宛先の横のチェック ボックスをオンにし、使用しない宛先の横のチェック ボックスをオフにします。 
差し込み印刷で数件のレコードのみを使用する場合は、見出し行のチェック ボックスをオフにして、必要なレコードのチェック ボックスのみをオンにします。同様に、リストのほとんどのレコードを使用する場合は、見出し行のチェック ボックスをオンにして、除外するレコードのチェック ボックスをオフにします。
レコードを並べ替える   並べ替えに使用する列の見出しをクリックします。この一覧はアルファベットの昇順 (A ～ Z) に並べ替えられます。一覧をアルファベットの降順 (Z ～ A) に並べ替えるには、もう一度列の見出しをクリックします。 
さらに複雑な並べ替えを行う場合は、[アドレス帳の絞り込み] の [並べ替え] をクリックし、[フィルタと並べ替え] ダイアログ ボックスの [レコードの並べ替え] タブで並べ替えの基本設定を選択します。たとえば、宛先の住所を、郵便番号の番号順に並べ替え、同じ郵便番号内では名前のアルファベット順に並べ替える場合に、このような並べ替えを使用します。
レコードをフィルタ処理する   この方法は、表示しないレコードや差し込み印刷に使用しないレコードがリスト内に含まれている場合に使用します。リストをフィルタ処理した後、レコードの横のチェック ボックスをオンまたはオフにし、差し込み印刷に使用するレコードを指定できます。
レコードをフィルタ処理するには、次の操作を行います。
[アドレス帳の絞り込み] で、[フィルタ] をクリックします。
[フィルタと並べ替え] ダイアログ ボックスの [レコードの比較] タブで、フィルタに使用する条件を選択します。
たとえば、国/地域がオーストラリアである宛先についてのみメイン文書のコピーを生成するには、[フィールド] ボックスの一覧の [国/地域] をクリックし、[比較] ボックスの一覧の [が値と等しい] をクリックし、[比較対象] ボックスの一覧の [Australia] をクリックします。
さらにフィルタを絞り込むには、[And] または [Or] をクリックし、条件を指定します。
たとえば、ミュンヘンにある企業についてのみメイン文書のコピーを生成するには、[市区町村] フィールドに "ミュンヘン" が含まれ、[会社名] フィールドが空白ではないレコードを抽出します。このフィルタで、[And] の代わりに [Or] を使用すると、ミュンヘンのすべての住所と、都市に関係なく会社名を含むすべての住所について差し込み印刷が実行されます。
メモ   アドレス確認ソフトウェアがインストールされている場合は、[差し込み印刷の宛先] ダイアログ ボックスの [住所の確認] をクリックして、宛先の住所を確認できます。
ページの先頭へ文書に差し込み印刷フィールドと呼ばれるプレースホルダを追加する
メイン文書をデータ ファイルに接続すると、文書の文字列を入力し、文書の各コピーに一意の情報を表示する場所を示すプレースホルダを追加できます。 
宛先やあいさつ文などのプレースホルダは差し込み印刷フィールドと呼ばれます。Word のフィールドは、選択したデータ ファイルの列の見出しに対応します。
メイン文書にフィールドを配置することによって、名前や宛先などの特定のカテゴリの情報を、その場所に表示することを指定します。 
メモ   メイン文書に差し込み印刷フィールドを挿入すると、フィールド名は常に二重の山かっこ ( ) で囲まれて表示されます。これらの山かっこは、差し込み印刷後の文書には表示されません。メイン文書で、フィールドと通常の文字列を区別するためにのみ使用されます。
差し込み印刷で実行される処理
差し込み印刷を実行すると、メイン文書のフィールドがデータ ファイルの最初の行の情報に置き換えられ、最初の差し込み印刷後の文書が作成されます。次に、フィールドがデータ ファイルの 2 番目の行の情報に置き換えられて 2 番目の差し込み印刷後の文書が作成され、これ以降も同様に処理されます。
フィールドの使用: 例
データ ファイルの任意の列見出しをフィールドとしてメイン文書に追加できます。これによって、定型書簡、ラベル、電子メール メッセージ、その他の差し込み印刷後の文書をデザインするときに柔軟に対応できます。次に例を示します。
地域の企業に今年度のシティ ガイドの掲載企業として選ばれたことを通知するレターを作成しているとします。データ ファイルに宛先の各企業の名前を含む [Company] 列がある場合は、個々の会社名を入力する代わりに Company フィールドを挿入できます。
四半期ごとに新製品や特別条件を顧客に通知する電子メール メッセージを送信しているとします。優良顧客向けに独自のメッセージにするために、「Miller 様、この新製品は以前にお探しになっていた商品です。」などのメモを入力できる [PersonalNote] 列をデータ ファイルに追加できます。メイン文書に PersonalNote フィールドを配置することによって、特定のメッセージの下部にこれらのメモを含めることができます。
宛先リストがニュースレターの購読者向けであり、データ ファイルに各購読の有効期限を保存する [ExpireDate] という列があるとします。ラベルのメイン文書に ExpireDate フィールドを配置して差し込み印刷を実行すると、購読者は宛名ラベルで購読の有効期限を確認できます。
フィールドを組み合わせたり、フィールドを句読点で区切ることができます。たとえば、住所を挿入するために、メイン文書で次のようにフィールドを設定できます。
First Name Last Name
Street Address
City, State Postal code
住所ブロックやあいさつ文など、頻繁に使用する項目については、複数のフィールドをグループ化する合成フィールドが Word に用意されています。次に例を示します。
住所ブロック フィールドは、名前、姓、町名、都市名、郵便番号などの複数のフィールドを組み合わせたものです。
あいさつ文フィールドには、選択したあいさつ文に応じて、1 つ以上の名前フィールドを含めることができます。
これらの合成フィールドの内容をカスタマイズすることができます。たとえば、宛先で "Mr. Joshua Randall Jr." のようにフォーマルな形式を使用したり、あいさつ文で "Dear" の代わりに "To" を使用したりできます。
差し込み印刷フィールドをデータ ファイルに対応付ける
Word がすべての住所またはあいさつ文要素に対応するデータ ファイル内の列を見つけられるようにするには、Word の差し込み印刷フィールドをデータ ファイルの列に対応付ける必要があります。
フィールドを対応付けるには、[差し込み文書] タブの [文章入力とフィールドの挿入] で [フィールドの対応] をクリックします。
[フィールドの対応] ダイアログ ボックスが表示されます。
宛先およびあいさつ文の要素は左側に表示されます。データ ファイルの列見出しは右側に表示されます。
Word により、それぞれの要素に一致する列が検索されます。この図では、データ ファイルの [名前] 列が [姓] に自動的に対応付けられています。しかし、Word は他の要素を対応付けることができませんでした。たとえば、Word はこのデータ ファイルから [名] を対応付けることができません。
右側のリストで、左側の要素に対応するデータ ファイルの列を選択できます。この図では、[名] 列が [名] に対応付けられています。[敬称]、[識別子]、[ミドル ネーム] には対応する列がなくても問題ありません。差し込み印刷文書ですべてのフィールドを使用する必要はありません。データ ファイルのデータを含まないフィールドを追加した場合は、差し込み印刷後の文書では空のプレースホルダ (通常は空白行または非表示フィールド) が表示されます。
