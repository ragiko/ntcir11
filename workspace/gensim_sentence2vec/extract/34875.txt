
動画や音声が再生されない
時々、Firefox で Web ページ上の動画や音声が表示されない場合があります。その場合、必要なプラグインがインストールされていない、バージョンが古い、または他のプラグインや拡張機能が干渉している、コンテンツが何らかの理由でブロックされている可能性があります。この記事では、これらの問題を解決する方法を紹介します。
必要なプラグインをインストールする
Firefox は、画像と Ogg や WebM 動画などの オープンメディア、Web ページ上のいくつかのメディアを表示することができます。しかし、動画や音声などの一部のコンテンツでは、他のメディアプレーヤーやブラウザプラグインなどをインストールする必要があります。動画や音声を再生するために必要なメディアプレーヤーがインストールされている場合でも、 Web ページ自体にメディアが埋め込まれている場合、適切なプラグインがインストールされている必要があります。
プラグインがインストールされていない場合、以下のことを行ってください:
ダウンロードリンクをクリックしてください。  
以下の手順に従ってください。
インストールに失敗する場合は、手動でインストールしてください。
Flash プラグイン: Flash プラグインをインストールする および Flash プラグイン利用ガイド の記事をご覧ください。
Flash プラグインは、YouTube.com の動画を含め、様々な種類の埋め込まれた音声や動画を再生するために使用されます。
Java プラグイン: Java プラグイン利用ガイド の記事をご覧ください。
Silverlight プラグイン: Silverlight プラグイン利用ガイド の記事をご覧ください。
Windows Media Player プラグイン: Windows Media Player プラグイン利用ガイド の記事をご覧ください。
Windows Media Player プラグインは、Web ページが Firefox や他の Mozilla ブラウザ向けに書かれていない、特定の Windows メディア形式 (.asf, .asx, .wm, .wma, .wax, .wmp, .wmv, .wvx) のみを再生できます。Internet Explorer は、mp3 や MIDI など多くの他の形式の再生に Windows Media Player を使用しますが、Firefox でこれらを再生するには QuickTime など他のプラグインが必要です。
QuickTime プラグイン: QuickTime プラグイン利用ガイド の記事をご覧ください。
QuickTime プラグインは、埋め込まれた多くの種類のメディアを扱えます。Windows XP や Mac OS X 10.5 とそれ以前の古いオペレーティングシステムでは、埋め込まれた mp3 オーディオなど、いくつかのファイル形式を扱うように QuickTime プラグインが設定されていないかもしれません。この場合は、Firefox 内で QuickTime に再生させたいメディア形式を QuickTime の環境設定で選択する必要があります。
他の一般的でない形式のメディアプレーヤーとそのプラグインについては、以下の Web サイトをご覧ください:
Shockwave
RealPlayer
VLC
プラグインを更新する
すべてのインストールされたプラグインが最新バージョンか確認してください。
プラグインチェック ページで各リンクから古いバージョンのプラグインを更新してください。
インターネットセキュリティソフトウェアで plugin-container を許可する
ご使用のインターネットセキュリティソフトウェア (ファイアウォール、ウイルス対策ソフトウェア、スパイウェア対策ソフトウェアなど) で、plugin-container.exe を許可してください。詳細は ファイアウォールの設定 の記事をご覧ください。この記事での Firefox は、Firefox の plugin-container を意味しています。
干渉しているプラグインを無効にする
同じ メディア形式 を扱えるプラグインが複数インストールされていると、一部の埋め込まれたメディアの再生が妨げられる原因になります。この問題は、干渉しているプラグインを無効にすることによって解決できます。

プラグインのリストから、問題を起こしているプラグインを選択してください。
プラグインを無効にしたいときは、無効化 ボタンをクリックします。
プラグインを再び有効にしたいときは、有効化 ボタンをクリックします。

例えば、埋め込まれた QuickTime コンテンツや Windows Media コンテンツを再生するために、VLC プラグインを無効にする必要があるかもしれません。プラグインが原因の問題を解決するための詳しい情報については、プラグインのトラブルシューティング の記事をご覧ください。
他の解決方法
キャッシュを消去する
Firefox のキャッシュを消去するには の記事をご覧ください。 
干渉している拡張機能や広告ブロックソフトウェアを無効化する
広告ブロックソフトウェアやインターネットセキュリティソフトウェアの広告ブロック機能、Flashblock や Adblock Plus などの拡張機能などが、音声や動画の再生を妨げることがあります。これらを無効化した状態でサイトの動画が正常に見られる場合は、このサイトを広告ブロックフィルタのホワイトリストに追加するとよいでしょう。
他の Firefox の拡張機能もプラグインをブロックする場合があります。すべての拡張機能を無効化して動画や音声が再生できるようになった場合は、インストールされたいずれかの拡張機能が問題の原因となっています。詳しくは 拡張機能とテーマのトラブルシューティング の記事をご覧ください。
