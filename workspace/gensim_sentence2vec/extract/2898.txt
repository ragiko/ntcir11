リリースシステムを使用しています.sakura.exeなど個別ファイルのダウンロードは SoruceForge Project Page からどうぞ.
サクラエディタはあとで外部ツールを追加して機能拡張することができます.それら関連ファイルを含めた導入方法についてはこちらをご覧ください.
関連ファイルをまとめてダウンロード&インストールするSakura
Downもあります.
パッケージダウンロード
本サイトの更新は遅れることがあります.パッケージの更新状況は SoruceForge Project Page でご確認ください.
注意: パッケージが最新版の実行ファイルを含んでいないことがあります.新規にインストールされる場合にはパッケージをインストールした後でsakura.exeのみを最新版EXEと差し替えることをおすすめします.
V2(Unicode版)
サクラエディタの Version 2 です.内部データを Unicode で保持し、SJIS で表現できない文字も扱えます.日本語版 Windows 2000/XP/Vista/7/8 で動作します.
最新版ダウンロード sakura_install2-1-1-4.exe (2.1.1.3)
主な新規機能
詳細は更新履歴をご確認ください.
V1(ANSI版)
最新版ダウンロード sinst1-6-6-0.exe (1.6.6.0)バグフィックスが中心のリリースです
詳細は更新履歴をご確認ください.
拡張ツール
エディタの機能のうちのいくつかは外部ツールを必要とします。バイナ
リとは別に以下のツールも併せてダウンロードしてください。
bregonig.dll
正規表現検索・置換および正規表現キーワード強調表示に必要で
す.(インストーラパッケージには含まれています)
V1(ANSI版)ではBREGEXP.DLLも使用可能です.
GNU diff 中のdiff.exe
Diff機能に必要です
Exuberant
Ctags 日本語対応版 中のctags.exe
ダイレクトタグジャンプのtagsファイル生成に必要です.
PPA.DLL
PPAマクロの実行に必要です.
C/Migemo
のmigemo.dll 及びdict/ディレクトリ以下の辞書ファイル
Migemoを使ったインクリメンタルサーチで必要です.置き場所は共通設定から変更することも出来ます.
いずれもsakura.exeと同じディレクトリに置いてください.(migemoの場合はdictディレクトリも)
実行ファイルのみダウンロード
コンパイル済み実行ファイルはSourceForge
Project Folderのリリースシステムに登録されています.更新履歴も
併せてどうぞ.
インストーラにてインストールされたものをこのファイルで上書きして
も何ら問題ありません.
ソースファイルのダウンロード
リリース版のソースコード一式はSourceForgeに
て公開されます。開発中のファイルはリポジトリより直接参照が可能です.
注意
ソースコードのサクラエディタ以外での取り扱いについては各Copyright
Holderに個別に問い合わせてもらうことになります。
なお、zlib/libpngライセンスに従った配布が可能と書かれ
ているファイルについてはその通りですので、ライセンスの範囲内で自由に転用していただいてかまいません。
ヘルプファイルのみダウンロード
ヘルプファイルの単体配布はSourceForgeに
て行っています。
