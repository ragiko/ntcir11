
隨著智慧型手機慢慢的在生活中有了一席之地
漸漸的越來越不能沒有它的存在
慢慢的所使用應用程式日益肥大
接著內部儲存就會有不夠的情形發生
不過比較新出來的機子都有幾十GB以上的容量
所以比較沒那麼擔心…
但是個人比較寒酸用不起那麼好的手機
只好…(遭毆)
在這提醒SD記憶卡內的資料請先備份到電腦中
本文教學中會將SD記憶卡格式化做分割
以上共勉之~
SD Formatter 4.0 格式化軟體SD/SDHC/SDXC記憶卡
SDFormatter : www.sdcard.org/cht/downloads/formatter_4
MiniTool Partition Wizard Home Edition
MiniTool Partition Wizard : www.partitionwizard.com/free-partition-manager.html
Link2SD
Link2SD : play.google.com/store/apps/details?id=com.buak.Link2SD
首先將SD卡插入讀卡機並連接電腦開始做格式化準備
開啟 SD Formatter 後,選擇磁碟代號(如:E)
並確認好所選擇的是SD記憶卡並檢查容量
接著按 “選項設定”
格式化選項設定
消除設定選擇 “快速格式化”
邏輯大小調整 “開啟(ON)”
選擇完畢後按下 “確定”
再按下 “格式化”
這時出現警告提醒視窗,點選 “確定”
這時也一樣出現警告提醒視窗,點選 “確定”
接著就開始格式化了~
完成後會出現上圖視窗
按下 “確定” 後再按 “完成”
接著開啟 MiniTool Partition Wizard
並將SD記憶卡與電腦連接
這時就會看到SD記憶卡的磁碟代號(如:E)
並點選(E) 後在上方功能中按下 “Delete”
這時你會看到原本的(E) 變成了(*)
上方磁碟容量顯示條的部分,變成了灰色
接著點選(*) 後在上方功能中按下 “Create”
這次我使用的SD記憶卡為16GB,實際上使用只有14.83GB
所以我將分成一半,一個做FAT32用來儲存
另一個做Ext4用來給Link2SD使用
這裡所要設定有 Create As 選擇 “Primary”
File System 選擇 “FAT32″
下方容量部分依照自己喜好去調整
這裡我直接切成一半 “7.42GB”
完成上面設定後按下 “OK”
這時就會看到上圖 (E) 切好了 7.41GB
接著點選(*)後再上方功能中按下 “Create”
會出現提醒警告視窗 按下”Yes” 就能繼續調整剛剩餘一半的空間
這裡所要設定有 Create As 選擇 “Primary”
File System 選擇 “Ext4″
容量部分就不用在去做調整了,下方直接點選 “OK”
這裡設定部分已經大致上已經告一段落了
接著在上方功能中按下 “Apply”
按下 “Apply” 後會出現提醒視窗,確認沒問題後按下 “Yes”
如果對於上面之前的設定有所疑慮,就按下 “No” 回頭檢查看看
按下 “Yes” 後就會開始跑剛剛的分割設定
跑完後會出現完成視窗,按下 “OK”
SD記憶卡分割Ext4在這告一段落
接著將SD記憶卡插回手機上
開始用Link2SD做連結
在PLAY商店安裝好Link2SD (手機必須要root權限)
在右上角選項中按下 “多選模式”
接著將所需要扔到SD記憶卡中Ext4空間內的打勾
勾選完畢後,右上角點選選項 “建立連結”
這裡依照個人喜好
至於最後一個選項就別問我了
PLUS版本才有,更何況我這是…恩~不多說了
勾選好後按下 “確定”
接著就開始做連結了~
完成後按下 “確定”
在儲存裝置資訊中可以看到
原先在內存的容量少了許多
可以使用的容量變多了~
原先的檔案都移到了SD卡第二分區
最後提醒,一些像系統的應用(如:鬧鐘等..)
就別移動做連結了
哪怕一不小心…恩?你懂得~
