ダンサーズコレクションによくある質問
Q.1 : 最低何枚、いくつからオリジナル商品を作れるの??
Tシャツ、キャップ、マグカップなどの小物に至るまで1枚、お1つからお作りできます。
Q.2 : 納期はどれくらい??
店頭であれば、簡単なデザインなら15分〜即日でお持ち帰りが可能です。
通販であれば、即日発送、関東近郊であれば最短で翌日、及び翌々日にはお客様の手元に着という事も可能です。
※こちらはクロネコヤマトさんのシステムと提携しておりますので、当店に日時、料金などをお気軽にお問合せ下さい。
Q.3 : 自分で作ったデザインの持込は可能??
はい、可能でございます。
当店ではより高いプリントのクオリティーを出す為にイラストレータを使用しプリント業務に連結させております。
お客様の方でイラストレーターでデータ入稿して頂ければスムーズに対応できます。
※またイラストレーターのデータでお持ちでない場合は当店で編集も可能でございます。
Q.4 : パソコン持ってないけど、オリジナルデザインを作りたい。
はい、可能でございます。
当店ではお客様のご希望のデザインをデザイナーチームでゼロから作成致します。
Q.5 : オリジナルプリント、オリジナル刺繍の通信販売はできるの??
はい、可能でございます。
南は沖縄、北は北海道まで全国対応しております。
※一部地域で対応できない場合もございますので、事前にご質問して頂ければ対応致します。
Q.6 : 通信販売の送料はいくら??
こちらは料金表でご確認頂けます。
こちらをご覧下さい。
Q.7 : オリジナルプリント、刺繍で5000円以上でも送料は無料??
はい、もちろん無料でございます。
※その他の商品をミックスしてお買上げ頂いても5000円以上であれば無料でございます。
Q.8 : 自分で購入した商品に持込でプリントや刺繍はできるの??
はい、可能でございます。
当店ではお客様ご自身でご購入頂いた商品などの持ち込みオーダーサービスも対応致します。
※但し商品の素材やプリント、刺繍箇所によってはお断りする場合がありますので事前にお問合せ下さい。
※別途持込料金が1点につき500円(税別)でかかります。
Q.9 : 100枚以上、1000枚などの大量オーダーもできるの??
はい、可能でございます。
お客様ご希望の商品が1点からお作りできるのもダンサーズコレクションの特徴ですが、文化祭やコンサートなどの大量オーダーの受注も製作も可能です。
お客様のご予算と納期に合わせてお作りさせて頂いております。
Q.10 : 土日、祝日も通販できるの??
はい、可能でございます。
通信販売、店頭と年末年始を除く全ての営業時間でスタッフが対応しております。
Q.11 : 以前作った商品を再度つくれるの?
はい、可能でございます。
当店ではお客様のデータは大切に保管しており、お客様のバックオーダーに迅速に対応できるように管理しております。
データ保存料は無料サービスしております。
Q.12 : プリントや刺繍で他のお客様の作った商品と同じものを作りたい。
いいえ、できかねます。
他のお客様がお作りしたデザインは大切に保管している為、他のお客様に同じ商品を提供する事はお断りさせて頂いております。
但し、家族やお友達や先輩、後輩などで、ご本人様の許可を頂きその承諾を当店で確認ができればお作りする事は可能です。
Q.13 : 当店スタッフに以前頼んだ同じデザイナーにお願いしたい。
はい、可能でございます。
通販、及び当店でも可能です。
スタッフ、デザイナーの名前をお伝えして頂ければ、お客様のご指名のスタッフへとお繋ぎ致します。
もちろんご指名料金などは御座いませんので無料です。
Q.14 : 営業時間は??
12時〜21時までとなっております。
Q.15 : 定休日はあるの??
年末年始、及び特例のイベントや改装工事など以外は平日はもちろんの事、土日、祝日も営業致しております。
Q.16 : 店頭に並んでいない商品でもお取り寄せは可能??
はい、もちろん可能でございます。
当店にてお取り扱いの商品以外でも、当店から発注やオーダーも可能です。
Q.17 : 店頭レッドブルを一緒に買いたい。
いいえ、できかねます。
店頭でのレッドブルはお客様に無料にてサンプリングさせて頂いており基本、販売等行っておりません。
また、サンプリングの際は転売防止の為、必ずプルトップを開けた常態でお配りさせて頂きます。
※サンプリングは店頭のみのサービスで通販はできかねます。
※こちらはレッドブルジャパン 株式会社 様との契約のもと、正規サンプリング店として業務を行っております。
Q.18 : イベントフライヤー、ポスターなど店頭に設置したい。
はい、可能でございます。
店頭に直接お持ち頂くか、郵送でも対応致します。
※但し無許可での郵送、設置、また、イベントの内容などによって当店や一般基準のモラルに反するイベントなどは厳重に品質管理させて頂きますので、
当店の判断の元お断り、破棄せて頂く場合がございますので予めご了承下さいませ。
Q.19 : ダンサーズコレクションの商品を卸販売してほしい。
はい、可能でございます。
当店では都内はもちろん、地方の小売業者様、ダンススタジオ様、ECサイトなどの経営者様、などの業者様向けに卸販売もしております。
卸販売の方もお気軽にお問合せ下さい。
Q.20 : ダンサーズコレクションで働きたい。求人募集はしてるの??
はい、一般求人募集もしております。
但し、時期やタイミングで募集をしていない場合がありますので当店にお電話かご来店、メールなどでもお問合せ可能です。
