 
座標変換とはなんだろうか?
数学的には、座標変換は次のように規定される。
座標変換とは、任意のベクトルを、その長さを変えないように変換する変換のことである
これは、次の定義と同値である。
座標変換は、内積を不変に保つ変換である
そのまま受け入れることは簡単だが、あえてこれらの定義の意味するものを考えてみよう。
座標変換とは、ものの見方を変えることである。
見方を変えるので、当然もの自体が変わってしまっては成り立たない。
ゆえに、座標変換はある"もの"を不変に保つ変換であるといえる。
上ではその"もの"としてはベクトルの長さ(or内積)をとったわけだが、なぜそのようにするのだろうか。
これが納得しにくいのは、ベクトルという言葉が入っているからだ。
ベクトルの長さを不変にする、という部分を、2点間の距離を不変にする、というように読み替えれば相当わかりやすくなると思われる。実際、ある変換で2点間の距離がころころ変わってしまうようでは、それは見方を変換しているのではなくて、空間自体を変換しているとい
わざるを得ない。
つまり、内積を不変にするという条件は、
ある変換が座標変換であるための必要条件
なのである。
さて、十分条件を考慮する必要はあるのだろうか?今のところ、それはないと考えられる。
なぜなら、必要条件だけで困るという状況が発生するとすれば、"座標変換"に分類される変換が多すぎて、分類自体が意味を成さなくなってしまうという状況だからだ。しかし、"座標変換の理論"の立場からすれば、この理論の適用対象が大きければ大きいほどいいのだから、わざわざ座標変換を十分条件の側から制限する必要はないのである。
以上より、座標変換の定義
座標変換は、距離を不変に保つ変換である
が妥当なことが確認された。
普通座標変換といえば3次元中の座標変換をさし、物理的に意味があるのもこのような変換である(曲座標や球座標への変換もある)。
しかし、光速が不変であるというアインシュタインの発見により、時間軸を加えた4次元中の座標変換が必要とされるようになった。これが、Lorentz変換(ローレンツ変換)と呼ばれる変換である。4次元中の座標変換の理論を、相対性理論という。
これから、特に扱うベクトル空間のバリエーションが増えない限りは、この2つの座標変換に限って理解しておけばよいだろう。
3次元中の座標変換
3次元の中で、2地点P(x,y,z),Q(x',y',z')の距離PQは、次のように定義される。
PQ = {(x'-x)^2 + (y'-y)^2 + (z'-z)^2}^0.5
これを不変にするような変換を求める。
まず、この変換が1次変換でなければいけないことはどうやってわかるだろうか?
それには、このように考えればよい。
まず、ある座標変換があるとする。その逆変換も、座標変換である。これらは属する種類の変換に固有で、共通の性質を持たなければいけない。
座標変換を行い、別の座標変換を行ったものもまた座標変換である。この操作によって、"共通の性質"は、2回行われている。しかし、結果は"共通の性質"を1回しか行っていない場合と変わらない。
また、ある座標変換は2つの適当な座標変換の合成に分解することができる。分解する前は"共通の性質"は1回しか行われていないのに、分解したあとは"共通の性質"は2回行われている。それなのに、結果はやはり変わらない。
これらのことから、座標の次数を変える変換(y=x^2など)はありえないということがわかる。もし座標変換としてy=x^2などをとれば、2つの変換の合成に分解したときにy=x^4となってしまい、矛盾するからだ。
よって座標変換は次数を保つ変換、1次変換でなければならないことがわかった。つまり、
座標変換は行列で記述できる変換
なのである。
ただし例外(というほどでもないが)がある。
例外:平行移動は次数を保ち、距離も不変にするので、座標変換である。
ただし、これは3次元の行列では記述できないということであって、4次元の行列で完全に記述できるので心配はないだろう。
ちなみに平行移動を考慮する場合、アフィン変換と呼ぶ。
上で言ったことをまとめると、
原点を固定する等距離変換は、線型変換である
ということになり、これは数学的に厳密に証明できる。が、ここでは証明を省いた方が得策だ。後で触れよう。
さて、座標変換は実際にどのような行列になるのだろうか?
問い:a = (a1,a2,a3),b = 
(b1,b2,b2)とする。座標変換の行列をA(成分aij)とする。内積を不変に保つとは、
Aa・Ab = a・b
となることであるが、これが
A(tA) = E (Eは単位行列)
に同値であること、すなわち等距離変換の行列Aが直交行列であることを証明せよ。
答え:省略(証明できればそれが答えであることは自明)
この単純な結果が上のような簡単な過程から厳密に導かれるというのは確かに面白い。興味のある人には、「線型代数と群の表現(Ⅰ) 
平井武著」の、7章でこれを厳密にやっているのでお勧めしておく。この本には、原点を固定する等距離変換が線型変換であることの証明も載っている。
ここでの結論:3次元中の座標変換は、直交行列なのだ。
4次元中の座標変換
相対性理論では、3次元空間に時間軸を加え、4次元空間とし、次のような内積<a,b>を与えたものをミンコフスキー空間という。
a,bについて (ct,x,y,z) : a 
=(a0,a1,a2,a3), b = (b0,b1,b2,b3) として、内積は
<a,b> = - a0b0 + a1b1 + a2b2 + a3b3
このように内積を与え、ある点の原点との距離は、その点の位置ベクトルの自分自身との内積の0.5乗とする。
この内積は普通のベクトル空間とは異なるが、これによって、どのようなことがわかるか。
ミンコフスキー空間では、今自分自身のいる点が原点である。
原点からの距離が正となる点は、自分自身がこれから存在できる可能性のある全ての点となる。これらの点を、「空間的」であるという。
原点からの距離が負となる点は、自分自身がこれから存在することが不可能である全ての点となる。これらの点を、「時間的」であるという。(時間的な隔たりがあるという意味であろう)
4次元中の座標変換は、この内積を不変にする。
つまり、座標変換をしても、ある点にこれから行けることは変わりなく、別のある点にこれからいけないことも変わりないのだ。
それはそうだ。座標変換したぐらいで、ある時刻の決まった場所にいけるかいけないかが変わってしまっては困る。事象が起こるかどうかが座標の取り方で決まってしまっては大変だ。
ちなみにこの条件(空間的・時間的)は、
全ての物体は光より速く動くことはできない
という条件の言い換えとして導かれる。
4次元中の座標変換を総じてLorentz変換と呼ぶ。相対性理論の神秘性が少し薄れてしまったかもしれない。
Lorentz変換も行列である。それでは、どのような行列になるか見てみよう。といっても自分で・・・。
問い:Lorentz変換Aは明らかに4×4行列である。そのi行j列(i,j=0,1,2,3)の成分をaijと書こう。内積不変の条件は
Ab・Ab = b・b
である。これと同値な条件を求めよ。すると、Lorentz変換はいくつの自由変数を持つか?
答え:これを展開すれば次のような面倒な式になる。
(a00b0 + a01b1 + a02b2 + a03b3)^2 + (a10b0 + a11b1 + a12b2 + 
a13b3)^2
+ (a20b0 + a21b1 + a22b2 + a23b3)^2 + (a30b0 + a31b1 + a32b2 + 
a33b3)^2 = b0^2 + b1^2 + b2^2 + b3^2
b0,b1,b2,b3は変数なので、恒等式の知識より、両辺の同じ変数の項の係数は等しくなければならない。よって、
b0b0より a00^2 + a10^2 + a20^2 + a30^2 = 1
b0b1より 2a00a01 + 2a10a11 + 2a20a21 + 2a30a31 = 0 (両辺2で割れる)
b0b2より 2a00a02 + 2a10a12 + 2a20a22 + 2a30a32 = 0
b0b3より 2a00a03 + 2a10a13 + 2a20a23 + 2a30a33 = 0
b1b1より a01^2 + a11^2 + a21^2 + a31^2 = 1
b1b2より 2a01a02 + 2a11a12 + 2a21a22 + 2a31a32 = 0
b1b3より 2a01a03 + 2a11a13 + 2a21a23 + 2a31a33 = 0
b2b2より a02^2 + a12^2 + a22^2 + a32^2 = 1
b2b3より 2a02a03 + 2a12a13 + 2a22a23 + 2a32a33 = 0
b3b3より a03^2 + a13^2 + a23^2 + a33^2 = 1
16個の変数に対し条件式は10個であるので、Lorentz変換は6個の自由変数を持つとわかる。//
この6という数字はとても合理的であることがわかるだろうか。なぜなら、Lorentz変換を特徴付ける量は、
観測系の速度成分vx,vy,vz、座標系の取り方の3変数
の計6個であるからだ。
問い:観測系では、必ずしも別の完成系と同一の方向を持つ直交座標を使っているとは限らない(回転している可能性がある)ので、速度ベクトルの方向を新しいx軸として、3つの変数をうまく設定すれば座標の置き方をただ一通りに決めることができる。座標系の取り方の3変数の一例を求めよ。
答え:原点を通り、新しいx軸に垂直な平面中で、y軸の方向を決めるために、y軸の方向ベクトルの3成分が必要である。z軸は、xyzが右手系をなすように取れば自動的に決まる。方向ベクトルの長さが1、という束縛条件があるので、Lorentz変換となるものも何らかの束縛を受けていることがわかる。
実は、数学がわかれば、相対性理論は赤子の手をひねるように理解してしまえるのだ。
