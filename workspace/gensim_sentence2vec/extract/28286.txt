
SDL Language Cloud Translation Toolkit とは
SDL Language Cloud Translation Toolkit は、機械翻訳と通常翻訳サービスを独自の形で組み合わせたクラウド翻訳プラットフォームで、どなたでもシンプルな翻訳を提供する API を介して簡単にアクセスできます。 SDL では、SDL Tridion や SDL Trados Studio などいくつかの他の SDL 製品を通じて SDL Language Cloud との統合を提供しています。
対象ユーザーはどのようなユーザーですか?
翻訳者およびプロジェクト マネージャ: 事前翻訳や機械翻訳コンテンツのポストエディットに SDL Trados Studio 2014 を使用。
コンテンツ所有者: 文書の翻訳が必要で SDL Language Cloud と統合する製品を使用。
ソフトウェア開発者と組織: SDL Language Cloud API を使用して通常翻訳と機械翻訳を既存のアプリケーションに直接統合して使用。今すぐお試しください。
SDL Language Cloud ではどのような種類の機械翻訳を提供していますか?
SDL Language Cloud では、無料翻訳のほか、次のような便利な購読ベースのサービスにアクセスが可能です。
機械翻訳にアクセス。
SDL が整備した業界固有の機械翻訳エンジンにアクセスして、特定の業界向けの翻訳の品質向上が図れます。
辞書を管理および使用することで品質が向上し、個別にカスタマイズされた機械翻訳出力が得られます。

SDL Trados Studio 2014 ユーザーはどのようにして SDL Language Cloud の機械翻訳サービスにアクセスできますか?
SDL Language Cloud の機械翻訳は、SDL Trados Studio 2014 のユーザー インターフェイスから直接アクセス可能です。 次の 3 つの簡単なステップで接続します。
SDL Language Cloud のアカウントを、SDL ID の電子メール アドレスとパスワードを使用して作成します。 これまでに SDL Language Cloud を利用されたことのない場合は、30 日間の無料試用版が自動的に開始します。SDL の業界固有エンジンの優れた機能をお試しいただくよい機会です。
試用版がアクティブ化されたら、新しい SDL Language Cloud 機械翻訳プロバイダを SDL Trados Studio 2014 のプロジェクト設定からプロジェクトに追加できます。
API とは何ですか? SDL Language Cloud Translation Toolkit を利用するのに要する時間は?
API (アプリケーション プログラミング インターフェイス) はアプリケーション間で通信をするための方法です。 SDL Language Cloud Translation Toolkit を使用することで開発者は簡単にアプリケーションとサービスを接続でき、発注、プロセスの管理、および SDL Language Cloud からの翻訳結果の取得が行えます。 開発者は 1 日もかからずに API に接続して、SDL とともにコンテンツの翻訳を開始できます。

SDL Language Cloud はどのように機能するのですか?
開発者は次の簡単な 3 ステップで接続できます。
開発者アカウントに登録すると、サンドボックス環境にアクセスして API を入手できます。
登録が済むと、開発者は SDL Language Cloud Translation Toolkit に統合するコードを作成して、サンドボックス環境でテストが行えます。
統合が機能し、組織のアプリケーションや Web サービスの翻訳に必要な機能が得られることが確認できたら、開発者は API を本稼働環境に配置できます。

SDL Language Cloud と従来の翻訳サービスやオンライン翻訳サービスとの違いについて。
SDL Language Cloud は、Web または API インターフェイスのいずれかを通じて機械翻訳と通常翻訳を提供する最初の完全なクラウド翻訳プラットフォームとして、お客様の翻訳予算に対して最高の成果をもたらします。 企業のお客様は、いつでも適切な翻訳品質と価格でコンテンツを翻訳でき、翻訳予算に対して最高の結果が得られます。
SDL とはどんな会社ですか?
SDL は 20 年以上にわたりプロフェッショナル翻訳と言語テクノロジーをリードし続けています (SDL.com) 。 世界の数千の企業が、高品質のローカライズ コンテンツを提供し、カスタマー エクスペリエンス マネジメントを向上し、そしてさまざまな言語を通じたグローバルなコミュニケーションを可能にする SDL の翻訳ソリューションを信頼しています。 SDL Language Cloud は、SDL の世界水準の翻訳技術と機能を、クラウド ベースのアクセスしやすい安全で拡張性に優れたプラットフォームで提供します。
