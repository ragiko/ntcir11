Stash のインターフェイスは、管理業務を簡単なものにします。Stashは、安全なファイアウォール中で大規模化する Git リポジトリをサポートします。  UI から直接、ユーザやグループの追加、そして詳細な許可のデリゲートが可能。Stash のプロジェクト構造を用いて、リポジトリの作成・管理が可能です。そして、ブランチ一覧ページを使用すればチームの全ブランチの最新情報を、ものの数秒で取得できます。 11 ユーザーとグループの追加や細かなパーミッション付与を UI から簡単に行うことができます。Stash が実現するプロジェクト構造の下でリポジトリの作成や管理が可能です。ブランチリストのページではチームのすべてのブランチの最新状況が確認できます &ndash; すべてが瞬時に可能です。
