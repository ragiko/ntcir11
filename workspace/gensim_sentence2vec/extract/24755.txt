コマンド拡張機能が有効な場合 (既定) 値を設定を実行、すべての値で始まる変数が表示されます。

環境変数の使用
バッチ ファイルやプログラムの動作を制御するための環境変数を使用し、Windows と MS-DOS を制御するのにはサブシステムの外観し機能します。
環境変数を設定するのには、Autoexec.nt ファイルで、多くの場合、 setコマンドが使用されます。
現在の環境設定を表示します。
Set
Set
コマンドを単独で入力すると、現在の環境設定が表示されます。
これらの設定には、ディスク上のプログラムを見つけるために使用されます、COMSPEC および PATH 環境変数、通常が含まれます。
Windows で使用される 2 つの環境変数をプロンプトおよび DIRCMD です。
パラメーターを使用してください。
