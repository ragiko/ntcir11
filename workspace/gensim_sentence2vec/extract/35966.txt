3. 店舗統合に関するお知らせ 
( 1 ) 統合後のお取引店と口座番号のお取り扱い
支店名・支店番号は変わりますが、口座番号は原則として変更はなく、
従来の口座番号をそのままご使用いただくこととなります。
変更になる場合は、当金庫から別途個別にご案内申し上げます。
( 2 ) 通帳・証書のお取り扱い
現在ご使用の通帳・証書は、引き続きご利用いただけます。
( 3 ) 出資証券のお取り扱い
出資証券の切り替えは必要ありませんので、そのままお持ちください。
( 4 ) キャッシュカード・ローンカードご利用のお客さまへ
現在ご使用のキャッシュカードおよびローンカードは、引き続きご利用いただけます。
( 5 ) 当座預金ご利用のお客さまへ
①
お手元の手形・小切手帳は統合日以降に、統合先支店扱いの新しい手形・小切手帳と差し替えさせていただきます。
②
統合日以降、旧支店を支払場所とする手形・小切手が支払い呈示された場合は、
統合先支店の新口座からお支払いさせていただきます。
( 6 ) 公共料金等の口座振替をご利用のお客さまへ
お取引店の変更にともなう各委託会社等への変更手続きは、当金庫でいたします。
ただし、契約上お客さまからお届けが必要な場合は、当金庫から別途個別にご案内申し上げます。
( 7 ) 各種お振込金をお受け取りのお客さまへ
①
統合日以降、振込金受取口座の支店名 ( 支店番号 ) が変更となります。
引き続きお振込金をお受け取りいただくために、お手数ではございますが、
お振込人様 ( 公的機関、保険会社等を含みます ) へご通知をお願い申し上げます。
②
統合日の 2 カ月後の月末までは、旧支店の口座あてのお振り込みにつきましても、
統合先支店の新口座へご入金させていただきますが、お早めにお振込人様へご通知をお願い申し上げます。
( 8 ) 給与振込をお受け取りのお客さまへ
統合日以降、給与振込指定口座の支店名 ( 支店番号 ) が変更となりますので、お手数ではございますが、
ご勤務先の給与支払事務ご担当者にお届けくださいますようお願い申し上げます。
( 9 ) 年金振込をお受け取りのお客さまへ
①
国民年金、厚生年金のお受け取りにつきましては、当金庫から変更手続きをいたします。
②
各種共済年金、厚生年金基金をお受け取りの場合は、それぞれの年金支払機関によりお手続きが異なります。
お手続きをお願いする場合には、当金庫から別途個別にご案内申し上げます。
( 10 ) マル優・マル特をご利用のお客さまへ
お取引店の変更にともなう税務署への変更手続きは、当金庫でいたします。
( 11 ) 融資・各種ローンをご利用のお客さまへ
ご融資残高・条件、担保・保証等はそのまま引き継ぎいたします。
また、約定返済につきましても、従来どおり引き落としさせていただきます。
( 12 ) インターネットバンキング・ファームバンキングサービスご利用のお客さまへ
引き続きご利用いただけますが、最終営業日以前のお取引照会・残高照会が行えませんので、
お手数をおかけいたしますが、必要に応じて事前にご照会をお願いいたします。
また、統合日以降を指定日とする予約振り込みは受け付けできませんので、
統合日以降にお手続きをお願いいたします。
なお、変更手続きが必要なお客さまにつきましては、当金庫から別途個別にご案内申し上げます。
( 13 ) FAX 振込サービスをご利用のお客さまへ
統合日以降、 「 お客さま番号 」 が変更となりますので、別途ご案内させていただきます。
また、統合日以降を指定日とする予約振り込みは受け付けできませんので、統合日以降にお手続きをお願いいたします。
( 14 ) でんさいサービスをご利用のお客さまへ
引き続きご利用いただけますが、統合日以降、決済口座の支店名 ( 支店番号 ) が変更となります。
お手数ではございますが、お取引先様へご通知をお願い申し上げます。
( 15 ) 投資信託・国債・保険をご利用のお客さまへ
そのまま引き継ぎいたしますので、お手続きは必要ございません。
( 16 ) 外国為替業務のお取り扱い(海外からのご送金をお受け取りのお客さまへ)
①
円預金口座でお受け取りの場合は、統合日以降、受取口座の支店名 ( 支店番号 ) が変更となります。
引き続きご送金をお受け取りいただくために、お手数ではございますが、ご送金人様へご通知をお願い申し上げます。
②
外貨普通預金口座でお受け取りの場合は、従来ご使用の口座番号をそのままご使用いただけます。
③
ご送金人様へのご通知に使用いただく用紙
( 「 被仕向外国送金指図フォーム 」 ) を当店に用意しておりますのでご利用ください。
( 17 ) 外為インターネットバンキングサービスご利用のお客さまへ
引き続きご利用いただけます。
なお、変更手続きが必要なお客さまにつきましては、当金庫から別途個別にご案内申し上げます。
( 18 ) その他の業務のお取り扱い
統合日以降も引き続きご利用いただけますが、お取り扱いの内容を変更させていただく場合には、
ご利用のお客さまへ別途個別にご案内申し上げます。
お問い合わせは、お取引店もしくは最寄りの窓口までお願いいたします。
