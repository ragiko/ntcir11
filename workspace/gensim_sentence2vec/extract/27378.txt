質問リスト
Q. 「XREA」の読み方は?
「エクスリア」です。「eXtreme Realm for Experts and All」が語源です。
Q. どこの会社が運営していますか?
XREA、CORESERVER、COREBLOG、VALUE SERVER、VALUE-DOMAIN は、株式会社 GMO デジロックが運営しています。
Q. サーバ 1 台あたりの収容ユーザー (アカウント) 数はどれぐらい?
以下の通りです。
XREA の一般向け無料サーバはおよそ 1000 〜 1600 アカウント
XREA の VALUE-DOMAIN ユーザー専用サーバは 500 〜 800 アカウント程度
XREA の携帯電話コンテンツ専用無料サーバは 2600 アカウント程度
CORESERVER の CORE-MINI プランは 300 アカウント以下
CORESERVER の CORE-A プランは 128 アカウント以下
CORESERVER の CORE-B プランは 64 アカウント以下
CORESERVER の CORE-C プランは 16 アカウント以下
VALUE SERVER は特に公表されていません
ただし、XREA の古い無料サーバには休眠しているアカウントも多くあるようですので、「無料サーバはアカウント数が多いから他より重い」といったようなサーバの種類による特定の傾向はあまり見られません。また、XREA はサービス期間が長いため、古いサーバと新しいサーバでマシンスペックにかなりの差があり、そのぶん収容アカウント数はサーバによって若干差があります。
Q. ウェブサーバの所在地は日本国内ですか?
はい。XREA および CORESERVER のサーバはすべて日本国内 (大阪市) にあります。
Q. 中国、台湾、韓国に住んでいる友人が XREA 上のサイトにアクセスできないそうです。
中国、台湾、韓国の一部のプロバイダからの接続が拒否されているためと思われます。この場合、パケット単位で拒否されているため、ウェブアクセス等だけでなく、PING すら通らないようになっています。
これらのプロバイダからの不正アクセスの試みが多数あり、それ対する調査・改善依頼に対して、対応はもとより、全く返答がないのが拒否理由です (参考記事)。

Q. 付属のアクセス解析機能と AccessAnalyzer.com の違いは?
ウェブアカウントに標準で付属しているアクセス解析機能は、サーバーログを内部から解析します。AccessAnalyzer.com は、外部から JavaSCRIPT を用いてページを解析します。
標準のアクセス解析機能のメリット
各ページに JavaSCRIPT のソースを貼り付ける必要がない。
転送量が分かる。
画像など、HTML ページ以外へアクセスが分かる。
アクセス数が正確に分かる。
AccessAnalyzer.com のメリット
即時性がある。
ログを 90 日保存でき、その期間であればいつでも参照できる。
ログの検索機能がある。
JavaScript や Java 等が有効であるか分かる。
