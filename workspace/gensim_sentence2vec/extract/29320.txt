この研究は、学習理論において現れる確率変数の漸近挙動を解明したものです。
この研究を行うにあたって直接に必要になった数学は「特異点解消定理」
「特異的な超関数の漸近展開の方法」「関数空間上の中心極限定理」の三つです。
この三つの数学を基礎として、汎化誤差と学習誤差の漸近挙動を
二つの双有理不変量で特徴づけることが可能になりました。これは、統計学としては、
AIC や BIC の概念を、Fisher 情報行列が正則でないモデルにおいても成立するように
拡張したものに相当します。
もちろん、数学においては、幾つかの定理だけがピンポイントで得られるということはありません。
数学の発展のためには、総合的で、自然で、自由な、生命体としての数学全体の発展が不可欠です。
上記の三つの数学が応用上で必要になるから研究しよう、という狭い考え方では、それらの
数学はできていなかったと思います。また、
上記の三つの概念のために必要な数学は何であったかと問われれば、
それは数学の全ての分野になります。数学とは全体で数学なのであり、ある部分だけを
取り出して、そこだけを作るということはできません。つまり、私たちの行っている研究のためには、
数学の全てが基盤として必要であったと思います。現代の数学の発展を支えて来られた
先生がたと研究者の皆様に感謝を申し上げたいと思います。(2010/3/24)。

情報科学で必要になる数学の課題を二つあげてみます。
まず第1の問題です。
学習モデルが複雑になって行く極限、つまり、空間の次元があがり、
特異点の複雑さも大きくなる極限において、
学習係数である Log Canonical Threshold が、どのような挙動を持つか、
という問題があります。これは、代数幾何、代数解析において、ランダム行列理論の非線形無限次元版を
作ることに相当し、非常に重要な問題であると思います。
数理物理学からの直観を述べれば、そこには複雑さの極限で初めて見える大きな普遍的な法則が
あるものと期待されます。この問題の解決を得られた先生は、ぜひ、私たちに御教示ください。
(例)次のような多項式の列を考えます。
f_{1}(a,b) = (ab)^{2}
f_{2}(a,b,c,d) = (ab+cd)^{2} + (ab^{3}+cd^{3})^{2}
f_{3}(a,b,c,d,e,f) = (ab+cd+ef)^{2} + (ab^{3}+cd^{3}+ef^{3})^{2} + (ab^{5}+cd^{5}+ef^{5})^{2}
・・・
この多項式は、ある法則に従って増大しています。次の関数
Σ_{j=1}^{K} a_{j} \sin( b_{j} x ) 
の x についてのテーラー展開の係数の二乗和だからです。
このとき f_{k} の実対数閾値 λ_{k} は、k  ∞ での漸近挙動
λ_{k} = k^{1/2}/4 
を持つことがわかっています(注)。このような普遍性を任意の法則で定義された多項式の増大列に
ついて作ってください。申し上げるまでもありませんが、この多項式の増大列は、k とともに
変数の数も増えて行くので、ヒルベルトの基底定理のような有限性は、少なくても見かけ上は
存在しません。この問題を解決するための、新しい数学的構造を発見してください。
「増大する多項式の繰り込み理論」のようなものがありうるでしょうか。
(注)多項式 f(x) の実対数閾値 λ は次のように定義されます。Re(z)>0 で正則な複素関数
ζ(z)=∫f(x)^{z}dx
は、負の実軸上だけに極を持つ有理型関数に一意に解析接続されます。その最大極 (-λ)が
実対数閾値λを決めています。λは一般に有理数です。
物理学の研究をされている方のために実対数閾値の性質を
ひとつあげると
λ= lim_{n∞} ( - log∫exp(-nf(x))dx / log n ) 
が成り立ちます。つまり、λは「次元」のような量です。

次に第2の問題を説明します。W を実ユークリッド空間 R^{d} の
コンパクトな部分集合とし、w∈W とします。
また X_{1},X_{2},...,X_{n} を実ユークリッド空間 R^{N} に
値を取る独立な確率変数で、同じ確率分布に従うものとします。
実数値関数 f(x,w) が、任意の w∈W について E_{X}[f(X,w)]=0 を満たすものと
します。関数 f(x,w) は w については十分に良い性質を仮定しておきます。
(十分に高階まで連続微分可能であるとします(※))。
W 上の経験過程(Empirical Process)を
ξ_{n}(w)=( 1/ sqrt{n} ) Σ f( X_{i} , w )
によって定義します。ここで Σは i=1,2,3,...,n についての和を
表しています。定義から、すぐに次の二つが出てきます。
(1) 任意の w について E[ ξ_{n}(w) ]=0。
(2) 各 w 毎に、ξ_{n}(w) は正規分布に法則収束(中心極限定理)。
さて、情報学への応用においては、次の確率変数の挙動を
調べたいのです。
F = F( ξ_{n} ) = -log ∫ exp( ξ_{n}(w) ) φ(w) dw
希望としては、(1) この確率変数が法則収束し、(2) ξ_{n}(w) が
正規確率過程に法則収束し、(3) F の性質を正規確率過程に
関する微積分を行うことで調べられるようにしたい、と願っています。
W がコンパクトだと仮定したので W 上の連続関数全体の集合は
完備可分な距離空間(ポーランド空間)になり、完備可分な距離空間では
次のすばらしい定理が成り立ちます。
(プロホロフの定理)完備可分な距離空間に値を取る一様にタイトな
確率変数の無限列からは、タイトな確率変数に法則収束する部分列を
取り出すことができる。
この定理は数学・物理学・情報学で、
学部3年生以上の人なら誰でも知っている基礎的なものです。
(中心極限定理のような法則収束を示す場合の常套手段です)。
もうしあげるまでもなく、
この定理は極めて強力であり、実際、W がコンパクトであれば、ξ_{n}(w) の
法則収束を示すことができ、Fも法則収束することを示すことが
できます。
第2の問題は、この議論を W がコンパクトでない場合にも使えるように
したいということです。W がコンパクトでないと、W 上の関数空間は、
たいてい完備可分にはなりません。私たちの希望は、その場合でも F の挙動を
調べられるようにしたいだけなので、必ずしも ξ_{n}(w) の法則収束を
示すことは必要ないかもしれません。もちろん、F が法則収束したとき、その
性質を正規確率過程についての微積分を使って調べたくなる(※※)ので、
できれば ξ_{n}(w) の法則収束も欲しいと思います。F が法則収束するための
必要十分条件を確立してください。法則収束しない場合には、どのような現象が起きるかを
明らかにしてください。
なお、数理物理学を御専門に研究されている先生は、この問題は、
ランダム物理学におけるレプリカ法を数学として確立することと
関係があるということを御存知でいらっしゃると思います。
(※)W がコンパクトであるとき、ξ_{n}(w) の
正規確率過程への法則収束が成り立つための f(x,w) についての
十分条件については、非常に詳しく調べられています。
f(x,w) の w についての何回まで連続微分可能ならよいかは、
W が入っている空間の次元にも依存します。下記の参考文献(1)をご覧ください。
(※※)情報学への応用としては、一般のモデルにおいて、正則モデルの AIC の概念を
一般化するときに、正規確率過程 ξ(w) についての部分積分が必要になります。
なお、第2の問題が重要であることは間違いないと思いますが、
この問題は確率論としては、相当に困難な問題であるかもしれません。
