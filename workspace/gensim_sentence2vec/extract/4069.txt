英語の接辞のクラス分け
英語の派生接辞は次の2つのクラスに分けられる。
クラス1接辞(Class I Affix)・強勢に関して中立的でない接辞(Non-Stress-Neutral Affix)
クラス2接辞(Class II Affix)・強勢に関して中立的な接辞(Stress-Neutral Affix)
この派生接辞のクラス分けは、Siegel(1974)およびAllen(1978)が中心になり、接辞の音韻的特性・意味的特性・統語的特性に基づいて、提唱された。
接辞の音韻的特性
接辞に関する音韻的特性とは、その接辞が付加されると音韻変化が起こるかどうかの違いである。
ある接辞が付加されて音韻変化を引き起こせば、その接辞はクラス1接辞である。
illegal  ( in + legal )
ある接辞が付加されて音韻変化を引き起こさなければ、その接辞はクラス2接辞である。
nonlegal  ( non + legal )
接辞の意味的特性
接辞に関する意味的特性とは、その接辞が付加されると意味が透明になるかどうかの違いである。
意味が透明(transparent)とは、接辞の意味と語幹の意味の和が全体の意味となることを言う。
意味が不透明(opaque)とは、接辞の意味と語幹の意味の和が全体の意味とならないことを言う。
ある接辞が付加されて意味が不透明になれば、その接辞はクラス1接辞である。
infamous ( in + famous )
infamous(悪名高い)は、famous(有名な)の単なる否定の意味ではない。
ある接辞が付加されて意味が透明になれば、その接辞はクラス2接辞である。
unkind ( un + kind )
接辞の統語的特性
接辞に関する意味的特性とは、接辞の相対的位置に関する性質である。
クラス1接辞はクラス2接辞の内側に位置する
originalitylessness ( origin + al(1) + ity(1) + less(2) + ness(2) \
)
non-illegal ( non(1) + in(2) + legal )

接頭辞
接尾辞の分類の場合、強勢の位置の変化を引き起こすことがクラス1とクラス2の区別に大きな役割を果たしていた。しかし、英語の強勢は、語末を起点として決定される。したがって、接頭辞が強勢の移動を引き起こすことは考え難い。しかし、実際に強勢の移動が生じる場合もある。
famous [f&eacute;iməs] ー> infamous [&iacute;nfəməs]
この場合、基体となるfamousが2音節と短く、付加された接頭辞in-が語末から3音節目に位置するため、in-が強勢の受け皿になり、強勢の移動が生じたと考えることが出来る。
しかし、上のように強勢の移動を示す事例は稀で、接頭辞の分類の基準は、強勢以外の音変化、意味の透明・不透明性の違いなどに限られ、接尾辞の場合より、その判断基準があいまいであることが多い。
接頭辞は、接尾辞と異なり、その付加によって品詞を変えることはほとんどない。ただ、以下のような接頭辞は、品詞を変える。
