PDF ファイルを再作成して結合します。フォントをすべて埋め込むか、または全く埋め込みせずに PDF ファイルを再作成し、その後結合します。PDF ファイルの再作成については、ファイルの提供元に依頼してください。 注意 : フォントをすべて埋め込んだ場合、結合後の PDF ファイルのサイズは大幅に増加する可能性があります。全くフォントを埋め込まない場合、PDF ファイルを適切に参照するために必要なフォントのコピーがシステムにインストールされている必要があります。フォントの埋め込みとサブセットに関する詳細については、Acrobat ヘルプを参照してください。B. Acrobat Distiller の Runfilex または Rundirex を使用して単一 PDF ファイルを作成します。PDF ファイルに変換する前の元のドキュメントから PostScript ファイルを作成し、同一フォルダ内に保存してから、Acrobat Distiller の Runfilex または Rundirex を使用して単一の PDF ファイルを作成します。PDF ファイルの作成後、不要なページがあれば削除します。 注意 : Acrobat 8.1 の場合はこの操作を行う前に、文書番号 232142 Acrobat Distiller 8.1 におけるアクセス制限の問題を参照してください。Runfilex および Rundirex の使用方法については、以下の弊社技術文書を参照してください。文書番号 220200 Acrobat Distiller 4.0/5.0(Mac/Win):Acrobat Distiller で Runfilex と Rundirex を使って複数の PostScript ファイルから 1 つの PDF を作成する 注意 : Acrobat 6.0 ～ 8 の場合、「Runfilex.ps」および「Rundirex.txt」ファイルは以下の場所にあります。Windows 版− Acrobat 6.0 : Program Files¥Adobe¥Acrobat 6.0¥Distillr¥Xtras− Acrobat 7.0 : Program Files¥Adobe¥Acrobat 7.0¥Distillr¥Xtras− Acrobat 8 : Program Files¥Adobe¥Acrobat 8.0¥Acrobat¥XtrasMacintosh 版− Acrobat 6.0 : Macintosh HD/ユーザ/共有/Adobe PDF 6.0/Example Files− Acrobat 7.0 : Macintosh HD/ライブラリ/Application Support/Adobe PDF/Example Files− Acrobat 8 : Macintosh HD/ライブラリ/Application Support/Adobe/Adobe PDF/Distiller/Example Files追加情報 (Additional Information)サブセットフォントを含む PDF ファイルを結合すると、サブセットされているフォントがコンフリクトを起こし、Acrobat でエラーが発生する場合があります。サブセットフォントを減らしてから PDF ファイルを再作成することによりコンフリクトを回避し、正常に PDF ファイルを作成できる可能性があります。Runfilex および Rundirex を使用して複数の PostScript ファイルから単一の PDF ファイルを作成することによって、ファイルごとにフォントに対するサブセットが設定されるのではなく、それぞれのフォントに対して単一のサブセットが作成されます。
