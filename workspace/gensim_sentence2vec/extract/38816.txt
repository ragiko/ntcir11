Internet Explorer(以下、IEと表記)を使い分けるにあたっての注意点は以下になります。
スタート画面のピン留めサイトは、ピン留めした方のIEで開かれる
Windows ストアアプリ版IEからピン留めしたサイトは、Windows ストアアプリ版IEで表示されます。
デスクトップ版IEからピン留めしたサイトは、デスクトップ版IEで表示されます。
フォルダー分けしたお気に入りはWindows ストアアプリ版IEから見えない
デスクトップ版IEのお気に入りはWindows ストアアプリ版IEでも開くことができます。
ただし、 お気に入りに作成したフォルダーについてはWindows ストアアプリ版IEから開くことができません。
CookieはそれぞれのIEに保管され、共有されない
パスワード保存やサイト訪問などを記録するCookieは共有されません。
このため、Windows ストアアプリ版IEでパスワード保存したサイトへデスクトップ版IEで訪問すると、再度パスワード入力が必要になります。
ホームページの設定は共有される
設定されたホームページは共用となり、別々のホームページの設定をすることはできません。
Windows 8.1 Updateの場合
それぞれのIEで設定をしても、最後に設定したページが有効になります。
Windows 8の場合
Windowsストアアプリ版には設定を行う場所がありません。
デスクトップ版 IEでホームページを変更すると、Windowsストアアプリ版 IEに反映されます。
Windows ストアアプリ版IEのAdobe Flashの対応について
OSにより、Adobe Flashの対応状況が異なります。
Windows 8.1 Updateの場合
Windowsストアアプリ版IEとデスクトップ版IEの両方でサポートされています。
デスクトップ版IE の設定[アドオンの管理]のダイアログボックスでFlashを有効または
無効の設定ができ、その設定はWindowsストアアプリ版IEにも反映されます。
Windows 8の場合
Windowsストアアプリ版IEでは、Flashに対応していません。(YouTubeやニコニコ動画など、
一部対応しているサイトもあります。)
非対応のFlashについては、Windows 8のWindowsストアアプリ版では空欄になってしまいます。
デスクトップ版はFlashに対応していますので、デスクトップ表示に切り替えることで、
Flashを見ることができます。
