この章では、Cisco Extension Mobility Cross Cluster 機能について説明します。この機能を使用すると、ある Cisco Unified Communications Manager クラスタ(ホーム クラスタ)のエンタープライズ ユーザが、外出先で別の Cisco Unified Communications Manager クラスタ(訪問先クラスタ)の Cisco Unified IP Phone に、ホーム オフィスで IP Phone を使用しているかのようにログインできます。
