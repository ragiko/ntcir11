 さて、今まで紹介した以外のキャラクタの動きコマンドをまとめてご紹介します。キャラクタにさせたい動作をTVMLのような言語で表現するとき、それはいわゆる「動詞」の表現ということになります。TVMLで用意されている動詞はそれほど多くはなく、15個前後です。
それでは、いままで出てこなかったコマンドを順に紹介します。

おじぎさせる
おじぎはbowコマンドです。
character: bow ( name = Bob )
と書くと、普通に一回おじぎをします。speedでおじぎの速さ、degreeでおじぎの深さを角度で指定できます。ちなみに、degreeにマイナスの角度を与えると後ろにエビ反りになります。また、style = casualとすると首だけを使っておじぎをします。
震えさせる
震えはshakeコマンドです。
character: shake ( name = Bob, state = on )
とするとブルブルと震えます。state= offでコールすると震えが止まります。震えのていどはlevelで調整してください。このshakeコマンドは、表情変化のないBobやMaryといったキャラクタが怒っているところを表現するときや、怖くて震えるときなどによく使います。
倒れさせる
立っているキャラクタを、バタッと倒すにはtumbleというコマンドを使います。
character : tumble ( name = Bob )
と書くと、まん前に向かってバタッと地面にうつぶせになるまで倒れます。倒れたキャラクタを元に戻すにはrotxにゼロを指定して
character : tumble ( name = Bob, rotx = 0 )
とします。実はrotxのデフォルトが90度で、rotxを省略すると90度、つまり床のところまで倒れるのです。ちょっと斜めに倒したい、などというときはrotzに角度を指定します。
ジャンプさせる
ジャンプはjumpコマンドです。これは、目的地のxyz座標と、そこにキャラクタが着地したときの向きを指定して現在位置からジャンプさせます。
character : jump ( name = Bob, x = 1.5, z = 2.0 , y = 0.5, d = 45.0 ) 
のようにすることで指定されたxyz座標と向きにジャンプして移動します。heightパラメータはジャンプする高さをメートルで指定します。
また、speedパラメータでジャンプのスピードをコントロールできるほか、spinパラメータでジャンプのときに回転を加えることができます。ちなみにspinにマイナスの回数を指定すると後方宙返りになったりしますので、ジャンプを使ってサーカスっぽい動きも表現できるかもしれません。
口を開かせる
openmouthコマンドで口を開けることができます。
character : openmouth ( name = Bob, state = on )
とすると、カパッと大口を開けます。「うわっ」と驚くときなどにこれがつかえます。口のあけ方の程度はdegreeで調整できますので、微妙に口をあけさせて「ぽかん」とした雰囲気も出すことができます。
また、speedで口を開ける速度を調整でき、これをゆっくりにすると「あくび」した感じが出せます。開いた口を閉じるにはstate = offとしてコールしてください。
揺れさせる
この揺れるは、震えるに似ていますが、ちょっと違います。swingというコマンドで、ゆらゆらとゆっくり上体を揺らすような感じが出せます。
character: swing ( name = Bob, state = on )
とすると揺れ始めます。揺れを止めるときはstate = offとしてコールします。degreeパラメータで揺れの程度、speedパラメータで揺れの速度が調整できます。
このswingを微妙にかけると、キャラクタが何もしていないときでも少し動いた感じになるので、何もしないキャラクタが微動だにしない不自然を解消できます。キャラクタによっては、スクリプト中でswingをコールしなくても、デフォルトでこの「揺れ」が設定されています(cherryなど)。そのような設定のあるキャラクタの揺れをなくしたいときは、スクリプトの最初にstate = offでswingをコールし、揺れを止めてください。
バイバイさせる
「バイバイする」というのが本当に正式な動詞なのか定かではないのですが、TVMLでは手を振ってバイバイするコマンドbyebyeが用意されています。
character : byebye ( name = Bob )
とすると「右手を上げて手を3回振って、手を下ろす」デフォルトのバイバイ動作をします。handパラメータで、右手、左手、両手のバイバイが切り替えられます。また、手を振る回数はtimesパラメータで指定できます。
このバイバイの動作は、まず腕を上げ、手を振り、下ろしますが、この腕の上げ下ろしについては、ユーザーが明示的に行わせることもできます。こうした任意のポーズを取らせるためにposeコマンドというものがあり(3-10:キャラクタに好きなポーズをとらせる参照)、このposeコマンドで腕を好きな位置に上げ、byebyeコマンドで手を振らせ、またposeコマンドで腕を下ろす、というやり方もできます。この場合、byebyeコマンドのwithpose パラメータはoffと指定します(デフォルトではこのパラメータはon)。
character : pose ( name = Bob, pose = udewoageru )character : byebye ( name = Bob, withpose = off )character : pose ( name = Bob, pose = default )
1行目のposeコマンドで、あらかじめ定義したudewoageruという名前のポーズをとらせ、次にバイバイし、最後にdefaultという腕を下ろしたポーズをとらせます。
まばたきさせる
まばたきをコントロールするのはblinkコマンドです。当然ですが、まばたきをサポートしているキャラクタにしか使えません。まばたきをサポートするキャラクタであればデフォルトでオンになっています。まばたきの頻度は、3秒に平均1回がデフォルトで設定されています。
これを変更するには、frequencyパラメータを指定してblinkをコールしてください。
character: blink ( name = Miyake, switch = on, frequency = 1.5 )
とすると、1.5秒に1回のまばたきとなり、かなり頻繁になります。しかし、テレビのアナウンサーは、1秒から2秒に1回程度まばたきをすると言われています。まばたきを頻繁にするほうが、見ている人に安心感を与える度合いが大きそうです。ただ、CGではまばたきが目立つので、この頻度ではちょっと頻繁すぎるようにも見えます。
