基本的に、「ロジック」に使うか「ビュー」に使うかで使い分けてます。
この語彙の意味が分からなければ「MVCモデル」「PHP フレームワーク」などで調べてみてください。
ロジック(例: 140字をオーバーしていたら140字以内にまとめる)
if (mb_strlen($str) > 140) {
$str = mb_substr($str, 0, 140);
}
ビュー(例: 140字をオーバーしていたらエラーを出す)
<?php if (($diff = mb_strlen($str)) - 140 > 0): ?>
<p>140字を<?=$diff?>字オーバーしています</p>
<?php endif; ?>
ビューで普通のif文で分岐させたりしたらとんでもなく見にくいでしょ?
