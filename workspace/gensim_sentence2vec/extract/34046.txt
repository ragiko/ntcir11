これまでに学んだことを生かして簡単なコンテンツを製作してみよう。すなわち:君は予備校教師であるとし、大学受験生のために2次方程式に関する基礎事項を簡潔にまとめた教材を製作するものとする。
ここでは W. Lee & D. Owens, Multimedia-Based Instructional Design, Pfeiffer, 2000(日本イーラーニングコンソシアム訳:インストラクショナルデザイン入門、東京電機大学出版局、2003)の「授業設計」手法に従って進めよう。
I.「製作前段」:「コンテンツ計画書」を作る
【ニーズ評価(Needs assessment)】:受験準備用
【初期分析】以下の9項目あり。印は最難間作業、印はこの課題で行う作業、印は考えない作業を表す。
目標分析(Objective analysis):大学受験生に2次方程式に関する基礎的事項をマスターさせる
受講者分析(audience analysis):大学受講生
技術分析(Technology analysis):考えない
タスク分析(Task analysis):考えない
環境分析(Situational analysis):考えない
メディア分析(Media analysis):考えない
コスト分析Cost-benefit analysis):考えない
重要項目分析(Critical-incident analysis)
既存資料分析(Extant-data analysis)
コンテンツ製作パラダイムにおける「素材の準備」と「構造化」がこれに相当する。既存資料をよく調査し、2次方程式に関してぜひ知っているべき事項を集め、どういう順序で並べるべきか、全体構成を決める。
以上の作業の結果「コンテンツ計画書」が出来上る。下に例を示す。(3部構成で書かれていることに注意する。「概要」が第1部、最後の「要約と結論」が第3部、その間の部分が第2部である。)
「コンテンツ計画書」例
製作方針:これまでに学んだことを生かす。
パラダイムの最初のステップに対応する作業から始める : Lee & Owens のニーズ評価/初期分析を参照し、製作予定文書の目的、誰が対象か、などの文書全体の性格に対する簡単な説明をまず書く。既存資料分析を通じてカバーすべき重要項目を決める重要項目分析はしっかり行う。
全体構成は3部構成とする(導入部、本論、要約と結論)。
バックストーリーの考え方をヒントにすれば、1次式のグラフと零点について簡単に言及する(結果だけのべる)。
前進的構成法と後退的構成法の考え方より、論理的な部分は前者の方法、ストーリー構成は後者の方法を応用しよう(グラフの話を先にするか、代数的考察を先に出すか、このあたりのストーリー構成はは自由度のあるところ)。以下では代数的取り扱いを先にしてある。
第1部(導入部) : 大学受験生のために二次方程式に関する基礎的事項をまとめた復習用教材。
第2部(中心部) : 教材本体。重要項目と提示の順序は以下の通りとする(重要項目分析&既存資料分析結果)。
2次方程式の定義 定義を書く : ax2 + bx +c = 0を2次方程式という。ここにa, b, cは実数または複素数、aは非零、xは未知数を表す。
因数分解形 : これが中心的事実。
証明 :  「右辺を展開すればよい」とだけ述べる。
導出 : 上式の左辺を変形して行く。ax2 + bx + c = a {( x + (b / 2a) )2 - ( b2 - 4ac )/(4a2)} =a ( X2 - Y2 ) = a (X + Y)(X - Y). ここにX = x + ( b / 2a), Y = ( b2 - 4ac)1/2 / ( 2a ).
根の公式 : (因数分解形から出てくることを指摘し、実際に書き下す。これら以外に根がないこともいう。)
x = ( - b ± ( b2 - 4ac )1 / 2) / ( 2a).
根と係数の関係 : 2根をp, qで表すと、和p + q と積pqが係数a, b, cで表わせる。根の公式を用いる
p + q = - b / a, pq = c / a
2次式のグラフ : 係数a, b, cはすべて実数とする。y = x2のグラフ(上開きの放物線、原点は頂点と呼ばれる) y = ax2のグラフ(a > 0なら、y = x2のグラフにおいてx軸の縮尺を1/ a1/2としたもの、a < 0の場合は…) y = ax2 + bx +cのグラフはy = ax2のグラフをx軸の正方向へだけ、yの正方向へだけ平行移動したもの、頂点は(, )、x=に関して対称な放物線。この事実は上の「導出」で得られた式
ax2 + bx + c = a {( x + (b / 2a) )2 - ( b2 - 4ac )/(4a2)} = a ( x + (b / 2a) )2 - ( b2 - 4ac )/(4a)
からいえること。グラフの例をいくつか出す。以上の幾何学的考察から判別式D = b2 - 4acの符号と実根の個数との関係が明らかになる(詳しく述べること)。
第3部(要約と結論) : これまで述べた事項のポイントと励ましの言葉を書く。主な代数的結果は2次式の因数分解形であり、これから根の公式、根と係数の関係が導出されること。幾何学的結果として実係数2次式ax2 + bx + cのグラフが特定されたこと。それは(, )を頂点とし、x = に関して対称な放物線であり、判別式の符号と実根の個数との関係が明らかになったこと。繰り返し復習すること。成功を祈ること。
「コンテンツ計画書」終わり
II. 製作:表出工程(コンテンツの執筆)
上で作った「コンテンツ計画書」に基づいてコンテンツを書く。追加説明、例題、注意事項はこのときに入れよう。指定されたメディアは日本語である!
III. 製作後段:改訂、編集工程
コンテンツを再読、再々読、…し、改訂作業を施す(削除、追加、表現の改良、など)。
手抜きしないこと!
参考:「取るに足らないような細かい修正でもそれが積み重なって美が完成する」(このミケランジェロの言葉を忘れるな)。
これでコンテンツ製作工程は終わりです!
要約と結論
