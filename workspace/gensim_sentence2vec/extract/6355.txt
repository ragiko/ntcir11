Google Chromeは、数分でインストールできる無料のウェブブラウザです。Windows、Mac、Linux 搭載のパソコンでご利用いただけます。Google Chrome は、Android や iOS を搭載した端末にインストールすることもできます。
Google Chrome をダウンロード
Google Chrome をパソコンにダウンロードする
Windows
自分のユーザー アカウント用に Chrome をインストールするには以下の手順を行います。インストールの途中で、パソコン上のすべてのユーザー アカウント用にブラウザをインストールするオプションが表示される場合があります。このオプションが表示されない場合は、Chrome を複数のユーザー アカウントにインストールする方法をご確認ください。
インストーラ ファイルをダウンロードします。
メッセージが表示されたら、[実行] または [保存] をクリックします。
インストーラを保存した場合は、インストーラ ファイルをダブルクリックしてインストールを開始します。既定のブラウザからホームページの設定やブラウザ履歴が自動的に読み込まれます。
Windows 7 / Vista / XP*
すべての設定が完了すると、Chrome ウィンドウが開きます。Chrome を初めて起動すると、アドレスバーから直接 Google で検索できることを案内するメッセージが表示されます。既定の検索エンジンを変更する場合は、[変更] をクリックします。
*2015 年 4 月以降、Chrome は Windows XP に対応しなくなります。
Windows 8[ようこそ] ダイアログが表示されるので、[次へ] をクリックして既定のブラウザを選択します。[Google Chrome] を選択すると、スタート画面から Windows 8 アプリとして起動できるようになります。
Chrome をインストールまたは起動できない場合は、トラブルシューティング情報をご覧ください。
インストーラ ファイルをダウンロードします。
ダウンロード後、Google Chrome.dmg というファイルを開きます。
開いたウィンドウで、Chrome アイコンをアプリケーション フォルダにドラッグします。この操作により、パソコンのすべてのユーザー アカウントに Chrome がインストールされます。ホームページの設定やブラウザ履歴が既定のブラウザから自動的にインポートされます。
管理者権限がない場合は、デスクトップなど、パソコン上でアクセス権のある任意の場所にアイコンをドラッグします。この場合、Chrome は自分のユーザー アカウントのみにインストールされます。
初めて Google Chrome を開くと、インターネットからダウンロードしたアプリケーションを開いてもよいかどうか、Finder から確認を求められます。[開く] をクリックして処理を続行します。
アドレスバーから直接 Google で検索できることを知らせるメッセージが表示されます。既定の検索エンジンを変更する場合は、[変更] をクリックします。
Google Chrome のディスク イメージを Dock の取り出しアイコンにドラッグし、ディスク イメージのマウントを解除します。Google Chrome アプリケーションのアイコンではなく、必ずディスク イメージをドラッグしてください。
Chrome は、ローカルのユーザー ディレクトリにコピーするだけで管理者権限がなくても正しく動作しますが、ネイティブ パッケージ管理システムからインストールすることをおすすめします。この処理を行うには、インストール時に管理者権限が必要になります。
インストーラ ファイルをダウンロードします。
[OK] をクリックしてパッケージを開きます。
[パッケージのインストール] をクリックして Chrome のインストールを開始します。
すべての設定が完了すると、Chrome ウィンドウが開きます。Chrome を初めて起動すると、アドレスバーから直接 Google で検索できることを案内するメッセージが表示されます。既定の検索エンジンを変更する場合は、[変更] をクリックします。
ブラウザを起動するには、[アプリケーション] > [インターネット] > [Google Chrome] の順にクリックします。Google Chrome のレポジトリがパッケージ マネージャに追加され、最新の状態を維持できるようになります。
