エンドユーザ層の拡大により,ユーザがソフトウェアを利用する際の操作学習支援がより重要になってきている.ユーザの操作学習支援を行うシステムとして,ソフトウェア上の操作を実演するチュートリアルシステムが提案されている.ユーザが効果的に学習をするためには,利用頻度の高い操作手順の学習を行うことが重要である.利用頻度の高い手順を学習することを支援するためには,ユーザがどのような手順で操作をするかを解析する必要がある.そこで本手法では,ユーザがソフトウェアを操作したときの操作ログを解析し,ソースコードから抽出したウィンドウの遷移を組み合わせることにより,チュートリアルシステムを自動生成し,ソフトウェアに付加する手法を提案する.
Because of increasing end-users of software, supporting to learn operaion methods becomes more important. For that purpose, tutorial systems are proposed. To learn operation methods in tutorial systems effectively, it is important to learn the operation methods which end-users use frequently. Then, it is necessary to analyse which sequences of operation methods are used more frequently. In our research, we propose a method for generating a tutorial system by analysing operation logs that end-users operates software, and by extracting windows sequences from source codes. A generated tutorial system is added to the target software automatically.
