ソフト詳細説明
英単語、英文等の音声ファイルを使用した学習に便利なように作成したソフトです。
一枚の音声フアイルを区切って再生することの出来るソフトです。
音声フアイルの再生中に予め指定したポジションに無音区間、若しくは停止を自動的に動作させます。
ソフトの機能で動作させる物で音声フアイルを加工するものではありません。
ポジションの指定も簡単で再生中にストップさせるだけです。
次からの再生はそこで自動的に停止・任意の秒数のポーズ(自動モード時)をします。
ポジションは複数いくらでも指定できます。
英語長文の音声ファイルの区切り聞き、シャドーイングの時などに便利です。複数の単語やフレーズが記録されているファイルでは、一語、一フレーズごと区切って聞くのにも便利です。
再生できる音声ファイルは、拡張子が WAV,MP3,WMA のものです。
本ソフト [ pause_setter ] で作成した区切れポジションデータを、別途作成したソフト [ pause_handler ] に使用するとさらに、逆順再生、スキップ再生、ランダム再生、エンドレス再生などといったより便利な動作をさせることが出来ます。[ pause_handler ] もフリーソフトです。[ pause_handler ] は当ホームページからダウンロードできます。
---
以下のような理由で作成しました。
音声ファイルの内容が [ 訳語 ] + [ 英文 ] + [ 訳語 ] + [ 英文 ] + … とか [ 英文 ] + [ 英文 ] + [ 英文 ] + [ 英文 ] + … の様に並んでいる、もしくは [ 長文 ] である場合に、各区切れ目に(ポーズ)無音区間が無い、あるいは充分でない時、いちいち句切れの度に再生ボタンと停止ボタン、あるいはポーズボタンをクリックするのは面倒なので、各区切れ目に任意の無音区間を動作させたい。あるいは、いちいち停止させる事を意識していると練習への集中の妨げになるので、再生ボタンをクリックした後は自動的に区切れ目で停止して欲しい。
そうすれば、区切り聞きや、音読練習等がもっとしやすくなり、また単語カードの様な学習も可能になる。
