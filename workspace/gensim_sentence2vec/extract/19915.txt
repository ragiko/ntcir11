解説
ハードディスクドライブ メーカー各社は、セクター サイズが4キロバイトの新しいドライブを発表しました。このドライブは一般的にAdvanced Format、4Kまたは512eドライブなどと呼ばれます。新しいセクターサイズを採用することで、データ容量とエラー訂正に関してハードドライブの性能が向上する反面、Microsoft Windows XPなどの古いオペレーティングシステムとの互換性に問題が生じます。弊社は4キロバイトのセクターサイズのドライブを採用したシステムの出荷を2012年初めより開始します。本ページでは、この互換性に関する問題について説明し、新しい世代のハードディスクドライブでWindows XPをうまく活用できるように案内します。
1. 対象となるシステムにWindows XPをインストールするときは、4キロバイト ドライブに合わせた方法でパーティションを作成する必要があります。
2. Windows VistaとWindows 7をインストールするときは、マイクロソフト社が提供する修正パッチとサービスパックをインストールする必要があります。
3. バックアップとリストア、イメージ作成、パーティション管理、仮想化などのハードドライブ管理ソフトウェアを使用している場合は、それらのソフトウェアの販売元に連絡して4キロバイト タイプのドライブに対応しているか確かめてください。
すべてのファイル システムはハードドライブをクラスターに基づいて構成しています。クラスターとはファイルを配置し、保持することができるディスク領域の最小単位のことです。通常、クラスター サイズは4キロバイトです。これまでハードドライブはこのクラスターを512バイトごとのセクターに置き換えてして使用していました。このセクター サイズを4キロバイトにすることで、1クラスターがちょうど1セクターと対応するようになります。4キロバイトのクラスターが正確に4キロバイトのセクターに対応しないと、アラインメントの問題が生じてきます。この種の問題はパフォーマンスの低下を引き起こす可能性があります。実際のパフォーマンスの影響はドライブの性能によって異なりますが、ほとんどのテストではアラインメントが正常なドライブに比べて問題があるドライブは20%から40%遅くなります。この問題に対処するために、ファイル システムのクラスターがハードドライブのセクターに一致するように配置されているか確かめてください。不運にもWindows XP等の古いバージョンのWindowsが4キロバイトのセクターで構成されたパーティションを正しくアラインしていない場合は、インストールを成功させるためにいくつか追加の作業を実施する必要があります。
関係するシステム構成
上記の問題は主に以下のシステムに関係します:
すべてのデスクトップ システム
詳細
1. Windows XP: インストールする前にハードドライブを4キロバイト単位でパーティション作成してある必要があります。Windows XPで作成したパーティションは使用できません。適切なパーティションはDISKPARTコマンドを使用して作成することができます。
a. Windows PE 1.6 (DISKPART バージョン5.2.3790) の場合: DISKPARTでパーティションを作成するときに、パラメーターに align=1024 を指定してください。(例: create partition primary align=1024)
b. Windows PE 3.0以降 (DISKPART バージョン6.1.7600以降) の場合: DISKPARTは自動的に適切な配置でパーティションを作成します。パラメーターを追加する必要はありません。
c. Windows自動インストール キットを使用する場合: 
i. 以下に挙げる、影響を受けないオペレーティング システムがインストールされたPCを用意します:
Microsoft Windows 7 およびサービスパック1 
Microsoft Windows Vista およびサービスパック1
Microsoft Windows Server 2003 およびサービスパック2
Microsoft Windows Server 2008 または Server 2008 R2
ii. Windows自動インストール キット(AIK) Windows® 7用をhttp://www.microsoft.com/downloads/details.aspx?FamilyID=696dd665-9f76-4177-a811-39c26d3b3b34&displaylang=en からダウンロードし、用意したPCにインストールします。古いバージョンのAIKはここで述べる作業には合いませんので注意してください。
iii. 「ステップ バイ ステップ ガイド: IT プロ用 Windows の基本展開」の「ステップ 3:起動可能な Windows PE メディアを作成する」 http://technet.microsoft.com/ja-jp/library/dd349348(WS.10).aspx#BKMK_3に記載された手順に従ってWindows PE起動メディアを作成します。
iv. 作成したメディアでWindows XPをインストールするPCを起動し、diskpart をコマンドラインから実行し、DiskPartコマンドでディスクにパーティションを作成します。 http://technet.microsoft.com/ja-jp/library/cc770877(WS.10).aspxをご覧ください。
v. Windows XPインストール メディアを起動し、先ほど作成したパーティションにWindows XPをインストールします。
2. Windows Vista: マイクロソフト社が提供する修正パッチ http://support.microsoft.com/kb/2470478をインストールします。
3. Windows 7: サービスパック1か、マイクロソフト社が提供する修正パッチ http://support.microsoft.com/kb/982018をインストールします。
4. ハードドライブ管理ソフトウェア (バックアップ、リストア、イメージ作成など) を使用している場合: ソフトウェアの販売元に連絡して4キロバイト タイプのドライブに対応しているか確かめてください。
a. Lenovo ThinkVantage Rescue and Recovery: Windows XP/Vistaではバージョン4.23以降を、Windows 7ではバージョン4.3以降を使用してください。
b. Symantec Ghost: Ghost Solution Suite バージョン2.5に含まれるバージョン11.5に、-align=1mb パラメーターをつけて使用します。GHOファイルに未調整のパーティションが含まれるとき、このパラメーターは正しく調整されているのと同じようにイメージを転回します。パーティションがあらかじめ正しく調整されているときはこのパラメーターは必要ありません。Ghostのパラメーターについて詳しく知りたい場合はシマンテック社に連絡してください。
c. Paragon Hard Disk Manager: バージョン2011以降をしてください。
d. Paragon Drive Copy: バージョン11以降を使用してください。
e. Paragon Migrate OS to SSD: バージョン2.0以降をしてください。
よくある質問
1. イメージが適切に調整されているかどうか調べるにはどうすればいいですか。以下の方法で調べることができます:
Windowsのシステム情報で調べます: 
i. スタート ボタンをクリックし、 「すべてのプログラム」 -> 「システム ツール」 -> 「システム情報」を選択します。
ii. ウィンドウ左枠のパネルから「コンポーネント」 -> 「記憶域」 -> 「ディスク」を選択します。
iii. 右枠のパネルから「パーティション開始オフセット」を捜します。
iv. 「パーティション開始オフセット」の値が4096で割り切れるかどうか調べます。割り切れる場合は、ディスクは適切に調整されています。
2. 512バイト ドライブ用のイメージは4キロバイト ドライブ用と違う調整をする必要がありますか。 いいえ。4キロバイトに調整されたイメージは512バイト ハードドライブ、4キロバイト ハードドライブ、SSDのどれでも正常に動作します。お使いのイメージが4キロバイトに調整されている場合は、イメージを展開する記憶デバイスの種類に気を遣う必要はありません。.
3. SSDは今回の4キロバイトへの変更により何か栄光を受けますか。 SSDはすでに4キロバイトの 物理セクターを使用しています。パフォーマンスを維持するためにSSDに適した調整をすることが重要です。
