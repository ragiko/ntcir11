DAISY3、ANSI/NISO Z39.86標準規格への移行
DAISY3の機能はDAISY2を遙かに超えており、最も重要な特徴としては、XMLの使用と、印刷物から変換される図書用に備えられた豊富な語彙があげられる。
プレーヤー、制作ツール、研修など、DAISYをサポートする全ての分野でDAISY3への移行を行う必要がある。
DAISY3の検証ソフトとXMLテキストは移行の初期の段階で導入することができる。
全ての要素の移行が終われば、最終的にDAISY3が最新の推薦仕様となる。
DAISY3、ANSI/NISO Z39.86は2004年に若干改訂される予定である。
今春、同標準規格に対する若干の改訂が完了する予定である。
変更を再検討し、維持委員会に参加するよう、開発者が招待されている。
変更の是非に関する投票は夏に完了すると思われる。
ナビゲーションに対する修正により、より効果的に実行できるようになるであろう。
DAISY2.02はDAISY3にアップグレードすることができる。
XMLボキャブラリーについていくつかの小さな改良点がある。
アメリカ合衆国におけるNational File Format制定に向けた動き
特殊教育プログラム局(Office of Special Education Programs)はNational File Format(NFF)技術委員会を設けた。
CAST(Center for Applied Special Technologies)がコーディネーター機関で、スキップ・シュタール氏がプロジェクトの責任者であった。
NFF技術団は2003年10月にOSEP(特殊教育プログラム局)に一連の提言を提出した。
同勧告は全国指導教材アクセシビリティー標準規格(National Instructional Materials Accessibility Standard; NIMAS)として知られている。
NIMASはDAISY3、ANSI/NISO Z39.86標準規格を適用している。
OSEPは技術委員会の勧告に関し決定を下すよう求められている。これにより公式にK-12(幼稚園から高校3年生に至るまで)の分野で全国的な標準規格が確立される。
IDEA(個別障害者教育法)の再認にはNIMASを盛り込むことを必要条件とし、これによりファイルの提供を法的に義務づける。
国連においても、世界情報社会サミット(WSIS)で同じ標準規格が推奨されている。
現在提案されているNIMASの規定———DAISY3、ANSI/NISO Z39.86の適用
出版社は、委託機関にDTBook(録音図書)と同じ内容のXMLファイルと、DAISY3及びANSI/NISO Z39.86で規定されているXML語彙を提供する。
標準規格が進歩するということは誰もが承知していることなので、言語は最新の推薦仕様のことである。
基本的なマークアップについて述べられているが、出版社はタグ・セットを全面的に適用することを選択できる。
画像は委託機関に提供されるものとする。これを合法化するために、出版社自体も委託機関として認められる。
画像は出版社が所有するPDFに入っているので、PDFファイルがXMLとともに提供されるものとする。
委託機関はファイルに付加価値をつけることが期待される。
委託機関はエンドユーザーに配信する。
委託機関は、正確にはどのようにしてANSI/NISO Z39.86DTBook(録音図書)ボキャブラリーによるNIMAS XMLファイルを使用するのか?
委託機関はファイルを強化し、基本的な水準から完全に標準規格に従ったファイル・セットへと進展させることができる。
NIMASに従ったファイルは、そのままDAISY3の制作ツールで利用することができる。
CSUNのプログラムで行われているように、話し言葉を合成することができる。
人による朗読は音声と同期化することができる。この具体的な例については金曜日に行われる比較検討を参考にしてほしい。
点訳ソフトもNIMAS標準規格を支持している。Duxburyのブースを訪れてほしい。
オープン・ソースのDAISYマルチ・フォーマット変換器を製作しようというプロジェクトが最初の段階に入っている。これによりWord、WordPerfect、Large printその他のフォーマットへのモジュールが作られ、これらのフォーマットで出力することが可能となる。
シンプルなスタイル・シート(CSS)を適用することにより、IE(インターネット・エクスプローラ)のようにウェブ・ブラウザで直接閲覧できるようにする。しかし、この場合はDAISYが持つ全ての能力を得ることはできない。
我々が求める全ての特質を備えた強力な情報システムの創造は、XMLによる正確なコンテンツにかかっている。
高等教育におけるNational File Format(NIMAS)の採用
最新の州高等教育法、ニューヨーク州法の第219章では、National File Format、NIMASに言及している。
ケンタッキー州法SB85は、National Standard(全国的な標準規格)によるファイルを要求している。
旧来のカリフォルニア州法AB422は、ANSI/NISO Z39.86がアメリカ合衆国の標準規格とされる前に可決されていた。
私は将来全ての法律がNational File Format、NIMAS、DAISY3或いはANSI/NISO C39.86標準規格に言及するようになることを奨励する。これらは全て同じである!
