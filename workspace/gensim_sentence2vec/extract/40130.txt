USB 3 (「SuperSpeed USB」) とは何ですか?
USB 3 (「SuperSpeed USB」とも呼ばれます) は最新の USB 規格で、USB 2 (「高速 USB」) を大幅に上回るパフォーマンスを実現します。
USB 3 の転送速度はどのくらいですか?
USB 3 のデータ転送速度は最大 5 Gbps です。
USB 2 のデータ転送速度は最大 480 Mbps です。 
USB 1.1 のデータ転送速度は最大 12 Mbps です。
USB 3 の給電能力は USB 2 (「高速 USB」) より大きくなっていますか?
はい。USB 3 デバイスの給電能力は最大 900 mA です (USB 2 デバイスの給電能力は最大 500 mA)。900 mA の電流を利用するには、USB 3 ケーブルが必要です。USB 2 ケーブルは 900 mA に対応していません。
一部のコンピュータでは USB ポートの差し込み口が青くなっています。Mac の USB ポートの差し込み口が青くないのはなぜですか?
メーカーによっては、USB 3 のポートまたはデバイスであることを示すために差し込み口を青くしています。USB 3 をサポートする Mac の USB ポートはすべて USB 3 に対応しているため、青い差し込み口で区別する必要がありません。
サポートされる USB 3 デバイスにはどのようなものがありますか?
一般に「USB 3.0 仕様バージョン 1 (2008)」と呼ばれる仕様に準拠しているデバイスはすべてサポートされます。
USB 3 のハブはサポートされていますか?
一般に「USB 3.0 仕様バージョン 1 (2008)」と呼ばれる仕様に準拠しているハブはサポートされます。
USB 3 ポートのパフォーマンスを最大限に引き出すにはどうすればよいでしょうか?
ポートの構成は、そのポートに最初につなぐデバイスによって決まります。したがって、USB 3 に対応したハブまたはデバイスを最初に接続するようにしてください。
USB 3 ポートに最初に USB 2 デバイスをつなぐとどうなりますか?
USB 2 のハブを最初につないでしまうと、そのハブに接続または「連結接続」されるすべてのデバイスの転送速度が USB 2 の速度 (最大 480 Mbps) になります。
USB デバイスを USB ポートにつなぐとどうなりますか?
デバイスは USB 2 ポートと下位互換性があるはずです。ただし、USB 3 の速度は得られず、給電能力は 500 mA に制限される可能性があります。
Boot Camp で Windows 7 および 8 を使用する場合も USB 3 はサポートされますか?
はい。USB 3 は Windows 8 でもネイティブでサポートされています。詳しくは、こちらの記事 も参照してください。
Boot Camp で実行される Windows 7 で USB 3 が使用できるようにするには、Windows サポートソフトウェア (ドライバ) をダウンロードしてインストールします。詳しくは、こちらの記事 を参照してください。
USB 3 ケーブルとそれ以外の USB ケーブルに違いはありますか?
はい。USB 3 ケーブルには以下の特長があります。
ワイヤーの数がほかの USB ケーブルより多くなっています。このため、ほかの USB ケーブルに比べて少し厚みがあります。
挿入部分が青くなっています (たいていの場合)。
コンピュータ側のコネクタに「SS」の記号がついています。
デバイス側のコネクタが USB 2 ケーブルと異なります。
接続するデバイスに、より多くの電流 (900 mA) を供給できます。
USB 2 ケーブルを使って USB 3 デバイスをコンピュータに接続できますか?
ケーブルのデバイス側のコネクタがそのデバイスのコネクタに適合するなら接続できます。ただし、ワイヤーの数が少ないため、USB 3 の速度は得られず、給電能力が 500 mA に制限される可能性があります。
Mac で USB 3 デバイスを使用する場合のトラブルシューティング
デバイスが USB 3 デバイスとして接続されているかどうかはどうすれば確認できますか?
システム情報ユーティリティで確認します。 
USB 3 セクションと USB 2 セクションがあります。USB 3 デバイスは「USB 3.0 SuperSpeed」セクションに表示されます。それ以外のデバイスは「USB 高速バス」セクションに表示されます。詳細表示三角形をクリックすると、接続されたデバイスの詳細が確認できます。
Apple () メニューから「この Mac について」を選択します。
「詳しい情報」をクリックします。
「システムレポート」をクリックします。
システムレポートの「ハードウェア」セクションで、「USB」をクリックします。
仮想化ソフトウェアを使って Mac で Windows を実行していますが、USB 3 デバイスが USB 2 のスピードしか出ていないようです。何に問題があるのでしょうか?
仮想化ソフトウェアの中には、まだ USB 3 に対応していないものがあります。詳しくは、仮想化ソフトウェアの製造元に問い合わせてください。
USB 3 のデバイスが動作しません。または USB 3 バスに表示されません。何が問題だと考えられますか?
コンピュータで利用できる最新のソフトウェアアップデートがインストールされていることを確認してください。チェックするには Apple () メニューから「ソフトウェア・アップデート」を選択して、利用できるアップデートがあるかどうかを確認します。
デバイスを取り外し、再度接続することで問題が解決する場合があります。
デバイスを別の USB ポートにつないでみてください。
デバイスのメーカーからファームウェアアップデートが出ていないか確認してください。
デバイスに付属の AC アダプタがある場合はそれを使用してください。
コンピュータの再起動を試してください。
別の USB 3 ケーブルを試してください。
別の USB 3 ハブを試してください。
USB 3 デバイスがコンピュータに接続されていると、Wi-Fi または Bluetooth デバイスが正常に動作しないことがあります。何が問題だと考えられますか?
一部の USB 3 デバイス、特にハードドライブは、無線周波数干渉を引き起こすことがあります。これによって、2.4GHz 帯を使用する Wi-Fi および Bluetooth デバイスとコンピュータとの通信に問題が生じる場合があります。ハードドライブまたはその他の USB デバイスを、Mac の背後、画面のヒンジ近くに置かないようにしてください。そこには Wi-Fi と Bluetooth のアンテナが埋め込まれているので、その近くに USB 3 デバイスを置くと、ワイヤレス接続との干渉が起こる可能性があります。詳しくは、こちらのホワイトペーパー (英語) を参照してください。
