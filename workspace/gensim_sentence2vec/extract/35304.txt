「スタート」ボタンの説明
Windows のタスクバーに設置されたスタートボタンは、コンピュータに保管されたアプリケーションを起動するアイコンが収納された引き出しみたいなものです。
このボタンは、Windows 95 から搭載され、Windows Vista から Windows ボタンに変更されています。
「スタート」ボタンに矢印を移動させ左クリックして下さい。
キーボードの「Windows」ボタンを押しても同じ動作になります。
「スタート」ボタンの上にメニューが表示されます。
頻繁に使うアプリケーションのショートカットを貼り付けておくと、起動させやすくなります。
「アプリケーション」とは、ワープロソフトや表計算ソフトの事です。
Windows には、
ハードディスクドライブ
のメンテナンスツールや、ホームページを閲覧するためのブラウザが付属されています。
これらもすべて「アプリケーション」です。「アプリケーション」ファイルは、「.exe」という拡張子が付いています。
本来この「.exe」ファイルをダブルクリック ( マウスの左ボタン ) する事で、アプリケーションを起動させる事が出来ます。
しかし、沢山ある「.exe」ファイルから、必要なアプリケーションを探すのは大変ですよね。
「スタート」ボタンのメニューは、それらアプリケーションが簡単に探せる様に分類されています。机の引き出しだと思って下さい。
なお、Windows 8(ウインドウズ エイト)でスタートボタンは消滅します。その後、Windows 8.1 で復活しますが、スタートメニューは表示されず、デスクトップとスタート画面の切り替えに使われます。
今後のアップグレードで、本来のスタートボタンが復活するかも知れません。
それでは、どれくらい便利なのか実際に確認してみましょう。
「スタート」ボタンを左クリックして、「Internet Explorer (インターネット エクスプローラ)」に矢印を移動させて左クリックして下さい。
Internet Explorer が起動しましたね。これは、ホームページを閲覧するためのアプリケーションです。
今、Internet Explorer を起動するのに、マウスの左ボタンを何回押しましたか?
答えは「スタート」ボタンと「Internet Explorer アイコン」の、2回です。
次に「Internet Explorer」の右上にある「×」印に矢印を移動させ左クリックして閉じて下さい。
「Internet Explorer」がハードディスクのどこに保存されているか調べてみましょう。
「スタート」を左クリックして「Internet Explorer」に矢印を移動させ右クリックして下さい。
次に、メニューの「プロパティ」に矢印を移動させ左クリックして下さい。
「Internet Explorer のプロパティ」が表示されましたね ( 右画像 )。
ここにアイコンの詳細が表示され、設定を変更する事も可能です。
まず、上の「ショートカット」タブを左クリックして下さい。さらに、Internet Explorer アイコンの絵を見て下さい。左下に矢印の絵がありますね。
この矢印が付いているアイコンは「ショートカットアイコン」を表します。
つまり、アイコンは作成されていますが、Internet Explorer その物ではありません。
このアイコンを左クリックすれば、Internet Explorer が起動する設定になっているだけです。
当然このショートカットアイコンを削除しても、Internet Explorer そのものは削除されません。
また必要ならば、別の場所に Internet Explorer のショートカットを作成する事も可能です。
次に「リンク先」を見てみましょう。この右に書かれているのが、本物の Internet Explorer アプリケーションファイルがある場所です。
"C:\Program Files\Internet Explorer\IEXPLORE.EXE"
と書かれています。上の太字を参照してリンク先を辿ってみましょう。
「Internet Explorerのプロパティ」の右上「×」印に矢印を移動させ左クリックして閉じましょう。
デスクトップのマイコンピュータに矢印を移動させダブルクリック(マウスの左ボタン)して下さい。
「マイコンピュータ」が開いたら「ローカルディスク(C:)」に矢印を移動させ、ダブルクリックして下さい。
Cドライブの次は、「Program Files」フォルダに矢印を移動させダブルクリックして下さい。
「Internet Explorer」フォルダに矢印を移動させ左ダブルクリックして下さい。
"C:\Program Files\Internet Explorer\IEXPLORE.EXE"
上の太字を見て下さい。最後に「IEXPLORE.EXE」と書かれていますね。これは最後に「.exe」と書かれているのでフォルダではありません。
ファイルです。今開いたフォルダの中に、この「IEXPLORE.EXE」があるはずです。探してください。
見つかったら「IEXPLORE.EXE」ファイルに矢印を移動してダブルクリックして下さい。Internet Explorerが起動しましたね。
それでは「Internet Explorer」を閉じて下さい。今、ショートカットアイコンを使用せずに自力で「Internet Explorer」を起動させました。
何回マウスの左ボタンをクリックしたか覚えていますか?
正解は5回です。しかも、みなダブルクリックです。
この様に、本来のアプリケーションファイルをダブルクリックするよりも、ショートカットアイコンをクリックした方が早い事が理解できたと思います。
ショートカットアイコンを作成してみましょう!
どんどん話がずれて行く様な気がしますが、これが個人ホームページの面白い所でもあります。
上の「Internet Explorer」フォルダの中の、「IEXPLORE.EXE」ファイルに矢印を移動させ右クリックして下さい。
さらにメニューの「コピー」を左クリックして下さい。これで「Internet Explorer」フォルダを閉じて下さい。
デスクトップの何も無い場所に矢印を移動させ、右クリックして下さい。表示されたメニューの「ショートカットの貼り付け(S)」を、左クリックして下さい。
デスクトップに Internet Explorer のショートカットが作成されましたね。
これをダブルクリックすれば Internet Explorer が起動します。このように、好きな所に頻繁に使用するショートカットアイコンを作成する事が出来ます。
さらにショートカットアイコンをドラッグ&ドロップして、自分の作成したフォルダの中に入れる事も可能です。
今は練習で作成したのでショートカットアイコンを右クリックして削除して下さい。
プログラム
「スタート」ボタンに矢印を移動させ左クリックして下さい。
次に「プログラム」の所に矢印を移動させて下さい。ここはクリックしなくても、右にメニューが展開されます。
さらに「アクセサリ」「ゲーム」「スタートアップ」フォルダと「Outlook Express」などのショートカットファイルがあります。
この「プログラム」メニューの中は、アプリケーションをインストールすれば自然と増えていきます。
ショートカットをさらに分類した場所です。
この中で特殊なのが「スタートアップ」フォルダです。   
このフォルダの中に、アプリケーションのショートカットを置くと、Windows が起動した時に、アプリケーションが自動起動します。
左図は私のパソコンのスタートアップです。
TV キャプチャ(テレビ番組を自動でダビングするアプリケーション)やファイヤーウォール(インターネットのセキュリティ対策アプリケーション)などがあります。
パソコンを稼動させている間、常に動かしておきたいアプリケーションが、スタートアップフォルダに入ります。
しかし、自分で入れる事はまれで、アプリケーションインストール時に自動で組み込まれます。あえてここを変更するならば、邪魔な物を取る時です。
今はこんなファイルもあるのだと知るにとどめましょう。
「アクセサリ」フォルダの中には、文章を書く「ワードパッド」、メモをとる「メモ帳」、画像を描く「ペイント」、「計算機」
などのアプリケーションもあります。左クリックでいろいろ起動して試して下さい。ただしアプリケーションを開く時、
前に起動したアプリケーションを閉じましょう。アプリケーション2個ぐらい同時に起動しても大丈夫ですが、何個も同時に起動すると動きが遅くなります。
システムのメンテナンスには、「アクセサリ」フォルダの中の、「システムツール」フォルダに入っているアプリケーションをよく使用します。
得に「ディスク デフラグ」アプリケーションはハードディスクのメンテナンスに有効です。「ディスク デフラグ」の説明は別のページでさせていただきます。
