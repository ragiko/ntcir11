
先日購入した1GBの外付けHDD「BUFFALO スリム&プレミアム USB3.0用 ポータブルHDD 1TB ブラック HD-PCT1TU3-BJ」ですが、購入時の状態ですとWindows仕様にフォーマットされていますので、Macにつなげて使うには、Mac用にフォーマットする必要があります。

ちなみに、TONELIKOが使用しているMacBookのCPUはintel。OSはOS X v10.8 Mountain Lionです。PowerPC搭載のMacや、OSがMountain Lion以外の場合は参考になさらないでください。
それでは、フォーマットの手順を簡単に説明します。
【ご注意!】使用済みの外付けHDDを利用する場合、フォーマットするとデータは消去されてしまいますのでご注意ください。
1.外付けHDDをMacBookのUSB端子に接続する。
2.アプリケーションからユーティリティーを開き、「ディスク・ユーティリティー」を起動する。
3.「パーティション」タブをクリックして、左側のカラムから外付けHDDを選択
4.「パーティションのレイアウト」は「1パーティション」を選択
5.名前欄に任意の名前を入力し、フォーマットは「Mac OS 拡張(ジャーナリング)」を選択
6.「オプション」をクリックすると、「このディスクの用途にふさわしいパーティション方式を選択してください」という画面が出てくるので、「GUIDパーテーションテーブル」を選択して「OK」をクリック。
7.最後に［適用］をクリック。
これでMac用外付けHDDとしてのフォーマットが完了です。
参考1:フォーマットの形式
上記5のフォーマットには以下の選択肢がありますが、
Mac OS 拡張(ジャーナリング)
Mac OS 拡張(大文字/小文字を区別、ジャーナリング)
MS-DOS(FAT)
ExFAT
Macのみで使用する場合、基本的には「Mac OS 拡張(ジャーナリング)」でOKですが、他フォーマット形式に関してもちょっと調べてみました。
Mac OS 拡張(大文字/小文字を区別、ジャーナリング)
参考サイト:Mac OS X Server 10.3:「Mac OS 拡張(大文字と小文字を区別/ジャーナリング)」ファイルシステムの使用
特別なケース以外には必要ないみたいですね。
MS-DOS(FAT)
参考サイト:Mac OS X v10.6:MS-DOS (FAT32) フォーマットのディスクをホームディレクトリに使用する
Microsoftの規格であり、Windows XPまで使用されていた方式です。WindowsとMac OS Xは読み込むことも書き込むこともできます。ただし、1ファイルの最大容量は4GBまでという制約があるので、Macのみで使用する場合は選択する必要はありません。
ExFAT
参考サイト:MacとWindowsで外付けHDDを共有するなら「exFAT」フォーマットが最強!
参考サイト:Mac(特にLion)とWindowsの共存環境にはNTFSよりexFATの方が良かった件
MacとWindows両方で使いたい場合には、MS-DOS(FAT)よりExFATの方が使い勝手がよさそうですが、こちらもやはりMacのみで使用する場合はあえて選択する必要はないでしょう。
