USAAE 母音についてあまり議論も深めぬまま、USKOnS 子音へと視点を移す。が、ひとまずこの USVO 音声に関する話題も、ひとまずここで一区切りとしておこうと思う。子音についてもまだまだ議論の余地があり、かつ音声に関してまだまだ話題に尽きないけれど、私自信の頑張りが続かないことと、飽きを避けるため他の話題も書きたいという感じになったので、とにかくここで休憩とする。今後ももちろんゆっくりこの USVO 音声については機会があれば適宜述べることとする。文法再検討としては一応形態論やなんかを徐々にはじめ、できればコーパス作成とか語彙とかそういう方面についても述べはじめたい。文法についても重要な変更が成されている途上にあるので、こういう点についても見逃せない。
PLAnGI の「語彙」で扱うべき内容かもしれないが、一応ノシロ語で現在用意されている「音声学・音韻論」的な用語について一覧してみる。部首 US「音声」にある項目をまとめてみる。

このようにまとめてみると、この部首US「音声」には①音声学的なもの(【音声】【音の素性】)②音韻論的なもの(【音韻論】【音節構造】後者を音韻論的と考えるのは私見)③会話④音象徴⑤音楽関係、が属していると見られる。このうち当ブログ当項目で扱うべきは①と②である。会話や口語というのはもちろん「音声」の大前提だが、これそのものは「音声学」や「音韻論」で扱われるべき内容ではない。また、音象徴を言語学や音声学の重要な研究項目であると考える人々もいるようだが、当方は一切このような考え方には賛同していない。
【音声】の項に「音声」と「声」があるが、両者の区別は判然としない。当方の了解では、言語音として使用されるもの…「声」を、「音声」と呼ぶものと見ていた。あらゆる「音」一般ではない。なお、UISAn「音」という語もある。ここでは他の部首については考慮にいれていない。
さて、「子音」について考える。【音の素性】に関する語としてノシロ語では、(基本的に)子音に関する
【音の素性】
USKLOT	 閉舌音
USKLOV 閉鎖音
USNAVOKLOKOnS 無声閉鎖子音
USNAVO 無声音
USRIVO 有声音
しか用意されていないことがむず痒い。が、これは単なる語彙の不備で今後の拡充が期待されるものである。ノシロ語の「音声」を考える上で必要な術語がこれだけで充分ということを示すものではないだろう。また調音器官(あるいは調音点)そのものは、人体部位などの部首によって示されるものかと思われる。
「無声閉鎖子音」のみ語彙が作られているのはまず「無声閉鎖子音」がよく使われるということを示している可能性もある。が、「これを参考に他の子音(調音法)についても自由に造語可能」ということを示しているのかもしれない。そもそも造語法については手引きが不足している感があるが、これに倣えば
US(部首)-(RI/NA 有声/無声)-VO(「音声」)-(調音方法)-KOnS(「子音」)
という形で、やや冗長が認められるが子音をそれぞれ示せるのかもしれない。おそらく、母音もいずれ同様に開発可能かもしれない。実際にはさらに「-(調音点)」もどこかに付したいところで、これはおそらく-VO(「音声」)と-(調音方法)の間になるのではないかと思う。ノシロ語学の上では、不必要な素性について省略して、例えば鼻音など無声音が生じにくいと考えられるものについては-(RI/NA 有声/無声)無しでもいけるかもしれない。
なお、調音方法らしい語彙として「閉鎖音」と「閉舌音」のみしか用意されていないが、後者が何を示すものかはよくわからない。ノシロ語の単語は英語を参考に作られている可能性が高く、US(部首)-K(USKOnS子音?)- の後に英語の plosive(破裂音) > -LOV が付されたのが USKLOV 「閉鎖音」だとすると LOT は何を指しているのだろうか。T は tongue「舌」か?
さて用語・術語的な問題はさておき、子音を一覧して本項を閉じようと思う。ラテン字表記の文字を、IPA的な配列を参考に並べてみる(調音点は前(唇)から後ろ(口蓋垂〜咽喉)へ、という感じで、無声・有声の順に並べる)。無論これも、音声・音価に関する正確な記述があるわけではないので私による想像にすぎないし、あくまで文字の分類であるので、実現する音声については今後再検討が必要である。あくまで便宜的な分類と考えられたい。
【閉鎖音】(破裂音)
P,B ; T,D ; K,G
【鼻音】
M ; N
【ふるえ音 or はじき音】
R
【摩擦音】
F,V ; S,Z ; X,J ; H
【接近音・側面接近音】
L ; Y ; W
【破擦音】
C ; Q 
R のように、ふるえ音なのかはじき音なのか不明なものもあるし、日本語の感覚からすると破擦音に属するかもしれない Z ; J (C ; Q の対として)、H ももうちょっと色々な可能性があるだろう。先に述べた「無声閉鎖子音だけ語彙が用意されている理由」として、「無声閉鎖子音が多いから」ではないかとしたが、こうした微妙な音を摩擦音から排除すれば、「閉鎖音」が一番多いということになる。W も日本語の感覚からすれば二重調音の「有声両唇軟口蓋接近音」と考えたい。こうした個々の音声・音韻事情については、またいずれ述べることになるだろう。
さらに子音として忘れてはいけないのが、y と n である。前者は、凡そ「子音(文字)のあとについて当該子音を口蓋化する」機能があると思われる。これは前接の子音によってすべての子音に後続しうるものではないし、後ろに来る母音をも制限するらしい。他方 n は唯一の音節末尾専用の鼻音子音である。日本語の「ん」が想定されているもので、その音価はやはり数種類あるようである。
なお、「「音」という記事」でも述べたとおり、どういう音素が必要であるとか、この音素は無駄であるとか、そういう点は考えないこととする。ただ、さまざまな母語背景を持つ人が学ぶ上で、どんな問題が生じうるか、音価の解釈の可能性を幅広く考えるということは、することになるだろう。
