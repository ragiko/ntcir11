説明X = corrmtx(x,m) は、X'X が長さ n のデータ ベクトル x に対する自己相関行列の (バイアス付き) 推定となるような、(n+m) 行 (m+1) 列の四角形テプリッツ行列 X を返します。m は入力の長さ x より真に小さい正の整数でなければなりません。X = corrmtx(x,m,'method')  では、以下のように文字列 'method' で指定されるメソッドに従って、行列 X が計算されます。'autocorrelation': (既定の設定) X は (n+m) 行 (m+1) 列の四角形テプリッツ行列です。m 次の予測誤差モデルを基に、ウィンドウ操作を処理の "前" に適用したデータと "後" に適用したデータを使用して導出された、長さ n のデータ ベクトル x に対する自己相関の推定を生成します。'prewindowed': X は n 行 (m+1) 列の四角形テプリッツ行列です。m 次の予測誤差モデルを基に、ウィンドウ操作を処理の "前" に適用したデータを使用して導出された、長さ n のデータ ベクトル x に対する自己相関の推定を生成します。'postwindowed': X は n 行 (m+1) 列の四角形テプリッツ行列です。m 次の予測誤差モデルを基に、ウィンドウ操作を処理の "後" に適用したデータを使用して導出された、長さ n のデータ ベクトル x に対する自己相関の推定を生成します。'covariance': X は (n-m) 行 (m+1) 列の四角形テプリッツ行列です。m 次の予測誤差モデルを基に、ウィンドウ操作を処理の "ウィンドウを使用しない"データを使用して導出された、長さ n のデータ ベクトル x に対する自己相関の推定を生成します。'modified': X は 2(n-m) 行 (m+1) 列の修正四角形テプリッツ行列です。m 次の予測誤差モデルを基に、前方および後方予測誤差の推定を使用して導出された、長さ n のデータ ベクトル x に対する自己相関の推定を生成します。[X,R] = corrmtx(...) では、X'*X として計算された (m+1) 行 (m+1) 列の自己相関行列の推定 R も返されます。
