首を長くしてお待ちいただいていた皆様に朗報です。ついにSkype for Windows Phoneのβ版アプリが公開されます。今回のリリースによって、主要なスマートフォンOSのほぼ全てでSkypeが利用可能になります。ゴールド版アプリは4月に公開されますが、本日より、アプリをダウンロード*して Skype for Windows Phoneのβ版をお試しいただけます(リンクはWindows Phoneで開くことができます)。ご意見がございましたらこちらからぜひお寄せください。
Skype for Windows Phoneのβ版**は、ユーザーの皆様には欠かせない非常に重要なSkype機能をサポートします。さらに、Windows Phoneに採用されているMetro UIの操作性はシンプルでわかりやすく、ご満足いただける仕上がりです。大切な人と今まで以上に簡単にコミュニケーションを取ることができます。
最新のSkype for Windowsのβ版アプリの詳細については、動画をご覧ください。
Skype for Windows Phoneのβ版があれば、Skypeユーザー同士の無料の音声・ビデオ通話***が3G・4GおよびWiFiネットワーク****上でご利用いただけます。さらにSkypeクレジットを使って国内・海外の固定電話と携帯電話に格安で通話がかけられ、1対1またはグループのチャットもお使いいただけます。他にもプロフィールやアカウント情報の更新など、さまざまなSkypeの機能をご利用いただけます。
Skype for Windows Phoneはまだスタートしたばかりです。今後ますます進化を続けていきます。ユーザーの皆様に優れた利便性を提供する、Skypeを搭載したWindows Phoneのポテンシャルは計り知れません。
Skype for Windows Phoneのβ版は、Windows Phone端末からこちらをクリックして無料*****でダウンロードしていただけます。また、下記のQRコードからもダウンロード*が可能です。なお、本アプリのご利用には、Windows Phoneのバージョン7.5が必要です。現在ご利用のWindows Phoneのバージョンがご不明の場合は、[設定] > [情報] をタップして確認できます。
