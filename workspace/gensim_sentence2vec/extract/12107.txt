文書長が大きく異ると、助詞や読点の頻度も当然変わってきて良くないので、全ての作品の先頭5,555語のみを使います。
以上のテキストから、8種類の助詞と読点の組の出現頻度を数えます。
載せませんが、8次元の頻度表が出来上がります。
この表をじっくり眺めて特徴を掴むのは人間には厳しいので、ここで主成分分析を使い、第一主成分と第二主成分だけを抽出し、2次元平面に作品を配置します。2次元平面へのプロットを見ると、上部に芥川クラスタ、下部に森クラスタ、右部に太宰クラスタ、左部に夏目クラスタと、上手く分かれています。
実は、これら4人の助詞と読点の使い方をよく見ると、それぞれ特徴があります。
みんな「が」の後に「、」を打つよりも「て」の後に「、」を打つことが多いのですが、芥川作品には「が」の後に「、」を打つことの方が「て」の後に「、」を打つよりも多いという特徴があります。
太宰作品には、「は」の後に「、」が特に多いという特徴があります。
森作品には、「て」の後に「、」が特に多いという特徴があります。
夏目作品は、全体的に「、」が少ない傾向があります。
8次元の助詞と読点の頻度データが、2次元に削減されても上手く特徴を捉えています。
ソースコード
# RMeCabの導入
install.packages("RMeCab", repos = "http://rmecab.jp/R")
library(RMeCab)
# 単語2-gramの抽出
bigram <- docNgram("/path/to/corpus", type = 0, N = 2)
bigram_freq <- bigram[rownames(bigram) %in% c("[と-、]", "[て-、]", "[は-、]", "[が-、]", "[で-、]", "[に-、]", "[ら-、]", "[も-、]"),]
# 主成分分析
bigram_freq.pc <- princomp(t(bigram_freq))
# ラベルの準備
bigram_freq.name <- unlist(strsplit(colnames(bigram_freq), ".txt"))
# 描画
plot(bigram_freq.pc$scores[, 1:2], type = "n")
text(bigram_freq.pc$scores[, 1:2], lab = bigram_freq.name)
