
Excel     
で作成した住所録を検索する 
ここでは、シートに保存されている住所録などを、別の      
Excel のシートで検索する方法をご紹介いたします。   
今回は、名前で検索して表示します。  
結果サンプル  
1. 住所録シートの作成     
シート名を「住所録」に変更します。  
横一行にその人のデータを入力します。  
2. 検索シートの作成     
以下の画面のように表示したい情報の見出しを作成します。  
次に「名前で検索した郵便番号を表示する」というような機能を作成します。  
「B3」をクリックして、次の文字を入力します。  
=vlookup     
そして、画面の = ボタンを押します。     
すると下のような画面が出てきますので、次のように入力します。 
検索値    : 検索するセル番号。   
範囲       :    
検索する範囲。今回の場合、「住所録」の A2〜D6    
を検索します。 
列番号    :    
表示する列の番号。「郵便番号」の場合は、左から 2    
番目なので 2 をいれます。   
検索の型 : TRUE か FALSE を入力します。FALSE    
をいれると、完全に一致した文字を検索します。 
同じように B4 B5    
に「住所」と「電話番号」を作成します。 
3. 完成   
ここまでできたら完成です。 
実際に検索できるかためしてみましょう! 
検索シートの「名前」の右側 B2 に "木村" 
と入力すると、「郵便番号」「住所」「電話番号」が検索されて表示されます。 
4. 補足   
・vlookup の範囲指定で、   
を押すと、マウスで選択範囲を選べます。わざわざキー入力する必要はないです。  
・vlookup の範囲で「住所録!A:D」と入力しておくと、今後「住所録」にデータを追加しても、検索ができるようになります。改めて、検索範囲を指定しなおす、手間が省けます。
