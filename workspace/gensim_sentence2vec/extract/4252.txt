概要 - iOS app / fo-Onmon 
fo-Onmon - サウンド・スペクトログラム
「fo-Onmon」は、iPad上で動作する音声/オーディオ向けスペクトログラム・アナライザーです。音源の時間変化による周波数成分の変動をスペクトログラム表示で、グラフィカルに確認することができます。
スペクトログラムは、横軸に時間、縦軸に周波数を表すような2次元の幾何学図形イメージであり、さらに特定時間、特定周波数でのエネルギー強度をピンポイントのカラー情報によって表現します。本アプリのトレース表示では、横軸が時間、縦軸が周波数(上に行くほど音が高い)、表示カラーが赤色に近いほど音が強いことを表し、音源の時間的な変化、音色、高さ、大きさを同時に読み取ることができます。
最新バージョン(Ver2.0)で、音声分析機能(ボイスモード)が新たに追加され、人の声に対するケプストラム解析など様々な分析ができるようになりました。
オーディオモードは、従来機能を踏襲した動作モードで、オーディオ信号を対象とした周波数成分解析をおこない、スペクトラムおよびスペクトログラムを同時表示します。あなたの周りの音、たとえば、環境音や音楽、鳥の鳴き声などに隠されたビジュアル・イメージを発見するために活用できます。オーディオシステムにおいては、不快な高周波ノイズを捕捉するのにも利用できます。 
ボイスモードは、音声信号を対象とした周波数成分解析をおこない、スペクトラム、ケプストラム、エンベロープ、フォルマント、スペクトログラムを同時表示できます。声や会話、動物の鳴き音などの可視化分析に使用できます。

Mode 
MODEは、オーディオモード(Audio mode)とボイスモード(Voice mode)を切り替える機能です。
各モードのビューにMODEボタンがあり、それぞれをタッチすることでモード間を移行します。アプリ起動時にはオーディオモードで起動されます。
Audio Mode - Top View
アプリ起動時には、オーディオモードのビューが表示されます。
オーディオモードのビューには8つの操作ボタンとトレースディスプレイ(TRACE DISPLAY)があります。
オーディオモードの操作ボタンは、最上段が電源(Power)、2段目が最大値ホールド(HOLD)、3段目がトレース表示機能(FUNCTION)、4段目が表示レンジ設定(RANGE)です。さらに、5段目が入力信号モニター(MONITOR)、6段目が一時停止(PAUSE)、7段がトレース・キャプチャ(SNAPSHOT)です。
最下段に動作モード切替ボタン(MODE)があり、タップするとボイスモードへ移行します。
Audio Mode - Power
POWERは、オーディオモードでの測定機能をON/OFFする設定です。
POWERをタップすることでONとなり、Powerインジケータが点灯し、スペクトラム解析を開始します。TRACE DISPLAYも測定表示を開始します。再度、POWERをタップすることでOFFになります。
Audio Mode - Hold 
HOLDはトレース表示のピークホールド機能の設定です。
HOLDは各周波数バンドの最高値を保持し続ける機能です(Max HOLD)。HOLDをONにすることで、音響信号のスペクトルの最大累積値を長時間にわたって観測することができます。デフォルトはOFFに設定されています。
Audio Mode - Function 
FUNCTIONはトレース表示機能の設定です。
FUNC.は信号解析方法としてピークレベルまたは電力平均レベルのいづれかから選択します。瞬間的なピーク値を観測したい場合にはPEAKを、電力平均レベルで観測したい場合にはLEVELを設定してください。デフォルトはPEAKに設定されています。
Audio Mode - Range
RANGEは表示レンジの設定です。RENGEボタンをタップすることで、表示レンジを選択できます。
TRACE DISPLAYにおける信号強度のダイナミックレンジは96dBです。RENGE設定により、0dBと-40dBの基準レベルでの信号強度で観測することができます。デフォルトは0dBに設定されています。
Audio Mode - Trace Display
TRACE DISPLAYは特定時間の周波数ごとのスペクトログラムをカラー表示します。横軸上に時間、縦軸上に周波数(上に行くほど音が高い)を、各周波数の信号強度をカラー表示(青色に近いほど音が弱く、赤色に近いほど音が強い)で描画します。
Audio Mode - Monitor
MONITORは入力ソース信号のモニター機能です。
MONITORボタンをタップするとONになり、入力された音響信号をスルーでモニター出力することができます。再度、MONITORをタップするとOFFになり、デフォルトはOFFです。
モニター出力する場合、内蔵スピーカーを使用すると、ハウリング等のノイズが発生する可能性があります。そのため、ヘッドフォン等を外部接続することを推奨いたします。
Audio Mode - Pause
PAUSEは、TRACE DISPLAYの測定画像表示の更新を一時停止させる機能です。
PAUSEボタンをタップするとONになり、瞬間的な測定結果の確認にご利用いただけます。
再度、PAUSEをタップするとOFFになり、トレース画像表示の更新を再開します。なお、PAUSEの設定状態によらず、スペクトラム測定機能は継続して動作しています。
Audio Mode - Snapshot
SNAPSHOTは、現在測定中のトレース画像を静止画としてキャプチャする機能です。
SNAPSHOTボタンをタップすると、TRACE DISPLAYの画像表示をカメラロール(写真)にPNG形式でファイル保存されます。ファイル名はIMGxxx.pngとなります。
なお、「fo-Onmon」の初回起動時に、SNAPSHOTボタンを押したときには、カメラロールへのアクセス許可の確認がiOSより求められます。もし許可をしなかった場合には、iOSの「設定」->「プライバシー」->「写真」にアクセス設定がありますので、こちらで許可に設定してください。
Voice mode - Top view 
モード選択ボタンをタップすると、音声解析をおこなうボイスモードのビューが表示されます。
ボイスモードのビューには4つの操作ボタンと5つのトレースディスプレイ(TRACE DISPLAY)と製品情報表示があります。
操作ボタンは1段目が録音(REC)、2段目がデータ(SET/RESET)、3段目が再生・停止(PLAY/STOP)、最下段がモード切替(MODE)です。モード切替ボタン(MODE)をタップするとオーディオモードへ移行します。
ボイスモードでの音声解析は、マイク録音した音声データに対して、その周波数成分分析をおこないます。
録音ボタン(REC)で録音データを保存し、解析用データとしてデータ読み込み(DATA->SET)することで、音声解析の準備が整います。実行ボタン(PLAY)で、音を再生しながら、解析結果を5つのディスプレイに同時に表示します。音声解析実行中に一時停止も可能です。
<音声解析の操作手順>
音声データの録音準備 : 録音 (REC) -> 録音停止 (STOP)
音声データのセット  : データセット (DATA/SET) 
音声解析の実行    : 実行 (PLAY)
ディスプレイ・クリア : データリセット (DATA/RESET)
<音声解析結果のディスプレイ表示>
FFTスペクトラム表示  : 入力信号のFFTスペクトラム波形を表示します。
ケプストラム表示     : ケプストラムを表示します。 
エンベロープ表示     : 入力信号のスペクトラム包絡を表示します。 
フォルマント分布     : フォルマント分布におけるF1、F2の周波数マトリックス・ポイントを表示します。
スペクトログラム表示 : エンベロープのスペクトログラムを表示します。 
Voice mode - Recording 
録音ボタン(REC)をタップすると録音を開始し、再度タップすると録音を停止します。
録音できる時間は最大3分です。
Voice mode - Data 
録音完了後、データのセットボタン(SET)をタップすると録音データを解析データとしてセットします。
リセットボタン(RESET)をタップすると、ディスプレイ表示がクリアされます。
Voice mode - Play 
録音データが解析データとしてセットされている場合、PLAYボタンがタップ可能になります。Playボタンをタップすると、音声解析をおこない、解析データを各ディスプレイに表示します。
Voice Mode - Info
Infoは、製品情報を表示します。
"(to toon.llc website)"をタップすると、Safariが起動してサポートページにジャンプします。
"(to App Store)"をタップすると、App Storeが起動して製品ページが表示されます。
