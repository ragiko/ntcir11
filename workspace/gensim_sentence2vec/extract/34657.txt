インターネット セキュリティ システムズ(ISS)は6月17日、ネットワーク型の不正侵入検知システム(IDS)、「RealSecure Network Sensor」の新バージョンを発表した。
新バージョンのRealSecure Network Sensor 7.0は、ISSによる米ネットワークICE買収の成果を反映し、両社の技術を統合した製品となっている。
従来のシグネチャベースの不正侵入検知に加え、レイヤ7レベルのプロトコル分析と異常検出(アノーマリ検出)機能を搭載した。これにより、既知の攻撃のみならず、未知の新しい攻撃も検出し、対応することができる。
ほかに、パケットフィルタとイベントフィルタの両方をサポートすることで、誤報の数を削減したほか、オープンソースのIDS「Snort」向けのルールの読み込み機能、不正アクセスの試みが成功したかどうかを検証する機能などが搭載された。
さらに、かねてよりシグネチャベースのIDS製品の課題とされていたパフォーマンスを改善。100Mbpsフルからギガビットまでをカバーするという。
RealSecure Network Sensor 7.0の価格は140万円から、またギガビットに対応したRealSecure Gigabit Network Sensor 7.0は、780万円から。7月1日より、まずWindows 2000対応バージョンが出荷される。
なおISSではRealSecure Gigabit Network Sensor 7.0のリリースを記念して、7月1日から2003年3月末まで、同製品を429万円で提供するキャンペーンを行う。
