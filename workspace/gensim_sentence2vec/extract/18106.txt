相互参照を挿入する場合、通常は相互参照エレメントを使用して他のエレメントを参照します。ソースエレメントとしては、文書内で定義されているエレメントに応じて、節全体、章全体、表や図、段落または段落内の特定の場所(定義など)を使用できます。
注意:エレメントを相互参照のソースとして使用できるのは、そのエレメントに固有の ID 属性が割り当てられている場合だけです。ID 属性が割り当てられていないソースへの相互参照を挿入する場合は、そのソースエレメントの定義に ID 属性を追加するよう開発者に依頼してください。
エレメントでない相互参照を挿入する方法、および段落や段落内の特定のテキストを参照する方法については、相互参照の挿入を参照してください。エレメントを使用して相互参照を挿入するには:
1  ソースエレメントが他の文書内にある場合は、その文書を開きます。ソースエレメントの ID 属性に値が割り当てられていない場合、参照先文書に対する書き込み許可がないと、FrameMaker が ID を設定できません。2  相互参照エレメントを挿入する場所をクリックします。相互参照はテキスト枠には挿入できますが、テキスト行(テキスト行描画ツールで作成したグラフィックオブジェクト)には挿入できません。3  エレメントカタログで相互参照エレメントを選択して、「挿入」をクリックします。  
スペシャル/相互参照を選択して、エレメントを挿入することもできます。使用可能なエレメントが複数ある場合は、相互参照ダイアログボックスのエレメントタグポップアップメニューからエレメントを選択します。
4  他の文書内の情報を相互参照する場合は、文書ポップアップメニューからその文書の名前を選択します。5  ソースの種類ポップアップメニューから、「エレメント(順にリスト)」または「エレメント(ID でソート)」を選択します。ソース文書のエレメント定義で ID 属性が定義されているすべてのエレメントのタグが「エレメントタグ」スクロールリストに表示されます。
