音素ごとの特徴に応じて異なる処理を行い、各音素の特徴を十分に表現した音声合成を行うことが可能な音素符号補正装置、音素符号データベース、および音声合成装置を提供する。To provide a phoneme code correction device, a phoneme code database, and a voice synthesizer which synthesize a voice sufficiently representing features of respective phonemes by performing processing different by features of respective phonemes. - 特許庁
音声合成装置は、同じ音素記号で表される音素毎に複数の異なるピッチを持つ音素片を記憶する記憶手段と、前記音素片をピッチをインデックスとして読み出す読出し手段と、前記読み出された音素片に基き音声を合成する音声合成手段とを有する。The voice synthesizing device has a storage means which stores phoneme pieces having different pitches by phonemes represented as the same phoneme symbols, a readout means which reads the phoneme pieces out by using the pitches as indexes, and a voice synthesizing means which synthesizes a voice according to the read phoneme pieces. - 特許庁
ここで、音声検出基準データは、音素(または音素グループ)ごとに異なるデータが格納されており、音声検出部130は音素(音素グループ)ごとに、その音素等の検出に好適な検出基準を用いて始点や終点の検出を行う。Here, data which are different by phonemes (or phoneme groups) are stored as the speech detection standard data and the speech detection part 130 detects the start point and end point of each phoneme (phoneme group) by using detection standards suitable for detection of the phonemes etc. - 特許庁
