スタイルシート、ちょっとだけ解説
スタイルシートの3つの実装方法
実装方法とは、HTML文書がスタイルシートを認識するための方法のことです。
タグの説明の時に、スタイルシートはHTML文書のヘッダ部に<STYLE>タグを使って書いてきました。これも一つの実装方法なのですが、この他にも実装方法があります。
スタイルシートは大きくわけて三つの実装方法があります。今回はその説明です。
1.外部スタイルファイルを使った実装方法
2.<STYLE>タグを使った実装方法
(今までの説明で使ってきた方法です。)
3.各タグの属性にスタイル属性を使った実装方法
上記の三つの方法があります。順番に説明します。
1.外部スタイルファイルを使った実装方法
スタイルシートの記述を別ファイルにして保存し、各HTML文書でこのファイルを参照する方法です。一番お勧めの方法です。
次のようなスタイルシートの場合、黄色の部分だけをテキストファイルで、保存して下さい。保存する際のファイル名は自由ですが、拡張子を「.CSS」にして下さい。
<STYLE TYPE="text/css">
BODY { background-color : #238E23 }
H1,H2 { color : yellow }
H3,H4 { color : red }
H5,H6 { color : white }
</STYLE>
このスタイルシートを"stlsht.css"という名前をつけたとします。このスタイルシートを、各HTML文書で参照できるようにするには、ヘッダ部に
<LINK REL=STYLESHEET HREF="stlsht.css" Type="text/css">
と書きます。下がその例です。
<HTML>
<HEAD>
<META HTTP-EQUIV="CONTENT-TYPE" Content="text/html;charset=Shift_JIS">
<LINK REL=STYLESHEET HREF="stlsht.css" Type="text/css">
</HEAD>
<BODY>
<H1>やっぱりお好み焼きが好き♪</H1>
<H3>たこ焼きには辛子をつけよう!</H3>
<H5>いか焼きにはソースをたっぷりつけて♪</H5>
</BODY>
</HTML>
【ちょっと注意】
実際に外部スタイルファイルを参照するホームページをUPすると、スタイルシートを認識してくれない場合があります。これは、プロバイダのサーバがスタイルシートを参照するための設定をしていないためです。
この場合、AddType text/css .cssという1行をテキストエディタで書き、「.htaccess」というファイル名をつけて、スタイルファイルがあるディレクトリに置いてください。
2.<STYLE>タグを使った実装方法
これは、このホームページでタグの説明の時に使っていた方法です。HTML文書の中に<STYLE>タグを使って直接記述します。
<STYLE>タグはHTML文書中のどこに書いてもいいのですが、ヘッダ部に書くのが一番いいでしょう。
HTML文書に下のように書きます。
<HTML>
<HEAD>
<META HTTP-EQUIV="CONTENT-TYPE" Content="text/html;charset=Shift_JIS">
<STYLE TYPE="text/css">
BODY { background-color : #238E23 }
H1 { color : yellow }
H2 { color : red }
H3 { color : white }
</STYLE>
</HEAD>
<BODY>
<H1>モダン焼きもいいけど、オーソドックスなぶた玉もおいしい</H1>
<H2>小さい頃、天カスがどこに消えるのか謎だった</H2>
<H3>紅生姜のみじん切りも忘れずに</H3>
</BODY>
</HTML>
<STYLE>タグの属性・属性値にTYPE="text/css"を書きます。これは、「スタイルシートがテキスト形式で、CSSという言語でかかれている」ということを意味しています。
3.各タグの属性にスタイル属性を使った実装方法
これは、HTML文書で使う各タグの属性に「STYLE」属性を使うものです。すべてのタグに「STYLE」属性を使用できます。
でも、この方法はあまりお勧めしません。これは、スタイルシートを使わない従来のタグの記述と同じです。この方法を使い過ぎると、HTML文書がタグだらけになり、見づらくなります。(でも、結果オーライで、ちゃんと表示されればどんな方法でもいいんだけど・・・)
下のようになります。
<HTML>
<HEAD>
<META HTTP-EQUIV="CONTENT-TYPE" Content="text/html;charset=Shift_JIS">
</HEAD>
<BODY>
<H1 STYLE="color : red ">でも、宇都宮の餃子もおしかったなぁ♪</H1>
<H3 STYLE="color : blue">皮がパリッとしてて、具がジューシーで</H3>
<H5 STYLE="color : yellow">なんだか、急にウニが食べたくなった。</H5>
</BODY>
</HTML>

これで、この節の説明は終わりです。この節で言いたかったことは、
スタイルシートには3つの実装方法がある。
1.外部スタイルファイルを使った実装方法。
2.<STYLE>タグを使った実装方法。
3.各タグの属性にスタイル属性を使った実装方法。
