以下の点をご確認ください。
1.
Windows Media Centerを終了し、開き直してどうか。
また、パソコンを再起動してどうか。
2.(USB接続: GV-MC7/VZ 、GV-MC7/HZ3の場合)
前回使用時から、接続するUSBポートを変更していないか。
USBポートを付け替えた場合は、Windows Media Centerを閉じて、
前回接続していたUSBポートにつなぎ直してください。
※付け替えたUSBポートで使用したい場合は、Windows Media Centerの
テレビの設定をし直す必要があります。
Windows Media Centerを起動し、[タスク]-[設定]-[テレビ]-[テレビ信号]
[テレビ信号の設定]を選び、画面に従って設定をし直してください。
3.
Windows Media Centerの設定をし直してどうか。
Windows Media Centerを起動し、[タスク]-[設定]-[テレビ]-[テレビ信号]
[テレビ信号の設定]を選び、画面に従って進めます。
4.(GV-MC7/VZの場合のみ)
製品が誤認識された状態となっていることが考えられます。
以下の操作を行ってください。
1) 製品がパソコンのUSBポートにつながれていることを確認します。
2) Windowsメニューから、「コンピューター」を右クリックして[プロパティ]を選択します。
3) [デバイスマネージャ]をクリックします。
4) 一覧中ほどの[サウンド、ビデオおよびゲームのコントローラー]の左側のをクリックして開きます。
5) 開いた中に「SAA7164A Multimedia Blaster」という名称がないか確認し、
ありましたら、その名称を右クリックし「ドライバーソフトウェアの更新」を
クリックします。
6) 「コンピュータを参照してドライバソフトウェアを検索します」をクリックします。
7) 「コンピュータ上のデバイスドライバの一覧から選択します」をクリックします。
8) モデルとして表示された中から「I-O DATA GV-MC7/VZ」をクリックし、
[次へ]をクリックします。
9) しばらくして更新が終了した画面となったら[閉じる]をクリックし、
開いている画面も閉じます。
以上で操作は終了です。
改めてWindows Media Centerを起動し、[タスク]-[設定]-[テレビ]-[テレビ信号]
[テレビ信号の設定]を選び、画面に従って進め、ご利用をお試しください。
