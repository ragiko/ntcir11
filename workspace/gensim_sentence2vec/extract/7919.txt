Windows 7 では、ワイヤレス ネットワークとそのセキュリティ設定の作成と構成を簡略化するため、Windows Connect Now をサポートしています。この機能により、ユーザーは USB フラッシュ ドライブにネットワーク構成の情報を保存できます。ワイヤレス ネットワークを構成するには、ユーザーは、まず、ワイヤレス ネットワークの設定を収集するネットワーク セットアップ ウィザードを実行します。続いて、Windows では、セキュリティで保護されたワイヤレス ネットワークに対応するため、認証と暗号化の設定を構成して、フラッシュ ドライブに構成を保存します。 新しいコンピューター (Windows XP SP2 以降、Windows Vista、または Windows 7 を実行しているコンピューター) を、ワイヤレス ネットワークに追加するのは、各コンピューターにフラッシュ ドライブを接続するのと同じくらい簡単です。ワイヤレス ネットワークでドメイン メンバーのコンピューターを構成するには、グループ ポリシーを使用することをお勧めしますが、フラッシュ ドライブを使用すると、暗号化されたワイヤレス ネットワークへのアクセスを Guest ユーザーに許可できます。ただし、イントラネットを保護するために、ワイヤレス ネットワークは社内ネットワークから分離する必要があります。 ユーザーが Windows Connect Now ウィザードにアクセスできないようにするには、"コンピューターの構成\管理用テンプレート\ネットワーク\Windows Connect Now" ノードまたは "ユーザーの構成\管理用テンプレート\ネットワーク\Windows Connect Now" グループ ポリシー ノードにある、[Windows Connect Now ウィザードのアクセスを禁止する] ポリシーを使用します。 また、"コンピューターの構成\管理用テンプレート\ネットワーク\Windows Connect Now" ノードには、[Windows Connect Now を使用したワイヤレス設定の構成] ポリシーも用意されており、この設定では次のオプションを使用できます。WCN を使用したイーサネット (UPnP) 経由の構成を無効にする: Windows で、UPnP をサポートするネットワーク デバイスを構成できないようにします。バンド内 802.11 Wi-Fi 経由の WCN を使用した構成を無効にする: Windows で、ワイヤレス ネットワーク デバイスを構成できないようにします。USB Flash ドライブを使用した構成を無効にする: Windows で、Windows Connect Now の構成を USB フラッシュ ドライブに保存できないようにします。USB フラッシュ ドライブに保存される Windows Connect Now 情報には、コンピューターにセキュリティで保護されたワイヤレス ネットワークへのアクセスを許可する情報が含まれるため、ワイヤレス ネットワークのセキュリティを強化するには、この設定を無効にします。Windows ポータブル デバイス (WPD) の構成を無効にする: Windows で WPD を構成できないようにします。WPD には、ポータブル メディア プレーヤー、デジタル カメラ、携帯電話などが含まれます。許可される WCN デバイスの最大数: Windows を実行しているコンピューターで構成できる、Windows Connect Now デバイスの数を制限できます。複数のメディアによって検出されたデバイスに対する優先順位の高いメディア: デバイスを有線ネットワークとワイヤレス ネットワークの両方で使用できる場合、どちらの種類のネットワークを使用するかを判断します。Windows Connect Now を使用する予定がない場合は、簡単に無効にできます。Windows Connect Now 関連のグループ ポリシーの既定の設定を使用すると、すべての Windows Connect Now の機能が有効になります。
