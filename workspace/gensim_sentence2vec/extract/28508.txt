音声認識装置において、音響認識の結果の信頼度によって、統計情報に一時的に無効にすることにより、連鎖的な認識誤りが発生するのを防ぐことを目的とする。To prevent generation of chain-reacted recognition errors by temporarily invalidating statistical information in accordance with the reliability of the result of acoustic recognition in a voice recognition device. - 特許庁
音声認識に代表される認識結果に誤りが含まれるテキストを入力としてマイニング処理した場合に、その認識の信頼度をテキストマイニング結果と共に利用者へ提供する。To provide, when text containing an error in a recognition result represented by speech recognition is inputted and processed for mining, degree of its reliability in the recognition to a user together with a text mining result. - 特許庁
認識エラーや、誤認識、記述ミスを自動的に修正することができ、修正作業を省力化できる文字認識における誤り修正方法を提供する。To provide an error correcting method for character recognition which can automatically correct a recognition error, misrecognition and a description miss to facilitate the correcting operation. - 特許庁
操作者は、音声出力される認識結果が誤りであると判断すると、認識音声が出力されている間に、誤認識された単語を修正するための音声入力を行う。When it is judged that the recognized result voice-outputted is erroneous, an operator performs voice-input for correcting a word being erroneously recognized while recognized voice is being outputted. - 特許庁
認識文字列中の誤りを文字列単位で一括して且つ効果的に修正し得ると共に、修正後の文字列の影響が他の文字列に波及するのを効果的に抑制し、もって、適正な認識文字列候補を簡易且つ迅速に確定し得る文字認識装置および文字認識方法を提供すること。To provide character recognizing device/method for collectively and effectively correcting an error in a recognized character string, effectively suppressing the influence of the character string after correction to the other character string and easily and speedily deciding an appropriate recognized character string candidate. - 特許庁
