pause_handler
(このソフトを利用するには「音声フアイル」と「それに当てて
[pause_setter] で作成したデータ」を設定する必要があります。)
英単語、英文等の音声ファイルを使用した学習に便利なように作成したソフトです。
一枚の音声フアイルを区切って再生させることの出来るソフトです。
ソフトの機能で動作させる物で音声フアイルを加工するものではありません。
音声フアイルの再生中に「 [pause_setter] で作成したデータ」をもとに
ポーズ若しくは停止を自動的に動作させます。
[pause_setter] には無いいろいろな動作をさせることが出来ます。
逆順再生、スキップ再生、ランダム再生、エンドレス再生などです。
音声フアイルのパターンとニーズに合わせて動作を選択して下さい。
途中で終了した場合、次の起動時にはその続きから始まります。
再生できる音声ファイルは、拡張子が WAV,MP3,WMA のものです。
--------------------------------------------------------------------------------
以下のような理由で作成しました。
音声ファイルの内容が [ 訳語 ] + [ 英文 ] + [ 訳語 ] + [ 英文 ] + … とか
[ 英文 ] + [ 訳語 ] + [ 英文 ] + [ 訳語 ] + [ 英文 ] + … とか
[ 英文 ] + [ 英文 ] + [ 英文 ] + [ 英文 ] + … の様に並んでいる、
もしくは [ 長文 ] である場合に、各区切れ目に(ポーズ)無音区間が無い、
あるいは充分でない時、いちいち句切れの度に再生ボタンと停止ボタン、
あるいはポーズボタンをクリックするのは面倒なので、
各区切れ目に任意の無音区間を動作させたい。あるいは、
いちいち停止させる事を意識していると練習に集中する妨げになるので
再生ボタンをクリックした後は自動的に区切れ目で停止して欲しい。
そして逆順再生、スキップ再生、ランダム再生、エンドレス再生などもさせたい。
しかしその四つの機能は、構造上 [pause_setter] には実装しにくかった。
--------------------------------------------------------------------------------
インストール
適当な場所にフォルダごと解凍してお使い下さい。
本体は [ pause_handler.exe ] です。
デスクトップにショートカットを作成しておくと便利です。
--------------------------------------------------------------------------------
使う前の準備
当ソフトを起動し設定ボタンをクリックして設定エリアを開きます。
そして「使用したい音声フアイル」とそれに充てて 「pause_setter で作成した
区切れ目ポジションデータを保存したテキストフアイル」を登録します。
登録の仕方は pause_setter と同様、各入力ボックスの横のボタン、
こちらでは、[ 参照 ] ボタンから開いて登録するか、あるいは
フアイルを直接、設定エリアにドロップします。
この時、それぞれ一枚ずつドロップして下さい。
2枚以上同時にドロップするとエラーになります。
登録が済んだらエリアを閉じて下さい。
登録直後のポジションは先頭位置になっていますので。
そのまま最初から聞くことが出来ます。
使用上の注意。
再生操作中に、その停止のタイミングにタイトルバーをマウスでグリップしていると
その停止をスキップしてしまいますのでご注意下さい。
各ボタンの説明
本体部
[ 再生 ]
音声フアイルを次の区切れ目ポジションまで再生します。
[ 無音で ] 
再生せずに次の区切れ目ポジションまで移動します。
テキストが入力されていれば、テキストのみ表示されます。
[ |… ]
ポジションを一番先頭の区切れ目区間まで戻します。
[  ]
ポジションを今のポジションの一つ手前に戻します。
今のポジションは再生終了後のポジションであることに留意して下さい。
要するに一回クリックして再生すると、今の音声がもう一度流れます。
一つ手前に戻すには二回のクリックが必要であるということです。
三回クリックすると二つ手前に戻るということです。
[  ]
ポジションを今のポジションの一つ先に送ります。
今のポジションは再生終了後のポジションであることに留意して下さい。
要するに一回クリックして再生すると、二つ先の区間の音声が流れます。
一つ先の音声は普通に再生ボタンをクリックすればいいわけです。
二回クリックすると三つ先に送ることになります。。
[ 10 ]
ポジションを今のポジションから10先に送ります。
クリックするごとに10送られます。
[ …| ]
ポジションを一番最後の区切れ目区間まで進めます。
主に逆行再生の時に使用することになります。
[ リピート ]
今の区間の音声をもう一度再生します。
[ 自動 ]
現在のポジションから自動で最後まで各区間を再生します。
この時 [ (ポーズ時間  秒) ] に入力された数値の秒数無音区間が
各区切れ目に設定されます。この秒数は再生中も変更可能です。
自動再生を解除するにはこのボタンを( [ 止める ] になってます)を
再度クリックします。
[ 書替え ]
表示文字列を書替え、修正する事が出来ます。
書換えた、もしくは書き加えたあと、このボタンをクリックして下さい。
(ハンテン 使用時はこの機能での書替えは出来ません。)
書換えたは後は内部データには保存されますが、読み込んだフアイルには
反映されません。[ 保存 ] ボタンから上書き・保存して下さい。
[ ランダム ]
ここにチェックを入れておくと、各区間をランダムに再生します。
通常、つねに全体から区間を選びますので同じ区間が再生されることもあります。
設定エリアで選択重複回避機能をオンにすれば重複無く全てが再生されます。
[ 逆行 ]
ここにチェックを入れておくと、通常再生と逆方向に
再生区間を移して再生して行きます。
[ ハンテン ]
ここにチェックを入れておくと、各区間を 2,1,4,3,6,5 と言う順で再生します。
ハンテン 使用時にはテキストの書換えは出来ません。
※注意1、これらのボタンを切り替えた時、直ぐに動作に反映しない、
あるいは、正しく動作しない場合があります。
これは動作完了時、既に次の動作が設定されているためです。
また ハンテン 時に最後まで再生しても、こちらの動作の特性から、
最後の一つ(あるいは二つ)手前の番号で終わることになります。
番号は終わっていなくても内部的には全て終わっていいます。
この時、他の設定に切り替えて再生ボタンをクリックしても動作しません。
その切り替えた設定に準じたポジションに移動して使用して下さい。
※注意2、複数チェックすると、これらは全て無効になります。
※注意3、これらをチェックすると、スキップは無効になります。
[ オート ]
チェックしておくと [ リピート ] ボタンでの再生をエンドレスに行います。
停止させる時は、 [ リピート ] ボタンが [ 停止 ] ボタンになってますので、
これをクリックして下さい。
[ ループ ]
これにチェックを入れておくと、
通常再生や逆行再生やランダム、ハンテンの自動がエンドレスになります。
通常再生では最後から先頭にもどって再度初めから再生します。
逆行再生では先頭から最後に戻って再度最後から再生します。
ランダムで、選択重複回避機能がオンの時は重複無く全てが再生された後
再生の消化情報が消去され、また重複なく再生されて行きます。
[ スキップ ]
これにチェックを入れておくと、
再生区間をつど都度一つずつ飛ばします。
先頭に [ |… ] で戻してからクリックした時は
1,3,5,7 …の様に進みます。
先頭に [ |… ] で戻し [  ] で一つ進めてからクリックした時は
2,4,6,8 …の様に進みます。
通常は今の区間の次の区間から スキップ 再生が始まります。
要するにチェックを入れると次の区間は再生され
その次の区間は飛ばされ、その次の区間は再生 … と言う様に進みます。
自動再生時にも有効です。
なお、こちらは逆行再生、ランダム、ハンテンには使用できません。
[ カクス ]
これにチェックを入れると、予めテキストが入力されていても表示されません。
[ 見る ]
「カクス」の状態で表示されないテキストを表示させたい時にクリックします。
[ 隠す ]
「見る」で表示されたテキストを再度隠す時にクリックします。
[ 設定 ]
設定エリアを開きます。
[ 保存 ]
表示文字列を書替え、あるいは修正した場合はこちらから、
読み込んだフアイルに上書きするか、新たに別途フアイルにして保存して下さい。
[ (ポーズ秒数) ]
自動再生時の各区間の間の無音区間を設定します。
0 〜 10,15,20,30 から選択して下さい。
自動再生中も変更できます。
設定エリア部
[ Voice-入力ボックス ]
使用する音声フアイルのパスを入力します。
参照ボタンを使用すると楽です。
[ Voice-参照 ]
フアイル選択ダイアログが開きます。
使用する音声フアイルを選択して下さい。
[ Text-入力ボックス ]
使用するテキストフアイルのパスを入力します。
参照ボタンを使用すると楽です。
[ Text-参照 ]
フアイル選択ダイアログが開きます。
使用するテキストフアイルを選択して下さい。
[ 奇数題 ](ランダム操作時)
これにチェックを入れると [ ランダム ] ボタンをクリックした時、
奇数題のみ選択される様になります。
[ 偶数題 ](ランダム操作時)
これにチェックを入れると [ ランダム ] ボタンをクリックした時、
偶数題のみ選択される様になります。
[ 再生 ](ランダム操作時)
これにチェックを入れると [ ランダム ] ボタンをクリックした後、
その次に [ ランダム ] ボタンをクリックした時は、その次の番号をヒットします。
その次はまたランダムになります。
[ 逆行 ](ランダム操作時)
これにチェックを入れると [ ランダム ] ボタンをクリックした後、
その次に [ ランダム ] ボタンをクリックした時は、その一つ手前の番号をヒットします。
その次はまたランダムになります。
[ OFF/ON ](ランダム操作時)
ランダム操作時の選択の重複を回避する機能のオンオフを設定します。
ON にすると重複せずに全ての区切り区間を再生します。
上の四つの何れかのチェックが入っている場合は、それぞれ一度づつ重複します。
OFF にすると、つど都度、全体から区間を選びますので
同じ区間が何度も再生されることもあります。
[ リセット ](ランダム操作時)
ランダム操作時の選択の重複を回避する機能がオンにしてランダム操作をしている時、
そのランダムにヒットした問題の消化の記録をリセットします。
[ 最前面を ON/OFF ]
最前面を ON で常に最前面。OFF で解除。
クリックするごとに切り替わります。
[ サイズを  ]
ソフト本体のサイズの変更を変更できます。
「サイズを 」のコンボボックスをクリックすと選択肢が現れます。
1.00/1.25/1.50/1.75/2.00 から何れかの倍率を選択し
起動しなおすと変更が反映されます。
[ 背景を ]
インストールフォルダに画像フアイルを置き、ここにそのファイル名を入力すると
ソフトの本体のデザインとして読み込み、起動しなおすと反映されます。
読み込めるファイルの種類は [ jpg/bmp/gif ] の三種類です。
初期状態では、添付のサンプル背景画像「haikei.jpg」を設定しています。
また、以下の書式で指定すると単色カラーに設定する事が出来ます。
rgb=赤の照度 0〜255.緑の照度 0〜255,青の照度 0〜255
例、rgb=255.255,255 なら白
例、rgb=0.255,0 なら緑
例、rgb=255.0,255 なら紫になります。
[ テキスト表示エリアの設定 背景色⇒文字色・文字サイズ・文字デザイン ]
このボタンをクリックすると、まずテキスト表示エリアの背景色を設定する
色の設定ウインドウが表示されますので、これを設定してください。
つぎにフォントを設定するフォントウインドウが表示されますので、
任意に変更して下さい。初期値は「MS ゴシック」「標準」「9」「黒」です。
[ ループ(自動)の再生回数 ]
ループ(自動)にチェックを入れている時のループ再生回数を設定できます。
入力した数値の数だけ再度はじめから再生されます。
ここに入力された数値はループ再生された数だけ自動的にカウントダウンされ
0になった時に自動的にループ(自動)が解除されます。
初めから0が入力されている場合はループ(自動)が解除されることはなく
エンドレスに何度もループ再生されます。
[ シャットダウン ]
ここにチェックを入れると、自動再生時に全ての区間を再生し終わったら
自動的にパソコンがシャットダウンします。[ 自動 ] ボタンが [ 自動! ] の様に
[ ! ] が付加されているのを確認して下さい。この時 [ 自動 ] ボタンを
クリックすると [ 止める ] ボタンも [ 止める! ] になってます。
一度止めると解除されますので、シャットダウンさせるには再度設定画面を開いて
チェックしなおす必要があります。また、この設定は保存されないので
この機能を使うときは、その都度チェックする必要があります。
また [ ループ ] にしている時は当然無効です。
[  ](左下隅)
「pause_setter で改行コードを指定した状態で作成されたデータ」の
テキストフアイルをこちらで使用する場合は pause_setter で
行った設定と同じ方法でこちらにも同じ改行コードを設定しておいて下さい。
