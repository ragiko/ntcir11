植木算 ( うえき ざん )

方陣算 ( ほうじん ざん )

過不足算 ( かふそく ざん )
鶴亀算 と 親せき です。 図式で整理すると、どの部分が親せきか見えてきます。
表示される図式 を 使って、かくれている「 数 」 を 見つけだしてください。

学習ソフト
内 容
旅人算 ( たびびと ざん )
同じ距離でも、「追いつく時間」 と 「近づき出会う時間」 に 差 が でます。
表示される図 を 見ながら、速さ と 時間 と 距離 の 関係 を 学習してください。
通過算 ( つうか ざん )
旅人算 と 親せき です。 ちがう点は、「電車どうし」 が すれ違うとき や 「長いトンネル」
を 通過するとき の 「走る距離」 を 考える必要がある点です。
流水算 ( りゅうすい ざん )
旅人算 と 親せき です。 違う点は、 船の走る方向 と 川の流れ が 同じ方向 では
船は速く進み、川の流れ が 反対方向 では、 船は 流れにおされて 遅くなります。

分配算 ( ぶんぱい ざん )
「比」 の 計算 に、 足し算 や 引き算 を 組合わせて、 考えさせる 問題です。
表示される図式 で 整理し、 比 を 「目盛」の数 に 置きかえて 解く方法 を 考えます。
相当算 ( そうとう ざん )
「分数」 の 計算 に、 整数 を 組合わせて、 考えさせる 問題です。
表示される図式 で 整理し、 分数 を 「目盛」の数 に 置きかえて 解く方法 を 考えます。
倍数算 ( ばいすう ざん )
「比」 の 数 に、 足し算 や 引き算 を 組合わせて、 考えさせる 問題です。
表示される図式 で 整理し、 「比」 の 計算 を 使って 解く方法 を 考えます。
年令算 ( ねんれ ざん )
父と子 の 年令の「比」 は 毎年変りますが、 「年令差」 は 変りません。
表示される図式 で 整理し、 この変らない「年令差」 を 利用し、解く方法 を 考えます。 
仕事算 ( しごと ざん )
「1日の仕事量の比」 から いろいろな「仕事の量」 を 計算する 問題です。
表示される図 を 見ながら、 考え方 を 整理して 解く方法 を 考えます。

平均算 ( へいきん ざん )
平均 の 計算方法 を 学習します。
いくつかのグループ の 平均から、 全体の平均 を 計算する方法 も 学習します。
消去算・代入算( しょうきょ ざん ・ だいにゅう ざん )
消去算 と 代入算 の 計算方法 の 学習です。 
慣れるまで、くり返し練習してください。
濃度算 ( のうど ざん )
食塩が水にとけて 食塩水になると、「食塩の濃度」 という 考え方 が 出てきます。
この 「濃度」の意味 と 計算の方法 を 学習します。 慣れるまで、くり返してください。
損益算 ( そんえき ざん )
定価、原価、売値、利益、利益率 など 聞きなれない ことば が 出てきます。
これらの ことばの意味 と 計算方法 を 学習します。 慣れるまで、くり返してください。
