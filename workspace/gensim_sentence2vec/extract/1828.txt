M1383002 [教官:阿江] 小原 雄樹
動的環境向けAST強化学習法
強化学習とはエージェント自身が試行錯誤することで環境に適応する学習の枠組みであり,不確実性を含んだ問題も扱えるという特徴がある.強化学習では環境をマルコフ決定過程(MDP)によりモデル化した場合最適解への収束が保証されるが,実環境は動的に変化するものであり,MDPが保証されることはほとんどない.
一方,AST学習法はニューラルネットとオートマトンを合成した学習法である.状態間の遷移にオートマトンを用いることで決定的に政策の決定を行うが,最適系列の獲得が困難であるという問題があった.
本研究では,この2つの学習法を合成した学習法であるAST強化学習法を提案することで,問題の解決を試みる.AST強化学習法は,強化学習により最適系列の学習を行い,AST学習により状態間の遷移を学習する.このAST強化学習法が動的に変化する環境においても有効であることを実験により検証する.
M1383004 [教官:阿江] 亀井 清華
A Self-Stabilizing Algorithm for the Steiner Tree Problem (Steiner木問題を解く自己安定分散アルゴリズム)
自己安定とは、耐故障性のある分散アルゴリズムの理論的な枠組みの1つである。
本研究では、分散システム上におけるSteiner木問題を考え、既存のheuristicアルゴリズムの1つであるDistance Network heuristicに基づいて問題を解く自己安定分散アルゴリズムを提案する。分散システム上においてのSteiner木問題とは、マルチキャストのルーティング問題であると言える。本アルゴリズムは、(1)最短経路森を構成し、(2)ネットワークを変換し、(3)最小重み生成木を構成、(4)必要でない枝や節点を刈り取る、という4つの階層的な構造になっており、本研究では(1)、(2)、(4)の3つの自己安定分散アルゴリズムを提案する。
M1383006 [教官:阿江] 川本 幸司
A Distributed Algorithm for the Maximal Local Leader Election Problem(極大局所リーダー選挙問題を解く分散アルゴリズム)
分散システムとはプロセスと通信リンクの集合から構成されるシステムであり,プロセス間でメッセージを交換しながら,自律協調動作をし,ある共通の目的を達成するシステムである.本研究では,独立点集合問題とリーダー選挙問題を特別な場合として含むd-局所リーダー選挙問題を扱う.局所リーダー選挙問題とは,あるd ≧ 1に対して,距離d内にあるプロセスは同時にリーダーになってはならない,という問題である.d=1の時は独立点集合問題であり,d=D(ネットワーク直径)の時はリーダー選挙問題に対応している.本研究では各プロセスが固有の識別子を持つ場合に極大局所リーダー選挙問題を解く分散アルゴリズムをまず,d=2に対して提案し,その正当性及び評価を示す.また,その後,一般的なdの値に対して問題を扱う.
M1383007 [教官:阿江] 木村 英志
Generation of Sequence associated with KANSEI using Structured Vector Addition System (Structured Vector Addition Systemを用いた感性を伴う系列の生成)
様々な計算モデルのうち,古典的モデルとしてKarpらが提案したVector Addition System(VAS)がある.VASは情報処理,特に人工知能の分野における知識獲得・表現の問題の1つの解決法として有益なモデルである.従来の人工知能の手法では,主観性・多義性・環境依存性をもった情報,特に感性と呼ばれる情報処理においては,その不明確な因果律や曖昧性により知識を獲得し,表現することは困難であった.ここで,感性をベクトル表現し,数値ベクトルを状態とすると,音楽のような系列に対しベクトル系列により感性を対応させることが可能となる.しかし,系列が長くなると近似を行ったとしてもあまり有効な結果が得られなかった.そこで,本稿では,VASを階層化したモデルであるStructured Vector AdditionSystem(SVAS)を用いることにより,感性を伴う系列を生成することを目的とする.例として音楽における楽譜,特に和音列の生成システムを構築し,評価する.

M1383021 [教官:阿江] 鶴見 誠悟
モデル検証を用いた並行プログラミング学習支援システムの試作
並行プログラムは複数のプロセスが相互作用しながら実行されるプログラムである.プロセスのスケジューリングは実行ごとに異なる場合があるため,非常に多くの実行例が存在する.この理由によりプログラムは各スケジューリングに対して正しく実行されなければならない.しかし無数に存在するプロセススケジューリングを全て把握することは困難であり,正しい並行プログラミングを行うことは難しい.本研究では初心者を対象とした並行プログラミング学習支援システムを試作した.このシステムでは,学習者にプログラムの空欄補充問題を提供する.学習者が解答したプログラムを形式的検証手法の一つであるモデル検証により検証し,全てのプロセススケジューリングにおいてプログラムの正誤判定を行う.またプログラムの直感的な理解を促すために,プロセスの状態や実行の履歴を学習者に表示する.

個人の価値観を含むマルコフモデルを用いた系列の獲得及び生成
現在、データ系列の分類及び生成に関する研究が行われ、その応用目的により様々な手法が提案されている。ここで系列の要素に画像、文章などのマルチメディアデータを含む系列について考えると、定型化できない部分が存在する。このようなデータは個人の価値観によって捉え方が異なるものであり、現在の状態にも依存する。
そこでこのような価値観をベクトルとして表現し、マルチメディアデータを系列の要素として含む系列の獲得及び生成法について提案する。
手法としては、データベースの要素を自己組織化マップによりクラスタリングし、価値観の変化に伴う要素系列を隠れマルコフモデルの状態に価値観ベクトルの代表を対応させたVMM(Vector-state Markov Model)により獲得する。
本手法の応用として、都市系列生成システムに適用し、評価を行う。
M1280804 [教官:阿江] 板垣 秀生
Vector Addition System 上の問題解法とベクトル値つき知識への応用
本研究では,まず,Karpらが提案した並列加算モデルの一つであるベクトル加算システム(Vector Addition System,以下省略してVAS)を導入する.梅田らによってVAS上の系列探索問題が定義され、拡張されたGAによりその問題を近似的に解くアルゴリズムが考案されている.VAS上の系列探索問題のサブクラスであるn到達問題とは,VAS上の時刻nにおいて,指定したベクトル点に到達する系列が存在するかどうかをYes/Noで判定する問題である.
n到達問題を,ベクトル値つき知識上の推論により生成されるベクトル点列の制御に用いるなら,求めた結論が目的に合ったものかどうかを調べることができる.しかしこの場合,求めた結論とそれに付加されたベクトル点しか評価の対象として扱われておらず,推論の途中過程であるベクトル点の系列全体は扱われない.ベクトル点の系列全体を扱うためには,系列全体の振舞を考慮した新しい問題が必要である.そこで本研究ではベクトル系列全体の軌跡の中で,最も大きいベクトル点を持つ系列を見つけるための系列探索問題を定義する.問題を定義するために,ベクトル系列上での振幅という概念を定義する.本稿で用いるベクトル系列上の振幅とは,系列中の始点と終点以外で指定された点でのユークリッド距離である.VAS上の時刻nにおいて,ゼロベクトルに到達する系列の集合の中で,振幅が最大である系列を求める最大振幅問題を定義する.この問題はNP完全であるので,実用で用いる場合には近似解法が必要となる.ゆえに,本研究ではその問題の特徴を考慮した近似解法を示す.
次にVAS上の最大振幅問題を,ベクトル値つき知識上の推論で生成されるベクトル点列の制御に適用する.これによって,ベクトル点列全体で評価された記号列を求めることができると考えられる.

An MPEG2 over IP Transfer System Using Dynamic Redundancy of FEC
近年のネットワークのフロードバンド化に伴い,インターネット上を流れる情報は映像,音声を含むマルチメディア情報がその多くを占めるようになってきた.しかし,ベストエフォート型のインターネットでは,伝送路上には多くのルータが存在し,さらに伝送路も他のトラフィックと共存する形が一般的である.このような環境下では,常に変化するトラフィックがネットワーク品質にも影響を与え,それがリアルタイム性を重視する映像や音声の品質の低下をもたらす.FECを用いることでパケット損失に対する耐性を強化することは可能だが,FEC回復後のパケット損失率は通信路自体の損失率に依存するため,通信路上で損失率が変化するような場合に脆いという側面もある.本稿では,冗長度動的変更による動画像伝送の損失率制御方式を提案し,これをMPEG2動画像伝送システムに適用する.また評価実験を行うことで,パケット損失率が変化するような環境下においてシステムが効果的に機能することを示す.

QoSルーティングにおけるパス候補集合生成問題に関する研究

ソフトウェア信頼性モデルはソフトウェアの信頼性を定量的に評価するための基本ツールであり,テストの進捗状況の把握やソフトウェアのリリース時期の決定などに用いることが可能である.実際にソフトウェア信頼性モデルを開発工程で用いる際に,観測されたフォールト発見時間データに基づいてソフトウェア信頼性モデルに含まれるパラメータを推定する必要がある.最もよく用いられているパラメータ推定手法は最尤法であるが,最尤推定値を求めるためには複雑な非線形方程式を数値的に解く必要がある.しかしながら,従来の Newton 法を始めとする非線形計画法に基づいた推定値の方法は初期値に依存し,常に望ましい解を得ることが困難であった.本稿で提案する EM アルゴリズムは比較的計算の容易な更新式を有限回実行するという単純な計算により,安定して最尤推定値を得ることが可能である.また,数値例において,既存のパラメータ推定手法と本稿で提案したパラメータ推定手法における解の安定性に対する比較を行い,提案した推定手法の有効性を検証する.

An algebraic method for finding travelling wave solutions to nonlinear differential equations
現代科学では非線形問題が非常に重要な課題となっており、過去非線形であるがゆえに解決できなかった(あるいは避けて通った)重要な問題も多く存在している。非線形偏微分方程式の厳密解を求めることは、理論的な興味だけではなく、数値計算による解の信頼性を評価する上からも重要な意味を持っている。そのなかでも、進行波解を厳密に求めることは解の漸近的な振る舞いを知る上でも重要であり、非線形現象の理解に重要な情報を提供してくれる。今までに、進行波解を代数的に求める手法として、sech やtanh型の解を個々に求める手法が提案されているが、本研究では、楕円関数を解として含む常微分方程式の解による展開によって、より広いクラスの進行波解が得られることを示すとともに、具体的な解を求める数式処理プログラムを示す。

A study on computer algebra system for solving ultra-discrete equations
ソリトン方程式と呼ばれる非線形偏微分方程式には無限個の保存量やN-ソリトン解が存在するが、この様な性質はソリトン方程式だけに限らず、独立変数を離散化した差分方程式や従属変数を離散化した超離散方程式にも存在する。しかし、超離散系においてはその計算の複雑性が研究のネックになっており、超離散方程式に現れるMax-Plus Algebraの数式処理化が望まれている。

An Automatic System for Counting and Capturing the Pictures of Moving Vehicles in Real Time
In recent years there has been an increase in the interest placed on Intelligent Traffic Monitoring Systems(ITMS).This paper presents a robust, inexpensive, automatic vision based moving vehicle detection and capturing system in real time using a single video camera.Vehicle counting is done considering a smaller window from the main frame.Predefining the window gives the upper hand on excluding the irrelevant movements in the main frame while focusing on the traffic movements from a far distance and also less computation time.We define a specific method based on frame comparison with the background and pattern matching avoiding multiple counts with adaptive updating of the background with the elapsing periodic time.The system is implemented and the experimental results from the traffic scenes are provided which demonstrate the effectiveness of the approach.
