当団体について
IBREA 財団は非営利の 501 (c) 3 団体です。その使命は、人間の脳の可能性を実現できるよう個人、地域、国を教育することによって、人類と地球に健康、幸福、平和をもたらすことです。以下のポリシーには、当団体のウェブサイト http://braintest.ibreaus.org/ (以下「ウェブサイト」) の閲覧者から得る情報の収集方法と利用方法が記載されています。このポリシーは、本ウェブサイト上で収集された情報にのみ適用され、その他 (オンラインまたはオフライン) の手段によってあなたから IBREA が収集するその他の情報には適用されません。

収集される情報、およびその収集方法について
IBREA は、当団体のサイトにログインして「ブレインテスト」にアクセスする目的で自らの意思でアカウントを作成する閲覧者から、(1) 「ログイン情報」および (2) 「人口統計情報」を収集します。「ログイン情報」には、電子メールのアドレスまたはニックネーム、ならびにアカウントのパスワードが含まれます。収集された「人口統計情報」には、性別および誕生日、最終学歴、母国語、居住国が含まれます。IBREA はまた、「ブレインテスト」を受ける利用者から成績データも収集します。このデータには、利用者の得点およびレベル、タイミング、ならびに「ブレインテスト」について利用者が自ら提供した意見・感想が含まれることがあります。
現在、IBREA がサイトの閲覧者から情報を受動的に収集することはありません。cookie や IP アドレス追跡、非表示のフォームフィールド、window.name、HTTP 認証などの電子追跡デバイスは使用していません。URL (クエリ文字列) は、現在ナビゲーション目的でのみ使用されており、ユーザーデータの取り扱いには使用していません。いつの時点でも、IBREA は、いかなる形式または表現によるユーザーの同意を求めることなく、上述の追跡技術を使用する決定を下すことができます。
現在、ユーザの登録情報は HTML 書式を使用して収集してから、POST メソッドを使用して PHP ダイナミックスクリプトに送信されています。PHP スクリプトで、MySOL データベースへのクエリを実行し、データを確認します。データの確認後、無作為に生成された個別のユーザ ID を含むPHP セッションが作成されます。この ID を使用して、ウェブサイトおよびブレインテストのあいだ、ユーザーを特定します。IBREA は、閲覧者がアクセスまたは閲覧するウェブページについての情報、およびアンケートによって集めた情報およびサイト登録時の情報など、閲覧者が自らの意志で提供する情報をのちほど収集する場合があります。
IBREA は児童オンラインプライバシー保護法 (COPPA) に従って行動します。氏名や住所、電子メールのアドレス、電話番号、他人が児童を特定または児童に接触することを可能とするその他の情報など、13 歳未満の児童に関する情報を個別に特定可能な情報を収集または保持することはありません。IBREA ブレインテストは、児童向けのものではありませんが、13 歳未満の児童がアカウントを作成し「ブレインテスト」にアクセスすることを選択する場合、ログイン目的で使用される児童の電子メールアドレスまたはニックネームは破棄され、「child」という単語と無作為に生成された数字から構成される、「child6376」のような匿名のユーザー名が代わりに使用されます。各閲覧者の年齢は、オンライン登録時に提供された誕生日にのみ基づき計算されます。
収集した情報の 、IBREA による使用方法
IBREA は、脳のさまざまな認知機能を測定するオンラインテストを開発しました。このテストは Elkhonon Goldberg 博士の科学的監修のもとで開発されたもので、このテストを使用して、記憶力、注意力、問題解決能力、空間推理能力、など脳機能のさまざまな側面を、標準となる標本に基づいて他の受験者と比較できます。収集したデータから標準となる標本が作成され、その標本を使用して IBREA のスタッフはテストを受けた母集団とテストの結果を比較し、分析できるようになります。結果は、年齢群および / または教育水準、性別、母国語、居住国ごとに区分されたり、グループ化されたり、分析されたりします。IBREA は利用者レポートを作成し、それぞれの利用者のテスト結果を、人口統計情報と比較します。
テストについてのフィードバックは改善および再開発のために使われ、もっとも有効で有用なテストを利用者に提供することを目標としています。
IBREA は、ウェブサイトのホスティングを行うパートナーや、当団体のウェブサイトの運営または当団体の商品またはサービスの提供を支援するその他の当事者と、一部の情報を共有する場合があります。IBREA はまた、法律で必要とされる場合、あなたの情報を共有します。上述の場合を除き、IBREA は閲覧者の個人情報を第三者と共有したり、第三者に譲渡したりすることはありません。
収集した情報の保護方法
収集した情報には、プロジェクトのデータベースまたはデータ分析に使用する社内の管理アプリケーションへのアクセスによる場合にのみアクセスできます。これらのリソースはいずれもパスワードで保護されています。アクセスして情報を収集できるのは、ウェブの開発者とプロジェクトの管理者のみです。データベースへのすべてのクエリは、悪質な SQL インジェクションや、不正アクセスを引き起こす可能性のあるハッキング方法から保護されています。
閲覧者による情報へのアクセスと情報の変更、ならびに将来のやり取りへの不参加
利用者は、「プロフィールを編集」リンクをクリックすれば、自分の「ログイン情報」または「人口統計情報」にアクセスし、それらの情報を更新できます。「プロフィールを編集」リンクは、ウェブサイトにログインしたあとに利用できます。利用者は、過去のテストの点数やフィードバックなど、現存するテストデータを変更することはできません。
パスワードのリセットの要求などの対応を求めてユーザが IBREA に連絡する場合を除き、IBREA には、「ブレインテスト」のユーザに連絡する計画はありません。
