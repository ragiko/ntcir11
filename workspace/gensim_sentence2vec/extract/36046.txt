Only が文頭にきたとき～入試で良く出題される落とし穴
英語—文法
今年の同志社大学文学部の英語入試問題を解いた記事を昨日アップしましたが、興味がある人でなければ見ていないと思います。記事は少し長くなりましたし、解答だけなので、手元に問題がなければ読んでも意味がないからです。今回はこの入試問題の中に、気になる問題がありましたので、此処で取り上げます。昨年より、各大学の入試問題を解いてきましたが、幾つかの大学で共通して設問としている形式がありました。それは、ONLY を文頭に出したときの文章構造についてです。文法問題は苦手ですが、その私でさえ目に付いた問題ですので、出題者にとっては出し易い問題なのかもしれません。今回取り上げる問題は、2011年同志社大学文学部英語入試問題Ⅰ番からの抜粋です。設問は、長文の中の一文を取り上げ、その文章の一部を空白にして、単語を当て嵌める問題です。本来なら、その長文も取り上げるべきですが、此処では気をつける点を示すことが目的ですので、問題だけを取り上げます。このような形式の文章について、知識を持ってもらいたいためです。設問は、Ⅰ-E の問題です。以下の空所( あ )～( か )に、その下に示す1～6の単語を入れて文章を完成させる問題です。同じ単語は二度使いません。Only ( あ ) ( い ) ( う ) were seen can ( え ) ( お ) ( か ) that were sending the right signals. 単語の選択肢は、次のもので、空白と数は一緒です。1by  2how  3make  4sure  5understanding  6we少し考えてみて下さい。主語と動詞の位置などを考慮に入れて考えれば、長文がなくても答えられると思います。なんだ、簡単じゃないか、という人は、この問題に仕掛けられた落とし穴に嵌ることはありません。では、考えていきましょう。まず、can のあとは動詞がくるのでしょうから、3しかありません。とすれば、34ではないかと仮に考えられます。これで、(え)と(お)は埋まりました。と、このように考えると、出題者の用意した落とし穴に陥ることになります。昨年から入試問題を解いていて、このOnly が文頭にきたときの倒置についての設問が多いことに驚いています。Only after an operation will he be able to walk again. 手術を受けて漸く彼は再び歩けるようになるだろう。手元のジーニアス英和辞書の例文です。これと同じで、Only が文頭に来ていますので、この空白の文章も倒置が起こっている筈なのです。このことを弁えて、この空白だらけの文章の意味を考えてみましょう。were seen についての何かだけ(Only)が、that were sending the right signals. について何かをできる(can)、ということのようです。我々がどのように見られているか、ということではないかと思い、how were seen となりそうだと推測してみます。そうすると、Only に続くのは、によってのみという意味になる、Only by ではないかと思い浮かんできます。このように理解すると、私達がどのように見られているかによってのみ、ということになります。そして、by に続くのは残りの中から、understanding しかないでしょう。つまり、Only by understanding how were seen となりました。次は倒置の部分です。この倒置に気付かないと、残りのwe がどこに入るかわからなくなってしまうのではないでしょうか。そのため、we の当て嵌まり先を探して、また最初から考え直すことになると思います。出題者の意図した罠に嵌っていくわけです。時間だけが過ぎて、結局解答できないようになると思います。can には主語が必要ですから、倒置型だとわかれば、can we となることは残った単語から明白です。次に来るのは動詞しかありません、make です。これで解答できます。can we make sure です。Only by understanding how were seen can we make sure that were sending the right signals. 選択順序は、1、5、2、6、3、4です。倒置がヒントです。このことが判っていれば、この問題は秒殺に近いでしょう。知らなければ、何分もかかって、しかも正答には至らないと思います。出題者の思惑通りとなってしまうのです。Only が文頭に来ているときは要注意です。追伸同じ問題中のⅠ-C (イ)の解答について、東進の入試データベースでは、選択肢3を正答としてあり、赤本では選択肢2を正答としています。私にはこれを断定する実力がありません。赤本の方は今日立ち読みで見ただけなので、もしかしたら見間違いかとも思いますが、どなたか、この問題に決着できる方いらっしゃいませんでしょうか。
