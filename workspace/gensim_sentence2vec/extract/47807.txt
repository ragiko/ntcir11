MMDAgentで遊んでみよう
MMDAgentとは、名古屋工業大学国際音声技術研究所によって作成されたオープンソースの音声インタラクションシステム構築ツールキットです。音声合成、音声認識、3D描画、リップシンクの技術を組み合わせることで、MMDモデルとの会話することができます。当ブログはMMDAgentでゲームをするためのシナリオMikuplusを動かすための手順を説明します。
MMDAgent概要Mikuplus概要MMDAgentを使用したゲーム?シナリオ、ここから始まる一連の動画を見ればわかるでしょう0. 動作環境ハイスペックなCPUとグラボの64bitWindows7/8.1 のPC。1. MMDAgent本体を入手する。MMDAgentのサイトhttp://www.mmdagent.jp/ から  Binary Package (for 32-bit Windows) とSample Script Binary Package をダウンロード。これを解凍するとMMDAgent_win32-1.4とMMDAgent_Example-1.4フォルダができるので、MMDAgent_Example-1.4フォルダ内の全ファイル・フォルダをMMDAgent_win32-1.4に移動します。この状態でMMDAgent_Example.mdfをMMDAgent.exeにドロップして起動します。これが動かなければそのPCでは無理ということです。マイクは起動前に接続しておく必要があります。無事にメイさんが出てきたらマイクのテスト「こんにちは」、返事が返ってくればOK、次に進みます。2. Mikuplusシナリオを入手する。version1.4シナリオ http://med.no.land.to/pub/Mikuplus1.4.zip 解凍してフォルダ内の全ファイル・フォルダをMMDAgent_win32-1.4に移動します。詳しくはシナリオに同梱されているMikuplus1.4.txtを見てください。3. ミクの音響モデルを入手する。Exampleにはmeiさんのボイスしか入っていないので、Mikuplus1.4.ojtでは男声の音響モデルとミクの音響モデルを追加しています。1.4では音響ファイルの形式が新しくなっているため、今までの形式の音響モデルはhtsvoiceに変換する必要があります。4. MMD用モデル、モーション、曲を入手する。一部のモデル・モーションには改造が必要になる場合があります。5.出来上がったフォルダ構成の例MMDAgent_win32-1.4の構成例MMDAgent_win32-1.4¥Modelの構成例MMDAgent_win32-1.4¥Motionの構成例MMDAgent_win32-1.4¥Pluginsの構成例MMDAgent_win32-1.4¥Soundの構成例MMDAgent_win32-1.4¥Stageの構成例MMDAgent_win32-1.4¥Voiceの構成例6. 起動する。MMDAgent_win32-1.4フォルダ内のMikuplus1.4.mdfをMMDAgent.exeにドロップして起動します。終了するのはEscキー7. ニコニコ生放送でライブチャットをする。CUBE370氏がMMDAgentを改造して、生放送のコメントに反応する、Project-NAIPを配布しています。http://cube370.blog87.fc2.com/http://cube370.wiki.fc2.com/それを利用して、CUBE370氏(不定期)http://com.nicovideo.jp/community/co1467509deg氏(不定期だが金土の夜が多い)http://com.nicovideo.jp/community/co106496くなつ氏(毎週金土の21時頃から)http://com.nicovideo.jp/community/co1486848などが生放送を行なっています。数日前に行われた放送も見ることができるので、どんな感じになるのかわかると思います。質問があったら生放送にコメントすると良いでしょう。￼
