
Cosminexus HTTP ServerとHitachi Web Serverで使用している乱数生成アルゴリズムについて,NIST(米国国立標準技術研究所)からの勧告がありました。
脆弱性の内容
Cosminexus HTTP ServerとHitachi Web Serverで使用している乱数生成アルゴリズムについて,NIST(米国国立標準技術研究所)からの勧告がありました。
SSL機能を使用する場合,勧告に従い,当該乱数生成アルゴリズムを使用しないようにいたします。
影響を受けるバージョンを下記に示しますので,対策版の適用をお願いいたします。
なお,SSL機能を使用していない場合は,本脆弱性は発生しません。
