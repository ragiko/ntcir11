Lotus Domino Server セットアッププログラムにおけるインド語派言語のサポート   
サーバーのセットアッププログラム画面のフィールドに文字を入力する際に表示されるフォントとアルファベットの両方を変更できます。通常は、デフォルト言語のアルファベットが表示されます。 
Lotus Domino Server セットアッププログラムでは、次のアルファベットを使用できます。
ベンガル
デーヴァナーガリー
グジャラート
グルムキー
カンナダ
マラヤーラム
オリヤ
タミル
テルグ
フォントを変更するには
メモ  デフォルトのフォントはデーヴァナーガリー文字に対応していないため、フォントを変更する必要があります。
1  Lotus Domino Server を起動してセットアッププログラムを開始します。
2  [Welcome] 画面で、[フォント] をクリックします。
3  使用する予定のアルファベットに対応しているフォントを選択します。
4  デフォルトの言語とは異なるアルファベットを選択する場合は、次の手順を参照してください。
アルファベットを変更するには
アルファベットの変更は、Microsoft Windows、IBM(R) AIX(R)、Linux でのみサポートされています。
1  Lotus Domino Server を起動してセットアッププログラムを開始します。
2  デフォルト言語のアルファベットとは異なるアルファベットを使用するテキストを入力する画面のタイトルバーを右クリックします。
3  [入力方式の切替え] を選択します。
4  使用するアルファベットを選択します。
5  画面上のフィールドにテキストを入力します。複数のフィールドに入力できます。
メモ  [次へ] をクリックして次の画面に進むと、デフォルト言語のアルファベットに戻ります。異なるアルファベットを使用する画面ごとに、上記の手順を繰り返してください。
