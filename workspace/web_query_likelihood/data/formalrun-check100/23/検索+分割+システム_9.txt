本論文ではプログラミングの初学者を対象に,モジュール分割の観点からプログラミングスタイル学習を支援するシステムについて述べる.本システムはデータフロー解析によるアクティブ変数の情報を利用してモジュールの自動分割及び,その分割パターンを提示することで学習者が自らのプログラミングスタイルについて診断を行うことができる.筆者らは本システムを開発を行い,実際のプログラミング演習においてアンケートによるシステムの評価を行った.本論文ではシステムの開発・評価の結果について報告する.
This paper describes a learning support system of a programming style for the programming beginners from the viewpoint of module division. This system performs automatic module division using the information on actives variable in data flow analysis and shows user the module division patterns.User can judge whether the program written by themselves has accomplished good module division based on the presentation information of the system.The authors developed the system and evaluated the system by the questionnaire in an actual programming exercise.This paper describes the development and the practice evaluation of the system.
