自宅のネットワーク
ネットワーク構成と接続機器について
以下はmydrive.jpのネットワーク構成図である。2つ離れた部屋どうしを無線LANでの接続に変更した。デジカメの写真やビデオカメラのAVCHDファイルをPCに取り込んだあとファイルサーバーに保存するのだが、やはり有線LANのほうがファイル転送に時間がかからない。見た目や気分のすっきり感には効果があるんだけど。
主に自宅で利用しているネットワーク接続機器について簡単にまとめてみた。
まずはネットワーク・インフラから。
インターネット接続回線・サービス : 回線はNTT東日本のBフレッツマンションタイプで、ひかり電話も利用。ただし、VDSL接続なのでフレッツTVには未対応。プロバイダは朝日ネットで、固定IPアドレスオプションを利用(+1000円/月)。学生時代のパソコン通信の時から朝日ネットは継続利用している。速度測定サイトでパフォーマンスを測定したところ20−30Mbps程度は出ていたので、普通にネットを利用する分には問題ない。
ブロードバンドルーター(無線LAN対応) : 2012年8月にNECのAtermWR9500Nに更改。11naに対応しており実測200Mbpsオーバーのスループットが得られるようで、IMPRESSのInternetWatchでも評価が高かったため11acを待たずにこれを購入した。子機とセットになったタイプを購入したことで部屋どうしに無理やり引っ張っていたLANケーブル(10メートル程度)を撤去できた。大容量のファイル転送は有線のほうがストレスはないのだが、普通にネットを利用する分にはまったくパフォーマンス的に問題はない。
ギガビット対応のスイッチングハブが急激に安くなって1台あたり3000円程度で購入できるようになったころ、家中のあちこちに3台のスイッチングハブを転がしていた。今回の無線LANルータセットの購入により、スイッチングハブ単体を設置する必要はなくなったが、現時点でスイッチポートに空きがない状況になっている。将来的に(いつになるのかまったく未定だが)TVやAVアンプがネット対応となったときに、それらが無線LAN対応でなければ無線LANルータ(親機)のスイッチポートに現状の構成のままでは有線LANでの接続ができない。ま、その時に考えればいいんだけど。
ネットワーク・インフラに接続されている端末について。
リビングにあるもの。
リビングの液晶テレビ : 2006年のドイツワールドカップが始まる前に大型TVが欲しくなり購入。当時は1インチ1万円をオーバーしているような時代で、しかもどうせ買うならソニーのフラッグシップモデルが欲しいということでBRAVIA X1000の46型を購入。50万円ちょっとした。ネット接続やDLNAクライアント機能を持っているのだが、機能自体が非常に貧弱なので、今はネットにも接続していない状態。最近(2012年9月執筆)、起動時に画面が少し赤くなることが多く、少し経つと正常に表示されるのだが、丸7年以上経過したこともあり、そろそろ壊れる時期か・・・。I・OのRec-POTと組み合わせることで当時としては珍しい録画できるTVを実現できていて、それなりに重宝している。今ではなんでもない機能だけど。
HDDレコーダー : パイオニアのDVR-DT90を上記の液晶テレビに少し遅れること購入。デジタル1ch、アナログ1chの同時2ch録画ができる当時としては最先端を行っていたモデル。地デジを見てしまうとアナログには戻れずに結局デジタル専用録画機として利用している。自作のTV録画サーバーが快適に稼動しているため、未だに買い換えていないし、今後もレコーダー専用機は買うつもりはない。
PlayStation 3 : 自作のTV録画サーバーとセットで利用するDLNAクライアントおよびブルーレイディスク・プレーヤーとして購入。2世代目の少し薄くなったCECH-2000Aというモデル。TV録画番組もデジカメの写真もビデオカメラのAVCHDファイルも問題なく再生してくれる。BDリモコンで操作するのだが、30秒スキップなどの機能があるともっと快適に使えるのにと少し残念。ゲームは全くしない。
ホームサーバー(バックアップ兼再生用) : AcerのAspire easyStore Home Server H342(WindowsHomeServer)。実はH340というモデルを最初に購入したのだが、子供の写真や映像を失ってしまうと二度と取り戻せないということで、バックアップ用として追加で購入。NTT-Xストアセールをやっているときに購入した。DLNAサーバ機能もあるためリビングに設置してPS3で再生している。ネットワーク経由なのでリビングに置く必要はないのだが・・・。
地デジ録画サーバー : 自作のTV録画サーバの2号機。PT2を2枚実装して地デジ4ch同時録画を実現。BSアンテナがないため現在は地デジ専用となっている。24時間稼動なので省電力とパフォーマンスを考慮してCore i3を採用。セットアップの詳細はTV録画サーバー(2号機)に詳述しているので参考までに。これもネットワークで再生することになるのでリビングに設置する必要はないのだが。
ネットワークには接続されていないけれど、その他のAV機器など。
AVアンプ : AVアンプといったらYAMAHAのCINEMA DSPでしょということで、モデル末期だったDSP-AX2600を約半額で購入。その後、2ヶ月ほどしてDSP-AX2700が発売される。AX2600はHDMI入力がわずか2ポートしかないため、PS3とDVR-DT90以外は何も接続出来ない。今では考えられないが、ネットには非対応で、もちろんUSB端子もない。薄型TVの次に買い換えたい機器なのだが、どうせ買うならそれなりのモデルが欲しい。でも、今は予算的な余裕がなく、、、。基本的な機能には満足している。
5.1chスピーカーシステム : オンキョーのD-057Fシリーズで5.1chを構成。音に迫力はないが5.1chって感じがして、購入当初は感激したことを覚えている。初めて購入したAVアンプはソニーのTA-V55ESの中古で、そのときから利用している。
DVDプレーヤー : パイオニアのDV-600Aというモデル。たぶん10年近く使っているはず。PS3もあるしオークション行きになる予定だったのだが、「TVの電源とか入れずにサクッとCDが聞ける機械は必要」という意見があり、ずっと現役のまま。確かに。
リビング以外に設置されている、その他のネットワーク接続機器。
自分のPC : VAIO Note Type Aビデオエディション(VGN-AW82JS)というマシン。フォトエディションというマシンもラインナップにあったが、こちらはちょっとだけスペックがよく、その分、価格も高かったためソニースタイルでカスタマイズしてビデオエディションを購入。CPUだけは当時のノートPCの最上位を選択。Core 2 Duoって絶対フリーズせずに頑張るんだな、とパフォーマンがかなり上がったことに満足したことを覚えている。今となっては、ビデオ編集やエンコード時にもっとパフォーマンスがあるものが欲しくてたまらない。普通に使う分にはまったく問題なしの良いマシン。
妻のPC : 2006年末に購入したVAIOノート Type F TVというモデル。もともとXpマシンでありVistaへのアップグレードが無料でついていた。Vistaリリース後にアップグレードしようとしたが、Xpでも困らなかったことと、Vistaの評判がよくないこともあってアップグレードせずに利用した。その後、Windows 7がリリースされたのでこれをパッケージで購入し、現在もWindows 7マシンとして利用している。ずっと古いOSを使い続けていると仕様がガラリと変わった時に操作を覚えたり用語を理解したりすることに苦労することもあるため、あまり古いOSは使わせないようにしている。
Webサーバー : このコンテツを公開しているサーバー。詳細は"Server"メニューを参照。
ホームサーバー : AcerのAspire easyStore Home Server H340(WindowsHomeServer)。AcerのHomeServerの初代にあたる。先ほどのHomeServerはバックアップがメインの用途だが、これは普段使いのサーバーとして利用している。おかげでノートPCのローカルディスクにはほとんどデータを残さずに利用できるようになった。HDDベイも4つあるため、相当の期間は容量が足りなくなるようなことはないだろうと思っていたのだが、写真やビデオ、木工関係のデータなどを保存し、ドライブエクステンダーも利用したりしているうちに残容量もかなり少なくなっている。ドライブエクステンダーはフォルダ単位で複製をする/しないを設定できて手軽にバックアップを確保することができるので重宝している。WindowsHomeServer2011では一般的なRAIDになってしまったとのことで、この機能も残しておいてくれると良かったのに。
NAS : BuffaloのLS-CH1.5TLというマシン。WHSを導入する前の主力のバックアップ機として利用していた。今もバックアップ用マシンとして利用している。ということは、写真やビデオはWHS 1号機(ドライブエクステンダー利用)+WHS 2号機+このNASということで、四重化!?保存になっている。WHS 2011もしくはLinuxなどで少し大きめのホームサーバを作りなおすことも検討してはいるが予算の関係もあり当面はこの形態を続ける予定。なお、AcerのマシンもこのNASもPCの電源と連動して自動でON/OFFしてくれるのでありがたい。
複合機 : CanonのPIXUS MP990。かれこれ3年ぐらい利用。最新機種も印刷画質はそれほど変わらないようなので特にスペックなど不満はない。スキャナもCISではなくCCDなので見開きの本なども綺麗にコピーできる。自宅コピーはやはり便利でもっと早く買っておけばよかったと後から思ったものだ。ちなみに、インクジェットプリンターは初めて購入した時からずっとCanonでこれは4代目になる。それにしても交換インク代の高いこと。すでに複合機数台分を購入できるぐらいのインク代がかかっている。何とかならないものか。あと、家庭用のA3対応複合機(もちろん写真画質)が発売されるとうれしい。
こうやって見ていくと、好きなメーカーが決まっていて、AV関係はソニー、YAMAHA、パイオニア、PC関係もソニーが多く、「なんだ、結局ソニーファンじゃないの」ってなってしまう。小学生の時にいとこのお兄さんがソニーのアンプやカセットデッキなどを持っていて、「ソニーってかっこいいなー」と強烈な印象を持ったことを覚えている。それ以来、ウォークマンやミニコンポ、VHSビデオなどソニーがいつも周りにはあふれていた。なので、もっとソニーにはこれからも頑張って欲しいと思う。
余談だけど、白物家電は実家の影響で圧倒的にNational(現在はpanasonicだけど)ひいきです!
