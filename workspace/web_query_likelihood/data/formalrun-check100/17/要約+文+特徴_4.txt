製品の機能や仕様などを記述した製品性能表は,多くのデータを含んでいるが,どれがその製品の特徴であるかは言及されていない.本論文では,HTMLで記述されている製品性能表を解析し,それぞれの製品の特徴となるデータを抽出し,その結果を用いて表を文章として出力するシステムについて報告する.製品の特徴データは,表層的な数値および文字列の比較処理によって推定される.特徴化された表データは,あらかじめ用意した出力文章フレームに適用され,文章が生成される.
Specifications contain many data, but it is not clear which is the most important data in them. In this paper, we propose a method for extracting important data from specifications of computer system and generating sentence from extracted important data. First, the table structure is generated from specifications written in HTML. The key data items such as the uppermost model and standard values are then extracted from the table structure. Next, important data are determined by comparing each data item with the key data. Finally, sentences are generated from sentence frames of which the slots are filled with important data.
